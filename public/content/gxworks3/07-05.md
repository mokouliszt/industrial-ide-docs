# 7.5 デバイスメモリデータのエクスポート/インポート

デバイスメモリデータをCSVファイルにエクスポートできます。

また、エクスポートしたCSVファイルをインポートできます。

### エクスポート

デバイスメモリデータをCSVファイルへエクスポートする操作手順を示します。

データをエクスポートするには、あらかじめ"表示形式"画面の"表示単位形式"に"ワード多点"を設定する必要があります。

(→ 表示形式の設定)

操作手順1.デバイスメモリエディタでエクスポートするデバイスの範囲を選択します。2. [編集][ファイルへエクスポート]()を選択します。3."エクスポート設定"画面でエクスポートする範囲を選択します。

"範囲指定"を選択した場合はエクスポートする範囲の先頭デバイス番号と最終デバイス番号を入力してください。4.必要に応じて"設定済みデバイスが含まれる行のみエクスポートする"にチェックを入れ、[OK]ボタンをクリックします。5."ファイルへエクスポート"画面で各項目を設定し、[保存]ボタンをクリックします。

デバイス値の表示形式に従って、デバイスメモリデータがCSVファイルへエクスポートされます。

必要に応じて表示形式を設定してください。ただし、"表示単位形式"には"ワード多点"を設定する必要があります。

表示形式を設定する方法は、下記を参照してください。

→ 表示形式の設定

### インポート

デバイスメモリデータのCSVファイルをインポートする操作手順を示します。

GX Works3でエクスポートしたデバイスメモリデータのCSVファイルのみをインポートできます。

操作手順1. [編集][ファイルのインポート]()を選択します。2."ファイルのインポート"画面でインポートするCSVファイルを選択し、[開く]ボタンをクリックします。3.表示されたメッセージを確認し、必要に応じて"インポート実行前の設定"の"インポートするデバイス種別のメモリデータをすべてクリアする"にチェックを入れます。4. [OK]ボタンをクリックします。

インポート前にデバイスメモリデータをクリアする"インポート実行前の設定"の"インポートするデバイス種別のメモリデータをすべてクリアする"にチェックを入れると、インポート前にデバイスメモリデータがクリアされます。

チェック状態ごとの動作を示します。

![図(元マニュアル p.551)](images/gxworks3/07-05-01.webp)

注意事項

- インポートするCSVファイルを編集した場合、正常にインポートできないことがあります。

- インポート後、意図したデバイスメモリデータが正しく反映されたことを確認してください。

- 複数のデバイス種別のデバイスメモリデータは同時にインポートできません。

デバイス種別ごとにデバイスメモリデータをインポートしてください。
