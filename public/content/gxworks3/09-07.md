# 9.7 デバイス使用状況の表示

指定したデバイスについて、プロジェクト内での使用状況を表示します。

| 対象エディタ | 対象データ | | 備考 |
|---|---|---|---|
| プログラムエディタ | / ラダープログラム / STプログラム / FBD/LDプログラム / SFCプログラム | | ― |
| ラベルエディタ | / グローバルラベル / ローカルラベル | | "(現在のウィンドウ)"を指定している場合は対象外です。 |
| 構造体エディタ | 構造体 | | ― |
| パラメータエディタ | CPUパラメータ | / マルチCPU間リフレッシュ設定 / デバイス間転送設定 | "(現在のウィンドウ)"を指定している場合は対象外です。 |
| パラメータエディタ | ユニットパラメータ | / リフレッシュ設定 / シンプルCPU通信設定 / シンプル機器通信設定 / 相手機器接続構成設定 / 安全通信設定 | / "(現在のウィンドウ)"を指定している場合は対象外です。 / シンプル機器通信設定の場合、有効な交信相手設定の"プロトコル設定"画面で、"受信照合結果"列および"デバイス" 列に設定されているデバイスが解析の対象です。また、 MX-Fモデル、FX5CPUは対象外です。 |
| パラメータエディタ | 増設ユニットおよび通信アダプタのパラメータ設定 | / リフレッシュ設定 / シンプルCPU通信設定 / シンプル機器通信設定 / 相手機器接続構成設定 / 安全通信設定 | FX5CPUのプロジェクトでのみ対象となります。 |
| その他 | CPUユニットの内蔵機能 | | FX5CPUのプロジェクトでのみ対象となります。 |

**画面の開き方**

[検索/置換] ⇒ [デバイス使用一覧]を選択します。

![図(元マニュアル p.577)](images/gxworks3/09-07-01.webp)

**操作手順**

各項目を設定し、[Enter]を押します。

検索範囲を指定する場合、デバイス/ラベルの検索場所をツールバー(1)ので選択します。(複数選択可)

表示形式を設定する場合、ツールバー(1)の[設定] ⇒ [一括設定]を選択後に表示される画面で設定します。

ラベルに割り付けられたデバイス、およびデバイスが割り付けられたラベルの使用状況を検索する場合、ツールバー(1)のをクリックします。

- デバイス使用一覧上で、デバイスコメントを入力/編集できます。"コメント"列にコメントを入力し、[Enter]を押してください。

- 下記いずれかの操作によって、コメントのみコピーまたは貼り付けできます。

- 右クリック ⇒ ショートカットメニュー[コメントコピー]を選択

- 右クリック ⇒ ショートカットメニュー[コメント貼り付け]を選択、または[Ctrl]+[V]を押す

- デバイス使用一覧で任意の行を選択後、右クリック ⇒ ショートカットメニュー[コメント編集]を選択すると、コメントが定義されているエディタが開きます。

### 注意事項

- コイル命令の検索出力命令として下記の応用命令も検索対象です。

SET、RST、PLS、PLF、FF、SFT、SFTP、MC

- 割付けデバイスの検索下記のデバイスは、ラベルに割り付けていても検索できません。

- ダブルワードデバイス(LT、LST、LC、LZ)

- 間接指定されているデバイス

- 桁指定されているデバイス(ラダーエディタ以外で使用した場合)

- ファイルレジスタ(R/ZR)の検索デバイス使用一覧において、ファイルレジスタ(R)とファイルレジスタ(ZR)は区別して検索されます。

検索時は、それぞれ個別に指定してください。

FX5CPUは、ファイルレジスタ(ZR)には対応していません。

- ステップリレー(S)の検索検索時は、ブロック指定付ステップリレー(BL□¥S)を指定してください。

- ユニットパラメータに設定されているデバイスの検索未装着状態のユニットのユニットパラメータに設定されているデバイスは検索できません。

- デバイスの使用箇所の確認デバイス使用一覧の使用デバイスからクロスリファレンスウィンドウを開き、デバイスの使用箇所を確認できます。デバイス使用一覧で任意の行を選択後、下記のいずれかの操作により、クロスリファレンスウィンドウを表示してください。

- [Ctrl]+[E]または[Enter]を押す

- ダブルクリックする

**- 右クリック ⇒ ショートカットメニュー[クロスリファレンス]を選択するクロスリファレンスの関連項目**

→ デバイスおよびラベルの参照情報の表示

- コメントの表示/編集

- プログラムを1つ選択して検索する場合、デバイスコメントは[ツール] ⇒ [オプション] ⇒ "プロジェクト" ⇒ "デバイスコメント参照/反映先"で指定されたコメントが表示されます。(→ デバイスコメントについて)

- 複数のプログラムを選択して検索する場合、共通デバイスコメントが表示されます。

- プログラムを1つ選択して検索した状態でコメントを入力すると、デバイスコメントは下記のオプションで設定したデバイスコメントに反映されます。(→ デバイスコメントについて)

- [ツール] ⇒ [オプション] ⇒ "プロジェクト" ⇒ "デバイスコメント参照/反映先"
