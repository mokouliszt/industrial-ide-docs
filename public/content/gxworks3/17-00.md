# 17.0 データの収集/再現(機能概要)

## 17 データの収集/再現

メモリダンプ機能、データロギング機能、レコーディング機能を使用して、データを収集/再現する方法を説明します。

データを収集/再現するための機能を示します。

| 機能名 | 機能概要 | 参照 |
|---|---|---|
| メモリダンプ | CPUユニットのデバイスの値を任意のタイミングでパソコンへ保存する。 | → メモリダンプ機能→ MELSEC iQ-RシーケンサCPUユニットユーザーズマニュアル→ MELSEC iQ-R CPUユニットユーザーズマニュアル(応用編)→ MELSEC iQ-F FX5ユーザーズマニュアル(応用編) |
| データロギング | CPUユニットのデータを指定した間隔または任意のタイミングで収集する。 | → データロギング機能→ MELSEC MXコントローラMX-Rモデルユーザーズマニュアル→ MELSEC MXコントローラMX-Fモデルユーザーズマニュアル→ MELSEC iQ-RシーケンサCPUユニットユーザーズマニュアル→ MELSEC iQ-RプロセスCPUユニットユーザーズマニュアル→ MELSEC iQ-R CPUユニットユーザーズマニュアル(応用編)→ MELSEC iQ-F FX5ユーザーズマニュアル(応用編) |
| レコーディング | 下記いずれかを使用して、レコーディング対象として指定したデータを管理CPUユニットから収集し、保存する。 / MX-Rモデルの内蔵レコーディング機能 / レコーダユニット / カメラレコーダユニット | → レコーディング機能→ MELSEC MXコントローラMX-Rモデルユーザーズマニュアル→ MELSEC iQ-Rシステムレコーダユーザーズマニュアル(応用編)→ MELSEC iQ-Rシステムレコーダ連動レコーディング機能リファレンスマニュアル |
| オフラインモニタ | シーケンサから収集したデータをGX Works3のモニタ画面で確認する。 また、GX LogViewerで波形表示して確認する。 | → 収集したデータをプログラムエディタ上で確認する→ GX LogViewer Version 1オペレーティングマニュアル |
