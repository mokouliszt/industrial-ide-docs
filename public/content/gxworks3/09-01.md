# 9.1 デバイスおよびラベルの検索/置換

プログラム中のデバイスまたはラベルを検索/置換します。

対応エディタ: ラダーエディタ*1、STエディタ、FBD/LDエディタ*1*2、SFC図エディタ*1、ラベルエディタ

*1下記のメニューにより表示されている割付デバイスは、検索/置換の対象外です。[表示] ⇒ [デバイス表示]*2インラインST部品のプレビュー表示エリアは対応していません。デバイス名で検索した場合、そのデバイスが割り付けられているグローバルラベルエディタも検索されます。

画面表示

[検索/置換] ⇒ [デバイス/ラベル検索]/[デバイス/ラベル置換]を選択します。

![図(元マニュアル p.559)](images/gxworks3/09-01-01.webp)

**操作手順**

各項目を設定し、[次を検索]ボタンをクリックします。

| 項目 | | | 内容 |
|---|---|---|---|
| 検索/置換オプ ション | 検索 | デバイス点数 | "検索デバイス/ラベル"から、何点分を検索/置換するか入力します。 "検索デバイス/ラベル"/"置換デバイス/ラベル"がラベルの場合は、1を入力します。 例)検索デバイスをX50、置換デバイスをX100、デバイス点数を3、入力値を10進で設定して置換した場 合 X50 → X100、X51 → X101、X52 → X102に置換されます。 |
| 検索/置換オプ ション | 検索 | 桁付 | 入力されたデバイスと、入力されたデバイスが含まれる桁付のビットデバイスを検索対象とする場合に チェックを入れます。 |
| 検索/置換オプ ション | 検索 | 複数ワード | 入力されたデバイスと、入力されたデバイスが含まれるダブルワード形式のワードデバイスを検索対象 とする場合にチェックを入れます。 |
| 検索/置換オプ ション | 検索 | FB/構造体の要素 で部分一致*1 | プログラム本体で使用しているファンクションブロックの引数*2、ファンクションの引数*2、および構造 体メンバも検索する場合にチェックを入れます。 |
| 検索/置換オプ ション | 置換 | コメント | "検索デバイス/ラベル"のデバイスコメントまたはラベルコメントを"置換デバイス/ラベル"にコピーする か、移動するかを設定します。*3(ステップリレー(S)とSFC移行デバイス(TR)は対象外です。) |

*1本項目にチェックを入れていても、"検索対象"の"FBインスタンスの入出力ラベルを含む"を選択していない場合、FBインスタンスの入出力ラベルは検索/置換されません。*2クラスが"VAR_INPUT"、"VAR_OUTPUT"、"VAR_OUTPUT_RETAIN"、"VAR_IN_OUT"、"VAR_PUBLIC"、または"VAR_PUBLIC_RETAIN"のラベルが対象です。*3"検索デバイス/ラベル"のラベルコメントは、"置換デバイス/ラベル"のデバイスコメントにのみコピー/移動できます。ラベルコメントにはコピー/移動できません。

ラベル名を入力するときに、ラベルの候補が表示されます。

下記のいずれかの操作によって、ラベルの候補を表示することもできます。

- 構造体型のラベル名またはインスタンス名の後ろに"."を入力する

- [Ctrl]+[Space]を押すさらに、下記のオプションを"はい"に設定すると、共通デバイスコメントおよびラベルコメントを入力することで、それらのコメントが設定されているデバイスおよびラベルを候補として表示できます。

- [ツール] ⇒ [オプション] ⇒ "編集" ⇒ "命令/デバイス/ラベル候補表示" ⇒ "候補表示設定" ⇒ "デバイスコメント入力時に対応するデバイスを表示する"

下記のメニューを使用することで、過去に検索/置換した位置にカーソルを戻したり進めたりできます。

- [検索/置換] ⇒ [前に戻る]/[次に進む]

"検索対象"の"FBインスタンスの入出力ラベルを含む"を選択すると、ラダープログラムおよびFBD/LDプログラムで使用しているFBインスタンスの入出力ラベルも検索/置換されます。

注意事項

- 配列型ラベルの配列要素配列要素の数字の前に付く0は無視して検索されます。

例)"Label1[1]"で検索すると、"Label1[1]"と"Label1[001]"のどちらも検索されます。

ただし、下記のように配列要素に数字以外の文字/記号が含まれている場合、配列要素が完全に一致しているラベルのみ検索されます。

- Label1[K001]

- Label1[001,002,K003]

- stLabel1[001].bLabel1[K002]

- bLabel1[+001]

- bLabel1[wLabel1[001]]

### デバイスの検索例

#### ・検索オプション • 検索オプション: なし

| デバイス指定例 | 検索対象例 |
|---|---|
| M0 | M0、K4M0、M0Z0、K4M0Z0 |
| K4M0 | K4M0、K4M0Z0 |
| D0 | D0、D0Z0、D0.1 |
| D0.1 | D0.1 |
| J1¥B0 | J1¥B0、J1¥B0Z0、J1Z0¥B0、J1Z0¥B0Z0、J1¥K4B0、J1¥K4B0Z0、J1Z0¥K4B0、J1Z0¥K4B0Z0 |

- 検索オプション: 桁付

| デバイス指定例 | 検索対象例 |
|---|---|
| X0～X3 | K1X0 |
| X0～X0F | K4X0 |
| X0～X1F | K8X0 |
| X0Z0 | X0Z0、K1X0Z0、K4X0Z0、K8X0Z0 |

- 検索オプション: 複数ワード

| デバイス指定例 | 検索対象例 |
|---|---|
| D0～D1 | DMOV K1 D0、EMOV E1 D0、DMOV K1 @D0 |
| D0～D9 | BMOV D0 D100 K10 |
| J1¥W0～J1¥W1 | DMOV K1 J1¥W0 |
| @D0～@D1 | DMOV K1 @D0 |
| T0～T1 | DMOV K1 T0 |
| D0～D1 | D0:D、D0:DU、D0:E |
| D0～D3 | D0:ED |

#### ・デバイス型指定子付きデバイス表記(STエディタ、FBD/LDエディタ、SFC図エディタのみ)

| デバイス指定例 | 検索対象例 | 備考 |
|---|---|---|
| D100 | D100 := 1; D100:D := 1; D100:E := 0.1; | デバイス型指定子に関係なく検索されます。 |
| D100:D | D100 := 1; D100:D := 1; D100:E := 0.1; | 指定されたデバイス型指定子付きのデバイスのみが検索されます。 |
