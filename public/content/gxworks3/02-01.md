# 2.1 起動と終了

GX Works3の起動/終了の操作方法を説明します。

### 起動

操作手順

Windowsのスタートで"MELSOFT"からGX Works3を起動します。

- 重要なお知らせのメッセージ下記の条件を満たす場合、GX Works3の起動時に「重要なお知らせがあります。」と記載されたメッセージが表示されることがあります。

- Version 1.012N以降のMELSOFT Update Managerがインストールされている。

- パソコンをインターネットに接続している。

このメッセージが表示された場合は、内容を確認し、必要に応じて処置を行ってください。

- MELSOFTカスタマーエクスペリエンス向上プログラムGX Works3起動時、MELSOFTカスタマーエクスペリエンス向上プログラムへの参加を確認する画面が表示されます。(参加を確認済みの場合は表示されません。)

参加いただいた場合、お客様の使用状況に関する情報を収集し、MELSOFT製品の品質およびパフォーマンスの向上に活用させていただきます。

使用状況に関する情報の収集にご協力ください。

参加を確認した後、下記の操作により参加または不参加を変更できます。

操作手順1. [ヘルプ] ⇒ [MELSOFTカスタマーエクスペリエンス向上プログラム]を選択します。2.表示された画面で下記のいずれかの項目を選択し、[OK]ボタンをクリックします。

- MELSOFTカスタマーエクスペリエンス向上プログラムに参加します。

- 参加しません。

- 表示位置とウィンドウサイズを指定してGX Works3を起動するコマンドラインに下記のオプションを指定することで、GX Works3起動時の表示位置とウィンドウサイズを指定できます。

また、GX Works3起動後にオフラインモニタを開始することもできます。

- 全画面表示で起動する。

-WINDOWSIZE

- 表示位置(left*1、top*2)とウィンドウサイズ(width、height)を指定して起動する。

-WINDOWSIZE=left,top,width,height

- 起動後、オフラインモニタで使用するレコーディングファイルの選択画面を表示する。

-OFFLINEMONITOR

- 起動後、ファイルのパス(path)に指定したレコーディングファイルのオフラインモニタを開始する。

-OFFLINEMONITOR=path

*1 「left」には、メインディスプレイの左上を座標(0,0)とし、右方向を正とした整数値を入力してください。0を指定した場合、Windowsの設定によりウィンドウ枠が透過表示されるため、画面左側に数ピクセル分の隙間が発生します。(Windowsの設定でハイコントラストをオンにすると、隙間は発生しません。)*2 「top」には、メインディスプレイの左上を座標(0,0)とし、下方向を正とした整数値を入力してください。

例コマンドプロンプトからGX Works3を起動する場合、下記のように「gxw3」の後ろにオプションを指定します。

なお、カレントディレクトリには、GX Works3のインストールフォルダーを指定してください。

- 全画面表示で起動する。

gxw3 "-WINDOWSIZE"

- 指定した表示位置とウィンドウサイズで起動し、GX Works3プロジェクト(D:\Files\Project1.gx3)を開く。

gxw3 "-WINDOWSIZE=10,20,800,600" "D:\Files\Project1.gx3"

- 起動後、オフラインモニタで使用するレコーディングファイルの選択画面を表示する。

gxw3 "-OFFLINEMONITOR"

- 起動後、レコーディングファイル(D:\Files\record.melrc)のオフラインモニタを開始する。

gxw3 "-OFFLINEMONITOR=D:\Files\record.melrc"

- 全画面表示で起動し、レコーディングファイル(D:\Files\record.melrc)のオフラインモニタを開始する。

gxw3 "-WINDOWSIZE" "-OFFLINEMONITOR=D:\Files\record.melrc"

例ショートカットからGX Works3を起動する場合、ショートカットのプロパティの"リンク先"に下記のようにオプションを指定します。

- 全画面表示で起動する。

"C:\Program Files (x86)\MELSOFT\GPPW3\GXW3.exe" "-WINDOWSIZE"

- 指定した表示位置とウィンドウサイズで起動し、GX Works3プロジェクト(D:\Files\Project1.gx3)を開く。

"C:\Program Files (x86)\MELSOFT\GPPW3\GXW3.exe" "-WINDOWSIZE=100,50,1200,900" "D:\Files\Project1.gx3"

- 起動後、オフラインモニタで使用するレコーディングファイルの選択画面を表示する。

"C:\Program Files (x86)\MELSOFT\GPPW3\GXW3.exe" "-OFFLINEMONITOR"

- 起動後、レコーディングファイル(D:\Files\record.melrc)のオフラインモニタを開始する。

"C:\Program Files (x86)\MELSOFT\GPPW3\GXW3.exe" "-OFFLINEMONITOR=D:\Files\record.melrc"

- 全画面表示で起動し、レコーディングファイル(D:\Files\record.melrc)のオフラインモニタを開始する。

"C:\Program Files (x86)\MELSOFT\GPPW3\GXW3.exe" "-WINDOWSIZE" "-OFFLINEMONITOR=D:\Files\record.melrc"

### 終了

操作手順

[プロジェクト] ⇒ [GX Works3の終了]を選択します。

MELSOFT Navigatorからも起動/終了できます。
