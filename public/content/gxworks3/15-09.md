# 15.9 データの不正な閲覧/改ざんの防止(セキュリティバージョン)

プロジェクト内のデータの不正な閲覧および改ざんを防止するには、セキュリティバージョン機能を使用します。

本機能を使用することで、脆弱性に対するリスクを低減できます。

MXコントローラでは対応していません。

## セキュリティバージョンについて

セキュリティバージョンには"1"または"2"があります。

セキュリティバージョンを"2"に設定すると、プロジェクトパスワードを登録できるようになります。プロジェクトパスワードを登録することで、プロジェクトおよびCPUユニット内のデータに対して操作を制限できます。

また、セキュリティバージョンを"2"に設定すると、データの操作時にデータが改ざんされていないか自動でチェックされます。

### プロジェクト認証

下記に対して操作または機能を実行するときに、プロジェクトパスワードの認証が必要になります。

- セキュリティバージョンを"2"に設定したプロジェクト

- セキュリティバージョンを"2"に設定したプロジェクトからシーケンサに書き込まれたデータ対象となる機能および操作を示します。

| 機能/操作 | 備考 |
|---|---|
| プロジェクトを開く | ― |
| プロジェクト照合 | ― |
| シーケンサからの読出し | ― |
| シーケンサとの照合 | ― |
| SDメモリーカードからの読出し | ― |
| システムパラメータの流用 | ― |
| ユニットパラメータ(安全通信設定)の流用 | ― |
| レコーディングファイルの読出し | レコーディングファイルを読み出すとき、"レコーディングファイルの読出し"画面で"プロジェクト データを読出す"にチェックを入れていると認証画面が表示されます。 |
| オフラインモニタの開始 | レコーディングファイルをモニタするとき、セキュリティバージョンが"2"のプロジェクトが開かれた 場合に認証画面が表示されます。 |

プロジェクトパスワードは、プロジェクトのセキュリティバージョンを"2"に設定するときに登録できます。(→ 862ページ セキュリティバージョンを"1"から"2"に変更する場合)

- モーションユニットデータの操作セキュリティバージョンが"2"に設定されたモーションユニットデータに対する操作の場合、プロジェクトパスワードの認証が必要になります。

対象となる操作を示します。

| 操作 | 備考 |
|---|---|
| ユニット変更 | ― |
| モーション制御設定機能の起動 | ― |
| モーションユニットへのデータの書込み/読出し | 詳細はモーション制御設定機能のヘルプを参照してください。 |
| モーションユニットデータのインポート/エクス ポート | 詳細はモーション制御設定機能のヘルプを参照してください。 |

同一のモーションユニットのデータであれば、プロジェクトパスワードが変更されない限り、1度認証すればそれ以降の認証は不要です。

また、プロジェクトのセキュリティバージョンが"2"で、プロジェクトパスワードおよび管理者パスワードがモーションユニットデータに設定されたパスワードと同一であれば、認証は不要です。

- ライブラリ操作ライブラリファイルを作成したプロジェクトのセキュリティバージョンが"2"の場合、下記の操作でライブラリ用のプロジェクトパスワードの認証が必要になります。

| ライブラリ操作 | 備考 |
|---|---|
| ライブラリをライブラリ一覧に登録 | ― |
| ライブラリの編集 | ライブラリ用のプロジェクトパスワードおよびプロジェクト用のプロジェクトパスワード両方の認証 が必要です。 |
| ライブラリ一覧の表示情報更新 | ライブラリ用のプロジェクトパスワードを認証していない場合のみ、認証画面が表示されます。 |
| ライブラリ部品の更新 | ライブラリ用のプロジェクトパスワードを認証していない場合のみ、認証画面が表示されます。 |
| ライブラリ部品の展開 | ライブラリ用のプロジェクトパスワードを認証していない場合のみ、認証画面が表示されます。 |
| ライブラリ部品の検索 | ライブラリ用のプロジェクトパスワードを認証していない場合のみ、認証画面が表示されます。 |

ライブラリ用のプロジェクトパスワードは、ライブラリファイルを作成するときに登録できます。(→ ライブラリファイルの作成)

### 注意事項

- セキュリティバージョンの変更セキュリティバージョンを変更すると、プロジェクト変更履歴が削除されます。

あらかじめ、プロジェクトデータを保存してから実行してください。

- セキュリティバージョンおよびプロジェクトパスワードの変更セキュリティバージョンおよびプロジェクトパスワードを変更すると、プロジェクトが未変換状態になります。

- その他のMELSOFT製品についてセキュリティバージョンが"2"のプロジェクトの場合、下記製品のバージョンによっては一部の機能が使用できない場合があります。

- CPUユニットロギング設定ツール

- GENESIS64

- GT Designer3

- GX LogViewer

- iQ AppPortal

- MELSOFT MX MESInterface-R

- MELSOFT Navigator

- モーション制御設定機能

- 高速データコミュニケーションユニット用ツール

- 高速データロガーユニット用ツール詳細は、各製品のマニュアルまたはヘルプを参照してください。
