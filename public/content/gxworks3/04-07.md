# 4.7 ユニット変更

本機能は、GX Works3プロジェクトに設定しているユニットの機種を変更するための機能です。

プロジェクトに設定しているユニットのユニット種別、ユニット形名、または局種別を、別のユニットのデータに変更できます。

### ユニット

下記のユニットにおいて変更できるユニット種別、ユニット形名、および局種別を示します。

#### 他号機のCPUユニット

| ユニット種別 | ユニット形名 |
|---|---|
| シーケンサCPU | R00CPU、R01CPU、R02CPU、R04CPU、R08CPU、R16CPU、R32CPU、R120CPU |
| プロセスCPU | R08PCPU、R16PCPU、R32PCPU、R120PCPU |
| 安全CPU | R08SFCPU、R16SFCPU、R32SFCPU、R120SFCPU |
| モーションCPU | R16MTCPU、R32MTCPU、R64MTCPU |
| NCCPU | R16NCCPU |
| ロボットCPU | R16RTCPU |

#### CC-Link IE内蔵Ethernetユニット

| ユニット種別 | ユニット形名 | 局種別 |
|---|---|---|
| 情報ユニット | RJ71EN71(E+CCIEC) | 管理局、通常局、拡張モード(管理局)、拡張モード(通常局) |
| | RJ71EN71(E+CCIEF) | マスタ局、ローカル局、サブマスタ局 |
| | RJ71EN71(E+E) | |

#### CPU拡張ユニット

| ユニット種別 | ユニット形名 | 局種別 |
|---|---|---|
| CPU拡張 | RJ71EN71(E+IEC) _ | 管理局、通常局、拡張モード(管理局)、拡張モード(通常局) |
| | RJ71EN71(E+IEF) _ | マスタ局、ローカル局、サブマスタ局 |

#### シンプルモーションユニット/モーションユニット(シンプルモーションモード)

| ユニット種別 | シリーズ | ユニット形名 |
|---|---|---|
| シンプルモーション | iQ-R*1*2 | RD77MS2、RD77MS4、RD77MS8、RD77MS16、RD77GF4、RD77GF8、RD77GF16、 RD77GF32 |
| | iQ-F*1 | FX5-40SSC-S、FX5-80SSC-S |
| モーションユニット(シンプル モーションモード)*3 | iQ-R*1 | RD78G4(S)、RD78G8(S)、RD78G16(S) |
| | iQ-F*1 | FX5-40SSC-G(S)、FX5-80SSC-G(S) |
| | iQ-L*1 | LD78G4(S)、LD78G16(S) |
| | L*4 | LD77MS2、LD77MS4、LD77MS16 |

*1シリーズの異なるユニットには変更できません。*2 RD77GF4、RD77GF8、RD77GF16、およびRD77GF32から、RD77MS2、RD77MS4、RD77MS8、およびRD77MS16には変更できません。*3シンプルモーションユニットには変更できません。*4 LD77MS2、LD77MS4、LD77MS16、LD78G4(S)、またはLD78G16(S)にのみ変更できます。

#### CC-Link IE TSN対応モーションユニット

| ユニット種別 | ユニット形名 |
|---|---|
| モーションユニット*1 | RD78G4*2、RD78G8*2、RD78G16*2、RD78G32*2、RD78G64*2、RD78GHV*3、RD78GHW*3 |

*1モーションユニットのデータのセキュリティバージョンが"2"の場合、プロジェクトパスワードの認証が必要なことがあります。ただし、Version 1.117X以前品ではユニットを変更できません。セキュリティバージョンの詳細は下記を参照してください。→ データの不正な閲覧/改ざんの防止(セキュリティバージョン)*2 RD78GHVおよびRD78GHWには変更できません。*3 RD78G4、RD78G8、RD78G16、RD78G32およびRD78G64には変更できません。

操作手順1.ナビゲーションウィンドウの"パラメータ""ユニット情報""(ユニット形名)"を選択します。2.右クリックショートカットメニュー[ユニット変更]を選択します。

ユニット変更画面が表示されます。3.ユニット変更画面の各項目を設定し、[OK]ボタンをクリックします。

注意事項

CC-Link IE内蔵EthernetユニットまたはCPU拡張ユニットを変更する場合 • 他号機のCPUユニットによって管理されているユニットは変更できません。

- ユニットを変更後、ナビゲーションウィンドウ上に(必須設定未確認)が表示されます。

[ツール][パラメータチェック]を選択し、パラメータをチェックしてください。(→ パラメータのチェック)
