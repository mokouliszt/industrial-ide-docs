# 14.8 プログラムのスキャンタイムを測定する

ラダープログラム(Zoomを含む)およびSFCプログラムの任意の区間でスキャンタイムを測定するには、スキャンタイム測定機能を使用します。

本機能では、サブルーチンプログラムや割込みプログラム内の処理時間も測定できます。

スキャンタイム測定機能の詳細は、下記を参照してください。

→ MELSEC iQ-R シーケンサCPUユニットユーザーズマニュアル

→ MELSEC iQ-R CPUユニットユーザーズマニュアル(応用編)

MXコントローラでは対応していません。

画面表示

[デバッグ] ⇒ [スキャンタイム測定]を選択します。

例ラダーエディタがアクティブな状態で表示した場合

![図(元マニュアル p.791)](images/gxworks3/14-08-01.webp)

操作手順1.各項目を設定します。

| 項目 | | | 内容 |
|---|---|---|---|
| 測定範囲 | プログラム名 | | 現在開いているプログラム名が表示されます。 |
| | 開始*1 | プログラムブロック名 | プログラムブロック名が表示されます。 |
| | | ステップNo. | 開始ステップを設定します。 |
| | 終了*1 | プログラムブロック名 | プログラムブロック名が表示されます。 |
| | | ステップNo. | 終了ステップを設定します。 |
| | ブロック名*2 | | SFCブロック名が表示されます。 |
| | ステップ名*2 | | 開始ステップ名を設定します。 |

*1ラダーエディタがアクティブな状態、またはSFC図上で動作出力を選択している場合に表示されます。*2 SFC図エディタがアクティブな状態、かつ動作出力を選択していない場合に表示されます。2. [開始]ボタンをクリックします。

初回値、現在値、最大値、最小値が測定されます。

計測時間は0.001msから測定できます。

- ラダーエディタで回路ブロックを選択したまま"スキャンタイム測定"画面を起動すると、選択した範囲が"

ステップNo."に自動で入力されます。

- ラダープログラムの場合、開始ステップ、終了ステップに指定したステップが回路の途中に存在する場合は、選択した範囲を含む回路ブロック単位で設定されます。

注意事項

- 安全プログラムの場合は、スキャンタイムを測定できません。

- 指定する開始ステップ、終了ステップによっては、スキャンタイムを測定できません。

スキャンタイムを測定できない場合の詳細は、下記を参照してください。

→ MELSEC iQ-R シーケンサCPUユニットユーザーズマニュアル

→ MELSEC iQ-R CPUユニットユーザーズマニュアル(応用編)
