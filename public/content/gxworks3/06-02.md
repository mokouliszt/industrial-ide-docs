# 6.2 プログラムの実行順序/実行タイプの設定

プログラムの実行順序および実行タイプの設定方法を説明します。

## プログラムの実行順序の設定

プログラムファイル内のプログラムブロックの実行順序を設定します。

SFCプログラムはブロックNo.に従って実行されます。ブロックNo.は、SFCブロックリストで確認してください。

SFCプログラムの実行順序を変更する場合は、SFCブロックの"プロパティ"画面でブロックNo.を変更してください。

プロセス制御拡張を有効にしたスキャン実行タイプのプログラムファイルの実行順序を設定する場合は、下記を参照してください。

→ 実行順序

画面表示

- [変換] ⇒ [プログラムファイル設定]を選択します。

- ナビゲーションウィンドウでプログラムファイルを選択後、右クリック ⇒ ショートカットメニュー[プログラムファイル設定]を選択します。

![図(元マニュアル p.360)](images/gxworks3/06-02-01.webp)

**操作手順**

1.プログラムファイル名を選択します。

2.プログラムブロックの実行する順序を設定し、[OK]ボタンをクリックします。

ナビゲーションウィンドウ内のプログラムファイルを選択し、右クリック ⇒ ショートカットメニュー[ソート] ⇒ [実行順]を選択すると、プログラムファイル設定で指定した実行順序でソートできます。

### ワークシートの実行順序の設定

プログラム部品内の複数のプログラム本体がある場合、プログラム本体の実行順序を設定します。

STおよびFBD/LDのプログラム部品で対応しています。

画面表示

- [変換] ⇒ [ワークシート実行順設定]を選択します。

- ナビゲーションウィンドウでプログラム部品を選択後、右クリック ⇒ ショートカットメニュー[ワークシート実行順設定]

を選択します。

![図(元マニュアル p.361)](images/gxworks3/06-02-02.webp)

**操作手順**

1.プログラム本体を選択します。

2.プログラムの実行順序を設定し、[OK]ボタンをクリックします。

ナビゲーションウィンドウ内のプログラム部品を選択し、右クリック ⇒ ショートカットメニュー[ソート] ⇒ [

実行順]を選択すると、ワークシート実行順設定で指定した実行順序でソートできます。

## プログラムの実行タイプの変更

プログラムの実行タイプは初期、スキャン、定周期、イベント、待機、実行タイプ指定なしを指定できます。

詳細は、下記を参照してください。

→ MELSEC MXコントローラ MX-Rモデルユーザーズマニュアル

→ MELSEC MXコントローラ MX-Fモデルユーザーズマニュアル

→ MELSEC iQ-R シーケンサCPUユニットユーザーズマニュアル

→ MELSEC iQ-R プロセスCPUユニットユーザーズマニュアル

→ MELSEC iQ-R SIL2プロセスCPUユニットユーザーズマニュアル

→ MELSEC iQ-R CPUユニットユーザーズマニュアル(応用編)

→ MELSEC iQ-F FX5ユーザーズマニュアル(応用編)

安全プログラムの場合は、定周期のみ指定できます。

### 設定方法

ナビゲーションウィンドウで、実行タイプを設定するプログラムを右クリック ⇒ ショートカットメニュー[プログラム登録]

から選択、またはドラッグ&ドロップすると、プログラムの実行タイプを設定できます。

設定した実行タイプは、"CPUパラメータ"の"プログラム設定"に反映されます。

## プロセス制御用FBD/LDプログラムの実行制御

## プロセス制御用FBD/LDプログラムの実行制御を設定する方法を説明します。

### 起動方法

プロセス制御用FBD/LDプログラムの起動方法には、タイマ起動と割込み起動(定周期)の2種類があり、プログラムの使用目的別に起動方法を選択します。

詳細は、下記を参照してください。

→ MELSEC iQ-R プログラミングマニュアル(プロセス制御FB/命令編)

### 実行順序

プロセス制御拡張を有効にしたスキャン実行タイプのプログラムファイルの実行順序は、"プログラムファイル設定"画面で設定します。

**操作手順**

1.下記いずれかの操作で"プログラムファイル設定"画面を表示します。

- [ツール] ⇒ [プロセス制御拡張] ⇒ [プログラムファイル設定]を選択する。

- プロセス制御拡張ツールバーの：(プログラムファイル設定)をクリックする。

- ナビゲーションウィンドウでプログラムファイルを選択し、右クリック ⇒ ショートカットメニュー[プログラムファイル設定]を選択する。

2.プログラムブロックを実行する順序を設定し、[OK]ボタンをクリックします。

- "プログラムファイル設定"画面の詳細は、下記を参照してください。

→ プログラムファイル設定画面

- ナビゲーションウィンドウ内のプログラムファイルを選択し、右クリック ⇒ ショートカットメニュー[

ソート] ⇒ [実行順]を選択すると、プログラムファイル設定で指定した実行順序でソートできます。

### 実行周期と位相

プロセス制御用FBD/LDプログラムの実行周期と位相は、下記の画面で設定します。

- プロセス制御用FBD/LDプログラムの"プロパティ"画面(→ プロパティ)

- "プログラムファイル設定"画面実行周期と位相の詳細は、下記を参照してください。

→ MELSEC iQ-R プログラミングマニュアル(プロセス制御FB/命令編)

本節では、"プログラムファイル設定"画面について説明します。

- プログラムファイル設定画面"プログラムファイル設定"画面では、プロセス制御拡張を有効にしたプログラムファイルに含まれるプログラムブロックの実行周期と位相を一括で設定できます。

画面表示

- [ツール] ⇒ [プロセス制御拡張] ⇒ [プログラムファイル設定]を選択します。

- プロセス制御拡張ツールバーの：(プログラムファイル設定)をクリックします。

- ナビゲーションウィンドウでプログラムファイルを選択後、右クリック ⇒ ショートカットメニュー[プログラムファイル設定]を選択します。

![図(元マニュアル p.363)](images/gxworks3/06-02-03.webp)

表示内容

| 項目 | 内容 |
|---|---|
| (1)実行順序 | プログラムブロックの実行順序が表示されます。 |
| (2)プログラムブロック名 | プログラムブロックの名称が表示されます。 |
| (3)実行周期(ヘッダー部) | 下記のオプションの値によって、表示内容が変更されます。• [ツール] ⇒ [オプション] ⇒ "プロセス制御拡張設定" ⇒ "プログラム実行" ⇒ "実行周期" |
| (4)実行周期 | 各プログラムの実行周期が表示されます。実行周期はプルダウンリストから変更できます。 |
| (5)位相 | 各プログラムの位相が表示されます。数値を入力することで、位相を変更できます。 |
| (6)実行タイミング | 各プログラムを実行するタイミングが表示されます。("*"がプログラムの実行を示します。) FBD/LD言語で作成したプログラムブロックに対してのみ、"*"が表示されます。 |
| (7)合計 | 実行タイミングごとに実行されるプログラムの合計("実行タイミング"に表示されている"*"の合計)が表示されます。プログラムブロックが存在しない場合、0が表示されます。 |
