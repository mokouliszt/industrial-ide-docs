# 18.0 ユニットの動作確認/変更(機能概要)

## 18 ユニットの動作確認/変更

GX Works3で各ユニットの動作を確認および変更できます。

| CPUユニット | | |
|---|---|---|
| 目的 | 機能名 | 参照 |
| CPUユニットの時計を設定する。 | 時計設定 | → CPUユニットの時計設定 |
| GX Works3から接続しているCPUユニットをRUN/ STOP/PAUSE/RESETする。 | リモート操作 | → リモート操作 |
| 二重化システム構成のCPUユニットの機能を実行する。 | 系切替え 運転モード変更 メモリコピー 他系起動待ち中の制御系強制起動 二重化機能ユニットの単体通信テスト A/B系設定 | → 二重化シーケンサ操作 |
| RnPSFCPUおよびRnSFCPUの機能を実行する | 安全動作モード切替え 安全データ同一性チェック 安全ユニット操作 安全バックアップ/リストア機能 | → 安全シーケンサ操作 |
| データメモリの使用状況を確認する。 | CPUメモリ操作 | → メモリ使用状況の確認 |
| データメモリの値をクリアする。 | 値のクリア | → メモリの初期化/ゼロクリア |
| ラベルメモリを読み出す。 読み出したラベルメモリを書き込む。 | ラベルメモリの読出し/書込み | → ラベルメモリの読出し/書込み |
| ユニットのファームウェアをアップデートする。 | ファームウェアアップデート | → ファームウェアアップデート |

| リモートヘッドユニット | | |
|---|---|---|
| 目的 | 機能名 | 参照 |

| 二重化システム構成のリモートヘッドユニットの機能 を実行する。 | 系切替え | → 二重化シーケンサ操作 |
| データメモリの使用状況を確認する。 | CPUメモリ操作 | → メモリ使用状況の確認 |
