# 12.1 接続先指定について

接続先指定での、パソコン側、シーケンサ側のI/Fや経由するネットワークなど、CPUユニットとアクセスするための通信経路の設定方法を説明します。

接続先の設定は最大128個作成できます。

プロジェクトの機種は、アクセスするCPUユニットの機種に合わせる必要があります。

### 接続先の設定を変更する

**画面の開き方**

接続先ウィンドウでデータを選択し、ダブルクリックします。

[オンライン] ⇒ [現在の接続先]を選択した場合、現在の接続先を変更します。

![図(元マニュアル p.677)](images/gxworks3/12-01-01.webp)

- 設定画面上でアンダーラインが付いている項目は、ダブルクリックして詳細設定が可能です。

- アイコンが黄色になっている項目は、設定済みを示します。

- MXコントローラおよびFX5CPUの場合、設定できない接続先経路のアイコンは表示されません。

- FX5CPUのプロジェクトにおいて、下記のオプションを"はい"に設定している場合、"接続先指定"画面が表示される前に"

簡易接続先設定"画面が表示されます。

- [ツール] ⇒ [オプション] ⇒ "オンライン" ⇒ "接続先指定" ⇒ "動作設定" ⇒ "簡易接続先設定画面を表示する"

**画面項目**

| 項目 | | 説明 |
|---|---|---|
| 他局指定 | 他局指定無し | パソコンに直接接続しているCPUユニットにアクセスする場合に指定します。 |
| 他局指定 | 他局(単一ネットワーク)*1 | 1種類のネットワーク(多階層システムを含む)を経由して、他局のCPUユニットにアクセスする場合に指定します。 Ethernet、CC-Link IEコントローラネットワーク、およびCC-Link IEフィールドネットワークは同種とみなされます。これらが混在しているシステムにアクセスする場合は、単一ネットワークを指定してください。 |
| 他局指定 | 他局(異種ネットワーク)*1 | 2種類のネットワークを介して他局のCPUユニットにアクセスする場合に指定します。 |
| ネットワーク通信経路 | 他局(異種ネットワーク)*1 | 他局アクセスするときに経由するネットワークのネットワーク種別、ネットワークNo.、局番、先頭I/Oを選択します。設定項目は、選択したネットワーク種別により異なります。 |
| 異種ネットワーク通信経路 | | アクセスするネットワークのネットワーク種別、ネットワークNo.、局番、先頭I/Oを選択します。設定項目は、選択したネットワーク種別により異なります。 |
| 対象システム | | / マルチCPU指定: マルチCPUシステムを構成しているときのアクセス先を指定します。 / 二重化CPU指定: 接続するCPUユニットの系を選択します。 |

*1自局指定の場合は、"他局指定無し"を選択します。

**注意事項**

- "接続先指定"画面で設定できない接続先にはアクセスできません。

**- MELSECNET/Hネットワークユニットを使用する場合の注意事項の関連項目**

→ MELSEC iQ-Rユニット構成マニュアル

### 接続先の設定を追加する

接続先の設定を新規に追加します。

**操作手順**

1. 接続先ウィンドウで右クリック ⇒ ショートカットメニュー[新しい接続先作成]を選択します。

2. 各項目を設定し、[OK]ボタンをクリックします。

### 接続先を切り替える

通常使用する接続先の設定を切り替えます。

**操作手順**

1. 接続先ウィンドウの"すべての接続先"の中から使用する接続先を選択します。

2. 右クリック ⇒ ショートカットメニュー[通常使う接続先として指定する]を選択します。

### 設定した接続経路のイメージ表示

[システムイメージ]ボタンをクリックすると、設定した接続経路がイラストで表示され、内容が確認できます。

**注意事項**

LHCPUでは対応していません。
