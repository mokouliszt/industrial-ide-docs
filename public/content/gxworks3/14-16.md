# 14.16 MXコントローラのモーション動作を確認する

下記の機能を使用して、MXコントローラのモーション動作を確認できます。

→ モーション共通モニタ

→ 軸モニタ

→ 位置データ履歴

## モーション共通モニタ

"モーション共通モニタ"画面では、軸および軸グループに依存しない演算周期などのCPUユニットの情報を確認できます。

画面表示

[オンライン] ⇒ [モーションモニタ] ⇒ [軸モニタ]/[位置データ履歴]を選択します。

![図(元マニュアル p.813)](images/gxworks3/14-16-01.webp)

表示内容

| 項目 | 内容 |
|---|---|
| (1)[モニタ項目選択]ボタン | "モニタ項目選択"画面を表示します。(→ モニタ項目選択) |
| (2)モニタ画面 | "モニタ項目選択"画面で選択したモニタ項目に対して、現在値が表示されます。 ビット型の場合は、現在値によってアイコンの色が切り替わります。 • TRUE: 緑/赤(モニタ項目によって異なります。) • FALSE: グレー |

### モニタ項目選択

"モニタ項目選択"画面では、"モーション共通モニタ"画面に表示するモニタ項目を追加/削除できます。

画面表示

"モーション共通モニタ"画面の[モニタ項目選択]ボタンをクリックします。

![図(元マニュアル p.814)](images/gxworks3/14-16-02.webp)

**操作手順**

- モニタ項目を追加する1."選択可能な項目"欄で、"モーション共通モニタ"画面に表示するモニタ項目を選択します。(複数選択可) 2. [追加]ボタンをクリックします。

モニタ項目が"選択可能な項目"欄から"モニタ項目及び表示順序"欄に移動します。

3. "モニタ項目及び表示順序"欄で、モニタ項目を選択します。

4. /：をクリックし、モニタ項目を任意の位置に移動します。

5. [OK]ボタンをクリックします。

- モニタ項目を削除する1."モニタ項目及び表示順序"欄で、"モーション共通モニタ"画面から削除するモニタ項目を選択します。(複数選択可) 2. [削除]ボタンをクリックします。

モニタ項目が"モニタ項目及び表示順序"欄から"選択可能な項目"欄に移動します。

3. [OK]ボタンをクリックします。

- [デフォルトに戻す]ボタンをクリックすると、"選択可能な項目"欄および"モニタ項目及び表示順序"欄の表示内容がデフォルトの内容に戻ります。

- モニタ項目をダブルクリックすることでも、モニタ項目を"選択可能な項目"欄または"モニタ項目及び表示順序"欄に移動できます。

## 軸モニタ

"軸モニタ"画面では、MXコントローラに書き込まれているすべての運転軸の現在値およびエラーコードを一括で確認できます。

画面表示

[オンライン] ⇒ [モーションモニタ] ⇒ [軸モニタ]を選択します。

![図(元マニュアル p.815)](images/gxworks3/14-16-03.webp)

表示内容

| 項目 | | | 内容 |
|---|---|---|---|
| (1)ツールバー | モニタ状態 | | モニタ状態がアイコンで表示されます。 • モニタ中: アイコン内の虫眼鏡が動きます。 • モニタ停止中: アイコン内の虫眼鏡が停止します。 |
| (1)ツールバー | | 軸モニタ | "軸モニタ"画面を表示します。 |
| (1)ツールバー | | 位置データ履歴 | "位置データ履歴"画面を表示します。(→ 位置データ履歴) |
| (1)ツールバー | | モニタ開始 | モニタを開始します。 |
| (1)ツールバー | | モニタ停止 | モニタを停止します。 |
| (2)[分割]ボタン | | モニタ停止 | モニタ画面を分割します。 |
| (3)フォントサイズ | | | フォントサイズを変更します。 • 最大値: 26pt • 最小値: 9pt |
| (4)モニタ画面 | | | "モニタ項目選択"画面で選択したモニタ項目に対して、"モニタ軸選択"画面で選択した軸の現在値が表示さ れます。 モニタ停止中、行または列をドラッグすることで、モニタ項目および軸を並べ替えることができます。 列幅を調整した場合、"軸モニタ"画面を閉じるまで列幅が保持されます。 |
| (5)分割バー | | | モニタ画面を上下に分割するバーです。 [分割]ボタンをクリックすると表示されます。 |

![図(元マニュアル p.816)](images/gxworks3/14-16-04.webp)

*1画面を分割していない場合、下画面用の項目はグレー表示され、設定を変更できません。

- 軸モニタを使用するには、軸データをMXコントローラに書き込む必要があります。

- [分割]ボタンをクリックするとモニタ画面が上下に分割され、実ドライブ軸と仮想エンコーダ軸、軸と軸グループなどの異なるモニタ種別を同時にモニタできます。

注意事項

モニタ画面の現在値は、モニタ中の場合にのみ更新されます。

モニタを停止した場合、停止直前の現在値が表示されます。

また、モニタ停止中に通信エラーが発生した場合、現在値には"--"が表示されます。

### モニタ軸選択

"モニタ軸選択"画面では、"軸モニタ"画面に表示する軸を追加/削除できます。

画面表示

"軸モニタ"画面の[モニタ軸選択]ボタンをクリックします。

例"軸モニタ"画面で"モニタ種別"を"実ドライブ軸"に設定した場合

![図(元マニュアル p.818)](images/gxworks3/14-16-06.webp)

**操作手順**

- 軸を追加する1."選択可能な軸No."欄で、"軸モニタ"画面に表示する軸/軸グループを選択します。(複数選択可) 2. [追加]ボタンをクリックします。

軸/軸グループが"選択可能な軸No."欄から"モニタ軸及び表示順序"欄に移動します。

3. "モニタ軸及び表示順序"欄で、軸/軸グループを選択します。

4. /：をクリックし、軸/軸グループを任意の位置に移動します。

5. [OK]ボタンをクリックします。

- 軸を削除する1."モニタ軸及び表示順序"欄で、"軸モニタ"画面から削除する軸/軸グループを選択します。(複数選択可) 2. [削除]ボタンをクリックします。

軸/軸グループが"モニタ軸及び表示順序"欄から"選択可能な軸No."欄に移動します。

3. [OK]ボタンをクリックします。

## 位置データ履歴

"位置データ履歴"画面では、軸の位置データ履歴(各軸の現在位置、電源ON/OFF、および原点復帰時の各現在値)を確認できます。

画面表示

[オンライン] ⇒ [モーションモニタ] ⇒ [位置データ履歴]を選択します。

![図(元マニュアル p.819)](images/gxworks3/14-16-07.webp)

表示内容

| 項目 | | | 内容 |
|---|---|---|---|
| (1)ツールバー | モニタ状態 | | モニタ状態がアイコンで表示されます。 • モニタ中: アイコン内の虫眼鏡が動きます。 • モニタ停止中: アイコン内の虫眼鏡が停止します。 |
| (1)ツールバー | | 軸モニタ | "軸モニタ"画面を表示します。(→ 軸モニタ) |
| (1)ツールバー | | 位置データ履歴 | "位置データ履歴"画面を表示します。 |
| (1)ツールバー | | モニタ開始 | モニタを開始します。 |
| (1)ツールバー | | モニタ停止 | モニタを停止します。 |
| (2)モニタ種別 | | モニタ停止 | モニタする軸種別を選択します。 • 実ドライブ軸(デフォルト) • 仮想ドライブ軸/仮想連結軸 • 実エンコーダ軸 • 仮想エンコーダ軸 |
| (3)軸 | | | モニタする軸を選択します。 プルダウンリストには、接続中の軸のうち、位置データの履歴が存在する軸No.と軸名が表示されます。 |
| (4)モニタ値 | | | "軸"で選択した軸の各現在値が一覧で表示されます。 |
| (5)原点復帰データ | | | "軸"で選択した軸の原点復帰時の各現在値*1が一覧で表示されます。 |
| (6)絶対位置データ | | | "軸"で選択した軸における、最大過去10回分(合計20回)の電源ON/OFF時の各現在値*1が一覧で表示されま す。 |
| (7)[最新の状態に更新]ボタン | | | 接続先のMXコントローラから位置データ履歴ファイルを読み出し、"位置データ履歴"画面を更新します。 |
| (8)[CSVファイル作成]ボタン | | | 位置データ履歴をCSVファイルに保存します。 |
| (9)"モーション共通モニタ"画面 | | | MXコントローラの"モーション共通モニタ"画面が表示されます。(→ モーション共通モニタ) |

*1[最新の状態に更新]ボタンをクリックすると、各現在値が更新されます。

### データが存在しない場合

各データが存在しない場合の表示内容を示します。

![図(元マニュアル p.820)](images/gxworks3/14-16-08.webp)

下記のいずれかの場合、すべての時刻およびデータの値が0になります。

- MXコントローラから原点復帰/絶対位置データを取得したとき、データが消失した。

- インクリメンタルシステムによりバックアップされなかった。

インクリメンタルシステムの詳細は、下記を参照してください。

→ MELSEC MXコントローラ MX-Rモデルユーザーズマニュアル

→ MELSEC MXコントローラ MX-Fモデルユーザーズマニュアル
