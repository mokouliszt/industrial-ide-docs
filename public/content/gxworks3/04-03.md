# 4.3 モーション設定

MXコントローラのモーションシステムを動作させるためには、MXコントローラ内蔵のモーション設定を設定する必要があります。

ナビゲーションウィンドウに表示されるモーション設定について説明します。

| 項目 | 内容 | 参照 |
|---|---|---|
| (1)CC-Link IE TSN構成*1 | CC-Link IE TSN構成ウィンドウへのショートカットです。 | → ネットワーク構成、相手機器の設定 |
| (2)モーションシステム設定 | 演算周期や全軸緊急停止信号などのシステムに関する項目を設定します。 | → モーションシステム設定 |
| (3)軸(軸データ) | ネットワーク設定以外の軸動作に関する項目を設定します。• 実ドライブ軸• 実エンコーダ軸• 仮想ドライブ軸• 仮想エンコーダ軸• 仮想連結軸 | → 軸データの作成→ 軸パラメータの設定→ 電子ギアの設定 |
| (4)軸グループ | 軸データをグループ化する際に必要な項目を設定します。 | → 軸グループの作成→ 軸グループ設定 |
| (5)モーション入出力データ | モーション制御で使用する入出力データを設定します。 | → モーション入出力データの作成 |
| (6)演算プロファイル | モーション制御で使用する演算プロファイルを設定します。• カムデータ• ロータリカッター• デジタルカムスイッチ• 多軸位置決めデータ | → 演算プロファイルの作成 |
| (7)アドバンスト同期制御 | アドバンスト同期制御で使用する設定データを作成します。• アドバンスト同期入力設定データ• アドバンスト同期出力設定データ | → アドバンスト同期入力設定/出力設定データの作成→ アドバンスト同期制御の設定 |

*1："CC-Link IE TSN構成"は、下記のオプションを"はい"に設定した場合に表示されます。・ [ツール] ⇒ [オプション] ⇒ "プロジェクト" ⇒ "ナビゲーション" ⇒ "表示設定" ⇒ "CC-Link IE TSN構成を表示する"MXF100-ECの場合、このオプションは設定できません。ナビゲーションウィンドウに表示される状態表示アイコンの詳細は、下記を参照してください。

→ 状態表示アイコン
