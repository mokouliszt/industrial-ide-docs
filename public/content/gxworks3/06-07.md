# 6.7 FBプロパティの初期値を設定

## FBプロパティの表示/設定

FBプロパティの初期値を"FBプロパティ"ウィンドウで表示および設定します。

タグFBの場合、タグデータの構造体メンバのラベル初期値も表示および設定できます。

"FBプロパティ"ウィンドウの初期値と各ラベルのラベルエディタ上の初期値は同期します。

"FBプロパティ"ウィンドウでの初期値の変更はプロジェクトデータに対して行います。実際のCPUユニット内の初期値を変更する場合はウォッチ機能を使用してください。

### 初期値の変更

"初期値"欄の値を変更します。

初期値がメーカ定義の初期値と異なる場合*1、"FBプロパティ"ウィンドウ上では太字で表示されます。

"FBプロパティ"ウィンドウで入力した初期値がメーカ定義の初期値と同じ場合、ラベルエディタ上では対応するラベルの初期値が空欄になります。

*1"FBプロパティ"ウィンドウの初期値とメーカ定義の初期値の比較は文字列として比較します。(例: 1.0と1.00は異なる、FALSEと0は異なると判断する)

注意事項

"ラベル初期値設定"画面で設定したFBインスタンスの初期値は"FBプロパティ"ウィンドウでは変更できません。

"ラベル初期値設定"画面でのみ変更できます。(→ 初期値の詳細設定)

## FBプロパティページの表示

FBプロパティページは、FBプロパティが設定内容ごとにわかりやすく分類された画面です。FBプロパティの初期値の確認および変更、また折れ線に関する設定をグラフで確認できます。

FBプロパティの詳細は、下記を参照してください。

→ FBプロパティの表示/設定

FBプロパティページでは、下記のファンクションブロックに対応しています。

| 分類 | ファンクションブロック |

| タグFB | ステータスタグ |
| タグFB | アラームタグ |
| タグFB | メッセージタグ |
| 汎用プロセスFB*1 | M+P FG _ |
| 汎用プロセスFB*1 | M+P IFG _ |
| ユーザ定義タグFB | ― |

*1"ラベル初期値設定"画面で初期値を設定している場合、"FBプロパティページ"画面は表示できません。

### フォーマット

CSVファイルのフォーマットを示します。

- エクスポートしたCSVファイル保存したCSVファイルを表計算ソフトで開くと、下記のように表示されます。

![図(元マニュアル p.502)](images/gxworks3/06-07-26.webp)

(1): ヘッダー部

(2): データ部

CSVファイルの詳細なフォーマットを示します。

- ファイル形式は、Unicode(UTF-16 Little Endian BOMあり)です。

- 項目間の区切りは「¥t」(タブ)です。

- 各項目は「"」(ダブルクォーテーション) で囲まれます。

- 項目内に「"」(ダブルクォーテーション)を含む場合 、項目内の「"」(ダブルクォーテーション) は「""」(ダブルクォーテーション2個) で表示されます。

- 行の最後は改行が設定されます。改行コードはLFです。

- "FBプロパティ管理(オフライン)"画面/"FBプロパティ管理(オンライン)"画面の列タイトルと一致する文字列がヘッダー部

(1)に表示されます。

- ヘッダー部(1)の項目名は、GX Works3の表示言語でエクスポートされます。(→ 表示言語の切替え)

- "FBプロパティ管理(オフライン)"画面/"FBプロパティ管理(オンライン)"画面の"更新選択"列にチェックが入っている場合、

CSVファイルの"更新選択"列には1が表示されます。

- "FBプロパティ管理(オフライン)"画面/"FBプロパティ管理(オンライン)"画面の"更新選択"列にチェックが入っていない場合、CSVファイルの"更新選択"列には0が表示されます。

ただし、"ラベル初期値設定"画面でFBインスタンスの初期値を設定している場合は、2が表示されます。

- "FBプロパティ管理(オンライン)"画面から出力した場合は、下記のように現在値もエクスポートされます。

![図(元マニュアル p.502)](images/gxworks3/06-07-27.webp)

- インポートするCSVファイル下記のフォーマットに従って、エクスポートしたCSVファイルを編集してください。

- 改行コードが含まれる項目は「"」(ダブルクォーテーション)で囲んでください。

- ヘッダー部(1)は3行目に記載してください。

- ヘッダー部(1)はエクスポートしたCSVファイルのヘッダー部(1)と項目の名称および順番を一致させてください。

- 言語が異なるGX Works3でエクスポートしたCSVファイルからデータをインポートする場合、CSVファイル内のヘッダー部(1)の文字列をインポート先の"FBプロパティ管理(オフライン)"画面の列タイトルと一致させてください。

- "FBプロパティ管理(オフライン)"画面の"更新選択"列にチェックを入れる場合、CSVファイルの"更新選択"列に1を入力してください。チェックを入れない場合は、0を入力してください。

- "初期値"列にはGX Works3のラベルエディタの"初期値"列に入力できる値を入力してください。入力できない値はインポートされません。

- E列以降に列を追加することもできますが、その列はインポートされません。
