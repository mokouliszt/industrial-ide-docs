# 18.2 リモート操作

CPUユニットおよびリモートヘッドユニットのRUN/STOP/RESETスイッチをRUNの位置にしたまま、CPUユニットおよびリモートヘッドユニットの動作状態(RUN/STOP/PAUSE/RESET)を切り替える方法を説明します。

手の届かない場所にあるCPUユニットおよび制御盤内のCPUユニットを外部信号でRUN/STOP状態にする場合などに本機能を使用します。

リモート操作でRESETにする場合は、事前にCPUパラメータでリモートリセットを許可に設定する必要があります。

詳細は、下記を参照ください。

→ MELSEC MXコントローラ MX-Rモデルユーザーズマニュアル

→ MELSEC MXコントローラ MX-Fモデルユーザーズマニュアル

→ MELSEC iQ-R シーケンサCPUユニットユーザーズマニュアル

→ MELSEC iQ-R プロセスCPUユニットユーザーズマニュアル

→ MELSEC iQ-R SIL2プロセスCPUユニットユーザーズマニュアル

→ MELSEC iQ-R CPUユニットユーザーズマニュアル(応用編)

本節では、"リモート操作"画面から動作状態を切り替える方法を説明します。

画面表示

[オンライン] ⇒ [リモート操作]を選択します。

![図(元マニュアル p.939)](images/gxworks3/18-02-01.webp)

上記はRnCPUと接続している場合の画面です。

**操作手順**

各項目を設定し、[実行]ボタンをクリックします。

| 実行先指定 | 内容 |
|---|---|
| 現在局指定*1 | ・CPUユニットの場合 接続先で設定している局のみリモート操作をする場合に選択します。 ・二重化システム構成のリモートヘッドユニットの場合 • [オンライン] ⇒ [リモート操作]から起動した場合: 接続先で指定しているリモートヘッドユニットの系が対象となります。 • "CC IE Field診断"画面から起動した場合: 診断先として指定したリモートヘッドユニットの系が対象となります。 |
| 全局指定*2 | 接続先で設定している局と、指定したネットワーク上の全局に対してリモート操作をする場合に選択します。"対象ネットワー クNo. 指定"で、対象ネットワークを設定します。 |
| グループNo.指定*2 | 接続先で設定している局と、指定したネットワーク上の特定グループの各局に対してリモート操作をする場合に選択します。 "対象ネットワークNo. 指定"で、対象ネットワークを設定し、"グループNo. 指定"で、グループNo. を設定します。 CC-Link IEフィールドネットワークはグループNo.を持たないため、グループNo.を指定してのリモート操作はできません。 |
| 両系指定*2 | 接続先指定で指定した経路で、両系(制御系/待機系)にリモート操作を実行する場合に選択します。二重化システム構成のCPU ユニットで選択できます。 |

*1二重化システム構成のCPUユニットおよびリモートヘッドユニットでRESETする場合、"現在局指定/両系指定"と表示されます。*2MX-Fモデル、FX5CPU、およびリモートヘッドユニットでは対応していません。

### リモート操作とRUN/STOPスイッチの動作について

リモート操作、RUN/STOPスイッチ、またはリモートRUN/PAUSE接点によるCPUユニットへの操作が異なる場合、下記の優先順位に従ってCPUユニットは動作します。

STOP1

PAUSE2

RUN3

CPUユニットのRUN/STOP、またはリモートRUN/STOP接点によるリモート操作実行後の動作を示します。

| リモート操作 | CPUユニットのスイッチ | | リモートRUN/STOP接点がON (CPUユニットはSTOP状態) |
|---|---|---|---|
| | RUN | STOP | |
| RUN | RUN | STOP | STOP |
| STOP | STOP | STOP | STOP |
| PAUSE*1 | PAUSE | STOP | STOP |
| RESET*2 | 操作不可*3 | RESET | RESET |

*1リモートヘッドユニットでは対応していません。*2"CPUパラメータ"の"動作関連設定"でリモートリセットを許可する必要があります。*3リモート操作でSTOPにした場合は操作可能です。

- 二重化システム構成のCPUユニット使用時の動作RnPCPU(二重化モード)は、運転モードにより動作が異なります。

RnPSFCPUの運転モードはバックアップモードで動作します。

| 運転モード | 動作 |
|---|---|
| バックアップモード | ・STOPの実行 両系(制御系→待機系の順番)に実行します。 ・RUN、PAUSEの実行 両系(待機系→制御系の順番)に実行します。 ・RESETの実行 実行時に制御系を指定した場合: 両系(制御系→待機系の順番)に実行します。 実行時に待機系を指定した場合: 待機系に実行します。 |
| セパレートモード | 接続系のCPUユニットに対して実行します。 |
