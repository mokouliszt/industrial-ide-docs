# 17.3 レコーディング機能

下記ユニットのレコーディング機能を使用して、レコーディング対象として指定したデータを管理CPUユニットから収集し、レコーディングファイルとしてレコーディング設定で指定した保存先に保存します。

- MX-Rモデル

- レコーダユニット

**- カメラレコーダユニットレコーディング機能の関連項目**

→ MELSEC MXコントローラMX-Rモデルユーザーズマニュアル

→ MELSEC iQ-Rシステムレコーダユーザーズマニュアル(応用編)

→ MELSEC iQ-Rシステムレコーダ連動レコーディング機能リファレンスマニュアル

MX-Rモデル、RnCPU(R04CPU、R08CPU、R16CPU、R32CPU、R120CPU)、RnENCPU、およびRnSFCPUを設定したプロジェクトでレコーディング機能を使用できます。

## レコーディング機能の使用手順

### MX-Rモデルの場合

MX-Rモデルのプロジェクトでレコーディング機能を使用するときの手順を示します。

1. レコーディング機能の設定を行います。(→ レコーディング機能の設定)

2. パラメータやプログラムなどのデータおよびレコーディング設定を、同時にMX-Rモデルに書き込みます。(→ 695

ページCPUユニットへのデータの書込み/読出し)

プログラムが実行されるとレコーディングが開始し、レコーディング結果がレコーディングファイルとして保存されます。

3. レコーディングファイルをSDカードに保存した場合は、ファイルをパソコンに読み出します。(→ レコーディングファイルの読出し/削除)

4. レコーディング結果をオフラインモニタ機能で確認します。(→ レコーディングファイルを確認する)

- カメラレコーダユニットを使用する場合カメラレコーダユニットを使用する場合は、あらかじめプロジェクトにユニットを追加してください。(→ ユニットの追加)

| ユニット種別 | ユニット名 | ユニット形名 |
|---|---|---|
| 情報ユニット | カメラレコーダユニット | RD81RC96-CA |

また、追加後に下記パラメータを"サブ"に変更してください。

- ナビゲーションウィンドウの"パラメータ" ⇒ "ユニット情報" ⇒ "(ユニット形名)" ⇒ "ユニットパラメータ" ⇒ "基本設定" ⇒ "各種動作設定" ⇒ "レコーディング動作設定"

### それ以外のCPUユニットの場合

RnCPU(R04CPU、R08CPU、R16CPU、R32CPU、R120CPU)、RnENCPU、およびRnSFCPUのプロジェクトでレコーディング機能を使用するときの手順を示します。

1. レコーダユニットまたはカメラレコーダユニットをプロジェクトに追加します。

2. レコーディング機能の設定を行います。(→ レコーディング機能の設定)

3. パラメータやプログラムなどのデータおよびレコーディング設定を、同時にCPUユニットに書き込みます。(→ 695

ページCPUユニットへのデータの書込み/読出し)

プログラムが実行されるとレコーディングが開始し、レコーディング結果がレコーディングファイルとして保存されます。

4. レコーディングファイルをSDカードに保存した場合は、ファイルをパソコンに読み出します。(→ レコーディングファイルの読出し/削除)

5. レコーディング結果をオフラインモニタ機能で確認します。(→ レコーディングファイルを確認する)

## レコーディング機能の設定

パラメータを除くレコーディング機能に関する設定は、レコーディング設定で設定します。

本項では、レコーディング設定を新規作成、編集、削除する手順を説明します。

**レコーディング設定を設定する"レコーディング設定"画面の操作方法および設定項目の関連項目**

→ MELSEC MXコントローラMX-Rモデルユーザーズマニュアル

→ MELSEC iQ-Rシステムレコーダユーザーズマニュアル(応用編)

### レコーディング設定の新規作成

- MX-Rモデルの場合MX-Rモデルのプロジェクトでレコーディング設定を新規作成する手順を示します。

**操作手順**

1. [レコーディング] ⇒ [レコーディング設定] ⇒ [新規作成]を選択します。

2. 表示されたメッセージを確認し、[はい]ボタンをクリックします。

3. 表示されたメッセージを確認し、[OK]ボタンをクリックします。

4. "レコーディング設定"画面で各項目を設定し、[OK]ボタンをクリックします。

下記の手順でも、レコーディング設定を新規作成できます。

- ナビゲーションウィンドウの"パラメータ" ⇒ "レコーディング設定"を選択し、右クリック ⇒ ショートカットメニュー[新規作成]を選択

- それ以外のCPUユニットの場合RnCPU(R04CPU、R08CPU、R16CPU、R32CPU、R120CPU)、RnENCPU、およびRnSFCPUのプロジェクトでレコーディング設定を新規作成する手順を示します。

**操作手順**

1. プロジェクトに、下記いずれかのユニットを追加します。(→ ユニットの追加)

| ユニット種別 | ユニット名 | ユニット形名 |
|---|---|---|
| 情報ユニット | レコーダユニット | RD81RC96 |
| 情報ユニット | カメラレコーダユニット | RD81RC96-CA |

2. [レコーディング] ⇒ [レコーディング設定] ⇒ [新規作成]を選択します。

3. "レコーディング設定"画面で各項目を設定し、[OK]ボタンをクリックします。

レコーダユニットまたはカメラレコーダユニットをプロジェクトに追加後、下記の手順でも、レコーディング設定を新規作成できます。

- ナビゲーションウィンドウの"パラメータ" ⇒ "レコーディング設定"を選択し、右クリック ⇒ ショートカットメニュー[新規作成]を選択

### レコーディング設定の編集

レコーディング設定を編集する手順を示します。

**操作手順**

1. [レコーディング] ⇒ [レコーディング設定] ⇒ [編集]を選択します。

2. "編集"画面の"レコーディング設定選択"のプルダウンリストから、任意のレコーディング設定を選択します。

3. [OK]ボタンをクリックします。

4. "レコーディング設定"画面で各項目を設定し、[OK]ボタンをクリックします。

下記の手順でも、レコーディング設定を編集できます。

- ナビゲーションウィンドウの"パラメータ" ⇒ "レコーディング設定" ⇒ 任意のレコーディング設定をダブルクリック

### レコーディング設定の削除

レコーディング設定を削除する手順を示します。

**操作手順**

1. ナビゲーションウィンドウの"パラメータ" ⇒ "レコーディング設定" ⇒ 任意のレコーディング設定を選択します。

2. 右クリック ⇒ ショートカットメニュー[データ削除]を選択します。

3. 表示されたメッセージを確認し、[はい]ボタンをクリックします。

下記いずれかの条件に該当する場合、RnCPU(R04CPU、R08CPU、R16CPU、R32CPU、R120CPU)、

RnENCPU、およびRnSFCPUのプロジェクトからレコーディング設定が削除されます。

- すべてのレコーダユニットおよびカメラレコーダユニットをプロジェクトから削除した。

- レコーダユニットまたはカメラレコーダユニットが自号機管理でなくなった。

- レコーダユニットまたはカメラレコーダユニットに対応していないCPUユニットの機種に変更した。

### レコーディングファイルの読出し

レコーディングファイルをユニットから読み出し、パソコンへ保存します。

**操作手順**

1. "表示対象"でレコーディング設定にチェックを入れます。

レコーディング設定に保存されたレコーディングファイルが画面に表示されます。

2. 読み出すレコーディングファイルにチェックを入れ、[パソコンへ読出し]ボタンをクリックします。

3. "フォルダーの参照"画面で読み出すフォルダーを選択し、[OK]ボタンをクリックします。

カメラレコーダユニットを使用して生成した動画ファイルは、レコーディングファイルに含まれた状態で読み出されます。

**注意事項**

- RnSFCPU、またはユーザ認証機能を有効化したMX-Rモデルからプロジェクトデータを読み出す場合、ユーザ認証が必要です。

- MX-Rモデル: → MXコントローラへのログオン

- RnSFCPU: → CPUユニットへのログオン

- CPUユニットにセキュリティバージョンが"2"のプロジェクトを書き込んだ場合、プロジェクトデータを読み出すときにプロジェクト認証が必要です。(→ プロジェクト認証)

### サーボシステムレコーダのロギングファイル読出し

レコーディングファイルの読出し時に、モーションCPU/モーションユニットからサーボシステムレコーダのロギングファイルを読み出します。

ロギングファイルの読出しは、CPUユニットを経由してモーションCPU/モーションユニットに接続して行われます。

モーションCPU/モーションユニットへの直結接続は不要です。

**サーボシステムレコーダの関連項目**

→ MELSEC iQ-Rモーションコントローラプログラミングマニュアル(共通編)

→ MELSEC iQ-Rモーションユニットユーザーズマニュアル(応用編)

MX-Rモデルは対応していません。

**操作手順**

1. "表示対象"でレコーディング設定にチェックを入れます。

レコーディング設定に保存されたレコーディングファイルが画面に表示されます。

2. 読み出すレコーディングファイルにチェックを入れます。

3. "サーボシステムレコーダのロギングファイルを読出す"にチェックを入れます。

4. [パソコンへ読出し]ボタンをクリックします。

5. "フォルダーの参照"画面で読み出すフォルダーを選択し、[OK]ボタンをクリックします。

- ロギングファイル読出しが動作する構成ロギングファイル読出しに対応する機種は下記のとおりです。

| ユニット種別 | ユニット形名 |

| モーションユニット | RD78G4、RD78G8、RD78G16、RD78G32、RD78G64、RD78GHV、RD78GHW |
| モーションCPU | R16MTCPU、R32MTCPU、R64MTCPU |

- ロギングファイルの読出し条件"レコーディングファイルの読出し"画面で選択したレコーディングファイルと関連するロギングファイルが読み出されます。

下記のいずれかの場合に、ロギングファイルがレコーディングファイルと関連するファイルとして読み出されます。

- ロギングファイルの保存時刻*1がレコーディングファイルの記録開始時刻から記録終了時刻の間に含まれる場合

### レコーディングファイルの削除

レコーディングファイルをユニットから削除します。

**操作手順**

レコーディングファイルにチェックを入れ、[削除]ボタンをクリックします。

ユニット内のSDメモリカードから、レコーディングファイルが削除されます。
