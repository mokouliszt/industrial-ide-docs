# 13.3 データのバックアップ/リストア

バックアップ/リストア機能は、CPUユニットのデータをバックアップ/リストアする機能です。

詳細は、下記を参照してください。

→ MELSEC MXコントローラ MX-Rモデルユーザーズマニュアル

→ MELSEC MXコントローラ MX-Fモデルユーザーズマニュアル本機能は、MXコントローラでのみ対応しています。

## バックアップ

"バックアップ/リストア"画面の[バックアップ]タブでは、CPUユニットのデータをバックアップできます。

画面表示

[オンライン] ⇒ [バックアップ/リストア] ⇒ [バックアップ]を選択します。

![図(元マニュアル p.721)](images/gxworks3/13-03-01.webp)

操作手順1.各項目を設定します。

![図(元マニュアル p.722)](images/gxworks3/13-03-02.webp)

| 項目 | | 内容 |
|---|---|---|
| | バックアップ先指定設定 | |

2. . [実行]ボタンをクリックします。

- バックアップデータの削除バックアップデータの削除方法を示します。

- ネットワークドライブ内のバックアップデータの場合: Windowsのエクスプローラーなどで該当のフォルダーにアクセスしてバックアップデータを削除できます。

- SDメモリカード内のバックアップデータの場合: 下記の操作で削除できます。

⇒ [オンライン] ⇒ [ユーザデータ] ⇒ [削除]を選択します。

"ユーザデータ操作"画面で、"SDの削除対象フォルダ"の[参照]ボタンをクリックします。

"フォルダの参照"画面で、削除するバックアップデータの番号フォルダを選択し、右クリック ⇒ ショートカットメニュー

[フォルダの削除]を選択します。

例: "ルートフォルダ" ⇒ "$MELPRJ$" ⇒ "Backup" ⇒ "CPU" ⇒ "(日付フォルダ)" ⇒ "(番号フォルダ)"

### バックアップデータ数上限値設定画面

"バックアップデータ数上限値設定"画面では、バックアップデータの保存先に作成されるフォルダ数の上限値を設定できます。

画面表示

[バックアップ]タブの"バックアップデータ数上限値設定"の[設定]ボタンをクリックします。

![図(元マニュアル p.723)](images/gxworks3/13-03-03.webp)

操作手順1.各項目を設定します。

| 項目 | 内容 |
|---|---|
| バックアップデータ数上限値有効設 定 | バックアップデータのフォルダ数の上限値を設定するかどうかを選択します。 |
| バックアップデータ数上限値 | バックアップデータのフォルダ数の上限値を設定します。 |
| 上限値に達した場合の動作 | バックアップデータのフォルダ数の上限値に達した場合の動作を、下記のいずれかの項目から選択します。 • 最も古い日付のフォルダを削除してバックアップを行う • 上限値を超えてバックアップを行わず異常完了する |

2. . [CPUへ書込み]ボタンをクリックします。

## 自動バックアップ設定

"自動バックアップ/リストア設定"画面の[自動バックアップ設定]タブでは、CPUユニットのデータを自動でバックアップする設定ができます。

画面表示

[オンライン] ⇒ [バックアップ/リストア] ⇒ [自動バックアップ設定]を選択します。

または、"自動バックアップ/リストア設定"画面の[自動バックアップ設定]タブを選択します。

![図(元マニュアル p.724)](images/gxworks3/13-03-04.webp)

操作手順1.各項目を設定します。

![図(元マニュアル p.725)](images/gxworks3/13-03-05.webp)

| 項目 | | 内容 |
|---|---|---|
| | 日、時間指定設定 | |
| | 時間、曜日指定設定 | |
| | 停止エラー発生時設定 | |
| | 自動バックアップ先指 定設定 | |

2. . [CPUへ書込み]ボタンをクリックします。

- [CPUから読出し]ボタンをクリックすると、CPUユニットから自動バックアップ設定を読み出せます。

- バックアップデータはWindowsのエクスプローラーまたは"ユーザデータ操作"画面から削除できます。

(→ バックアップデータの削除)

"自動バックアップ/リストア設定"画面の[自動リストア設定]タブでは、CPUユニットのデータを自動でリストアする設定ができます。

![図(元マニュアル p.726)](images/gxworks3/13-03-06.webp)

![図(元マニュアル p.726)](images/gxworks3/13-03-07.webp)

2. . [CPUへ書込み]ボタンをクリックします。

[CPUから読出し]ボタンをクリックすると、CPUユニットから自動リストア設定を読み出せます。

### 自動リストア詳細設定画面

"自動リストア詳細設定"画面では、リストア対象のデータを設定できます。

画面表示

[自動リストア設定]タブの"自動リストア詳細設定"の[詳細]ボタンをクリックします。

![図(元マニュアル p.727)](images/gxworks3/13-03-08.webp)

操作手順1.各項目を設定します。

| 項目 | 内容 |
|---|---|
| リストア対象データ設定 | リストアの対象データを、下記のいずれかのデータから選択します。 • 全対象データ • データを指定 "データを指定"を選択した場合は、下記のいずれかのデータにチェックを入れます。 • 下記を除く全データ • ラッチ指定したデバイス/ラベルデータ • モーションデータ リストアの対象データの詳細は、下記を参照してください。 → MELSEC MXコントローラ MX-Rモデルユーザーズマニュアル → MELSEC MXコントローラ MX-Fモデルユーザーズマニュアル |
| ラッチエリアの特殊リレー/特殊レジ スタのリストア | ラッチエリアの特殊リレーおよび特殊レジスタをリストアするかどうかを選択します。 |
| 自動リストア時の初期化設定 | 自動リストア実行時に、リストア対象のドライブを初期化するかどうかを選択します。 |

2. . [設定]ボタンをクリックします。
