# 9.5 デバイスおよびラベルの一括置換

プログラム中のデバイスやラベルをまとめて置換します。

対応エディタ: ラダーエディタ、STエディタ、FBD/LDエディタ*1、SFC図エディタ、ラベルエディタ(ラベルの置換の場合)

*1インラインST部品のプレビュー表示エリアは対応していません。

画面表示

[検索/置換][デバイス一括置換]を選択します。

![図(元マニュアル p.566)](images/gxworks3/09-05-01.webp)

操作手順

各項目を設定し、[すべて置換]ボタンをクリックします。

| 項目 | | 内容 |
|---|---|---|
| 検索デバイス、置換デバイス | | 検索対象および置換後のデバイス名、ラベル名を入力してください。 構造体型ラベルを置換する場合、構造体メンバを含めた文字列を"検索デバイス"列または"置換デバイス"列に入力 してください。 配列型ラベルを置換する場合は、配列の添字を含めた文字列を"検索デバイス"列または"置換デバイス"列に入力し てください。*1 |
| 点数、点数形式 | | "検索デバイス"のデバイスから、何点分を変更するか入力します。 例)検索デバイスをX0、置換デバイスをX10、点数を5、点数形式を10進で設定して置換した場合 X0 X10、X1 X11、X2 X12、X3 X13、X4 X14に置換されます。 "検索デバイス"または"置換デバイス"がラベルの場合は、1以外を入力しても1として処理されます。 |
| 置換オプション | コメント | "検索デバイス"のデバイスコメントまたはラベルコメントを"置換デバイス"にコピーするか、移動するかを設定し ます。*2(S/TRは対象外です。) |

*1配列要素の数字の前に付く0は無視して検索されます。(→ 配列型ラベルの配列要素)*2"検索デバイス"のラベルコメントは、"置換デバイス"のデバイスコメントにのみコピー/移動できます。ラベルコメントにはコピー/移動できません。

範囲選択を行い、ラダーエディタからドラッグ&ドロップすると、複数のデバイス/ラベルを一括で登録できます。

ラベル名を入力するときに、ラベルの候補が表示されます。

下記のいずれかの操作によって、ラベルの候補を表示することもできます。

- 構造体型のラベル名またはファンクションブロックのインスタンス名の後ろに"."を入力する

- +を押す

さらに、下記のオプションを"はい"に設定すると、共通デバイスコメントおよびラベルコメントを入力することで、それらのコメントが設定されているデバイスおよびラベルを候補として表示できます。

- [ツール][オプション]"編集""命令/デバイス/ラベル候補表示""候補表示設定""デバイスコメント入力時に対応するデバイスを表示する"

注意事項

ステップリレー(S)/SFC移行デバイス(TR)を置換する場合"検索デバイス"にステップリレー(S)/SFC移行デバイス(TR)を指定し、"(プロジェクト全体)"を対象とした場合、すべてのブロックのステップリレー(S)/SFC移行デバイス(TR)が置換されます。(例: TR0を置換した場合、BL0¥TR0もBL1¥TR0も置換されます。)

他のアプリケーションからデバイスをコピー&貼り付けする場合Excelなどの他のアプリケーションから"デバイス一括置換"画面に、検索または置換するデバイスをコピー&貼り付けする場合、未入力の点数にはデフォルト値"1"が入力されます。

元に戻す/やり直しの対象範囲開いている画面のデータのみが、元に戻す/やり直しの対象です。

例: プロジェクト内の「X0」デバイスを「X1」デバイスに一括置換した後、元に戻す操作を行った場合、開いている画面の「X1」のみが「X0」に戻ります。

また、下記のいずれかの操作を行うと、元に戻す/やり直しの操作を実行できません。

- プログラムを変換する。

- 画面を閉じる。
