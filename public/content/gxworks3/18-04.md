# 18.4 安全シーケンサ操作

RnPSFCPUおよびRnSFCPUの機能を操作する方法を説明します。

## 安全動作モード切替え

CPUユニットの安全動作モード("セーフティモード"/"テストモード")を切り替えます。

動作の詳細は、下記を参照してください。

→ MELSEC iQ-R SIL2プロセスCPUユニットユーザーズマニュアル

→ MELSEC iQ-R CPUユニットユーザーズマニュアル(応用編)

画面表示

[オンライン] ⇒ [安全シーケンサ操作] ⇒ [安全動作モード切替え]を選択します。

![図(元マニュアル p.944)](images/gxworks3/18-04-01.webp)

動作中のCPUユニットの安全動作モードは、"ユニット診断"画面の[ユニット情報一覧]タブから確認できます。

注意事項

- データが一致しない場合GX Works3のプロジェクトデータとRnSFCPU/RnPSFCPU内のデータが一致しない場合、セーフティモードに切り替えることができません。データの一致/不一致はファイル識別子が一致するかどうかでチェックされます。

データが一致していない場合は、下記いずれかの操作を行ってください。セーフティモードに切り替えられるようになります。

- GX Works3のデータをCPUユニットに書き込む

- CPUユニットからデータを読み出す

## 安全動作モード切替え前の注意事項

## 注意

- セーフティモードに切り替える前に、すべての安全ループに対して、システムレベルでテストを十分に実施してください。・セーフティモードに切り替える前に、下記の操作により意図した安全プロジェクトのプログラムとパラメータが正しく書き込まれたかどうかを必ず確認してください。

**操作手順**

1. 編集した安全プロジェクトをパソコンに保存します。(以降"プロジェクトA"とします。)

2. GX Works3を起動し、安全プロジェクトのプログラムおよびパラメータをCPUユニットへ書き込みます。(→ 698ページ シーケンサへの書込み)

3. 別のGX Works3を起動した後、CPUユニットからプログラムとパラメータを読み出します。(→ シーケンサからの読出し)(以降"プロジェクトB"とします。)

4. 目視により、プロジェクトAとプロジェクトBに相違がないことを確認します。(→ プログラムの確認、

→ ユニットパラメータ(安全通信設定)の確認、→ 安全リモートI/Oユニットのパラメータの確認)

5. プログラムをアプリケーション環境で十分にテストします。

- プログラムの確認プロジェクトBにて、照合先に"プロジェクトA"を選択してプロジェクト照合し、"照合結果"ウィンドウの詳細表示の"照合元

"とプロジェクトAのプログラムエディタを比較し、意図したプログラムが正しく書き込まれていることを確認します。

![図(元マニュアル p.945)](images/gxworks3/18-04-02.webp)

- ユニットパラメータ(安全通信設定)の確認プロジェクトBの"ユニットパラメータ"の"安全通信設定"をファイルへ出力し、保存したCSVファイルとプロジェクトAの"安全通信設定"画面を比較し、意図したパラメータが正しく書き込まれていることを確認します。

## 安全データ同一性チェック

"安全データ同一性チェック"画面では、GX Works3のプロジェクトデータがRnSFCPUまたはRnPSFCPU内のデータと同一かどうかをチェックできます。これにより、セーフティモードで運用するプログラムがユーザが書き込んだデータかどうかを確認できます。

本機能を使用する上での注意事項は、下記を参照してください。

→ MELSEC iQ-R 安全アプリケーションガイド画面表示

[オンライン] ⇒ [安全シーケンサ操作] ⇒ [安全データ同一性チェック]を選択します。

![図(元マニュアル p.947)](images/gxworks3/18-04-06.webp)

### プロジェクトとCPUユニット内のデータが一致しているかの確認

比較結果を確認します。

意図したデータで稼働しているかの確認1.プログラム/パラメータをCPUユニットへ書込み後にセーフティモードに移行したとき、GX Works3を使用してCPUユニット内の安全データ同一性チェック情報をCSVファイルに保存します。

2. 後日、GX Works3で安全データ同一性チェック機能を使用し、保存したCSVファイル内の"ファイル識別子"が一致しているか確認します。

不一致の場合は、ユーザ自身が書き込んだデータではない可能性があるため、実際の運用には移行しないでください。

ファイル識別子は変換時に付与されます。

## 安全バックアップ/リストア機能

バックアップしたデータをCPUユニット内に自動でリストアします。

### 自動リストア設定

**操作手順**

[オンライン] ⇒ [安全シーケンサ操作] ⇒ [自動リストア設定] ⇒ [有効化]/[無効化]を選択します。

## 安全ユニット操作

RnSFCPUまたはRnPSFCPUを使用したシステムで使用するユニットに対して、安全通信機能を有効化/無効化します。

画面表示1. [オンライン] ⇒ [安全シーケンサ操作] ⇒ [安全ユニット操作]を選択します。

2. "安全ユニット操作 - ユニット選択"画面で操作するユニットを選択し、[OK]ボタンをクリックします。(RnPSFCPUの場合のみ)

例RnPSFCPUを使用したシステムの場合

![図(元マニュアル p.948)](images/gxworks3/18-04-07.webp)

### ユニットの位置を確認する

有効化/無効化を切り替える前に、ユニットの装着位置を確認します。

**操作手順**

1. 位置を確認するユニットにチェックを入れます。

2. [S MODE LED点滅開始]ボタンをクリックします。

選択したユニットのS MODE LEDが点滅していることを確認後、[S MODE LED点滅停止]ボタンをクリックします。

### 安全通信機能の有効化/無効化状態を切り替える

RnSFCPUまたはRnPSFCPUを使用したシステムで使用するユニットに対して、安全通信機能の有効化/無効化を切り替えます。

RnSFCPUの場合は安全通信機能の無効化に対応していません。

**操作手順**

1. 安全通信機能を有効化するユニット、または無効化するユニットにチェックを入れます。

2. [有効化]ボタン/[無効化]ボタンをクリックします。

3. リモートヘッドユニットのリセットまたは電源OFF→ONをします。
