# 12.8 QJ71E71-100経由

パソコンからQJ71E71-100を経由してCPUユニットにアクセスするときの設定方法を説明します。

MX-Fモデル、LHCPU、およびFX5CPUへはQJ71E71-100を経由したアクセスができません。

シーケンサ側I/Fとして使用できるQCPU(Qモード)は、下記のとおりです。

- ユニバーサルモデルQCPU

- ユニバーサルモデルプロセスCPU

- ユニバーサルモデル高速タイプQCPU

パソコンからQJ71E71-100を経由してRCPUにアクセスする場合の設定例を示します。

注意事項

- MELSECNET/10ネットワークユニットまたはMELSEC-QシリーズのMELSECNET/Hネットワークユニットを経由して、

RCPUにアクセスすることはできません。

- QJ71E71-100のEthernetポートに設定されたリモートパスワードは、GX Works3で解除できません。あらかじめGX

Works2を使用してリモートパスワードを削除しておいてください。

- GX Works3を起動している複数台のパソコンを1つのCPUユニットに接続して交信する場合は、TCP/IPまたはUDP/IPで交信してください。

- 複数ネットワークシステムの場合は、ネットワーク動的ルーチング設定をする必要があります。

→ MELSEC iQ-R Ethernetユーザーズマニュアル(応用編)

→ MELSEC-Q/L Ethernetインタフェースユニットユーザーズマニュアル（応用編）
