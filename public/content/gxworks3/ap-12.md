# 付12 サンプルプログラムを使用する方法

e-Manualに記載されているサンプルプログラム(ラダープログラム)をGX Works3で使用する方法を説明します。

e-Manual Viewerの操作方法は、e-Manual Viewerのヘルプを参照してください。

### 対応バージョン

本機能を使用するには、GX Works3で使用したいサンプルプログラム(ラダープログラム)に合わせて、下記バージョンのGX

Works3とe-Manual Viewerがインストールされている必要があります。

- ラダープログラムにファンクションブロックが含まれていない場合

- GX Works3: Version 1.035M以降

- e-Manual Viewer: Version 1.13P以降

- ラダープログラムにファンクションブロックが含まれている場合

- GX Works3: Version 1.055H以降

- e-Manual Viewer: Version 1.18U以降

**操作手順**

サンプルプログラム(ラダープログラム)をGX Works3で使用する操作手順を示します。

**操作手順**

- ラダープログラムにファンクションブロックが含まれていない場合1. e-Manualの内容に従って、GX Works3プロジェクト内にユニットラベルを作成し、パラメータを設定します。

2. e-Manual内の"ラベルをコピーする(GX Works3用)"をクリックし、グローバルラベルまたはローカルラベルをコピーします。

3. 下記の2項目の順番を合わせます。

- 手順2でコピーしたラベルの列項目(ラベル名、データ型、クラス、割付け(デバイス/ラベル))

- GX Works3のラベルエディタの列項目4. GX Works3のグローバルラベルエディタまたはローカルラベルエディタに貼り付けます。

5. e-Manual内の"ラダープログラムをコピーする(GX Works3用)"をクリックし、ラダープログラムをコピーします。

6. GX Works3のラダーエディタに貼り付けます。

7. プログラムを変換します。

- ラダープログラムにファンクションブロックが含まれている場合1. e-Manualの内容に従って、GX Works3プロジェクト内にユニットラベルを作成し、パラメータを設定します。

2. e-Manual内の"ラベルをコピーする(GX Works3用)"をクリックし、グローバルラベルまたはローカルラベルをコピーします。

3. 下記の2項目の順番を合わせます。

- 手順2でコピーしたラベルの列項目(ラベル名、データ型、クラス、割付け(デバイス/ラベル))

- GX Works3のラベルエディタの列項目4. GX Works3のグローバルラベルエディタまたはローカルラベルエディタに貼り付けます。

5. e-Manual内の"ラダープログラムをコピーする(GX Works3用)"をクリックし、ラダープログラムをコピーします。

6. GX Works3のラダーエディタに貼り付けます。

サンプルプログラムに含まれているファンクションブロックは、未定義のファンクションブロックに変更されます。

下記の操作に従って、ファンクションブロックを再定義してください。

7. 該当するファンクションブロックを、部品選択ウィンドウからラダーエディタ上の未定義のファンクションブロックにドラッグ&ドロップします。

8. "FBインスタンス名入力"画面で各項目を設定し、[OK]ボタンをクリックします。

9. プログラムを変換します。

### 注意事項

- ラベル名が重複する場合サンプルプログラムのラベル名と、貼り付け先のプロジェクト内のラベル名が重複する場合、プログラムの変換時にエラーが発生します。

下記の操作に従って、再度サンプルプログラムをコピー&貼り付けしてください。

**操作手順**

1. エディタに貼り付けたラダープログラムを削除します。

2. ラベルエディタで重複するラベル名を変更します。

3. サンプルプログラムをGX Works3のラダーエディタに貼り付けます。

4. ラダープログラム内で使用しているラベル名を、手順2で変更したラベル名に修正します。

- 未定義ラベルが存在する場合下記の場合、サンプルプログラムのユニットラベルは、未定義ラベルとして貼り付けられます。

最新のGX Works3がインストールされていることを確認してください。

- サンプルプログラムとGX Works3のユニットラベルのバージョンが異なる

- サンプルプログラムのユニットラベルを、GX Works3がサポートしていない

- 間違ったファンクションブロックを再定義した場合プログラムを変換する前に、再定義したファンクションブロックがサンプルプログラムのファンクションブロックと同じかどうかを確認してください。

再定義するファンクションブロックを間違えた場合、下記の操作に従って、再度サンプルプログラムをコピー&貼り付けしてください。

**操作手順**

1. 下記のデータをプロジェクトから削除します。

- エディタに貼り付けたラダープログラム

- 再定義したファンクションブロック(FBインスタンスを含む)

2. サンプルプログラムをGX Works3のラダーエディタに貼り付けます。

3. サンプルプログラムのファンクションブロックと同じファンクションブロックを部品選択ウィンドウからラダーエディタにドラッグ&ドロップします。

4. プログラムを変換します。

- FBインスタンス名がサンプルプログラムのFBインスタンス名と異なる場合再定義したファンクションブロックのFBインスタンス名がサンプルプログラムのFBインスタンス名と異なる場合、変換エラーが発生することがあります。

その場合、エラーが発生した箇所のFBインスタンス名を再定義したファンクションブロックのFBインスタンス名に変更してください。
