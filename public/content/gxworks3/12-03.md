# 12.3 ネットワーク経由(単一ネットワーク)

パソコンからEthernet内蔵形CPUにアクセスし、単一ネットワークを経由して他局のCPUユニットにアクセスする場合の設定例を示します。

![図(元マニュアル p.684)](images/gxworks3/12-03-01.webp)

| No. | 項目選 | 選択項目 | 内部設定 | 入力値 |
|---|---|---|---|---|
| (1) | パソコン側I/F Et | hernetボード | アダプタ*1ネットワークNo.局番プロトコル | 指定なし――UDP |
| (2) | シーケンサ側I/F C | PUユニット | Ethernetポート直結接続 | ― |
| | 他局指定他 | 他局(単一ネットワーク) | 交信タイムチェックリトライ回数 | 30秒0回 |
| (3) | ネットワーク通信経路C C | C IE TSN C IE Field | ネットワークNo.局番 | 1 0 |

*1：FX5CPUの場合は設定項目がありません。

注意事項

- GX Works3を起動している複数台のパソコンを1つのCPUユニットに接続して交信する場合は、TCP/IPまたはUDP/IPで交信してください。

- 複数ネットワークシステムの場合は、ネットワーク動的ルーチング設定をする必要があります。(RCPUのみ)

→ MELSEC iQ-R Ethernetユーザーズマニュアル(応用編)

- 同一のIPアドレスが複数表示される場合"シーケンサ側 I/F Ethernetユニット詳細設定"画面の[検索]ボタンをクリックすると、接続先Ethernetユニットの一覧に重複したIPアドレスが表示される場合があります。

その場合、"パソコン側 I/F Ethernetボード詳細設定"画面でアダプタを選択してください。
