# 1.2 MXコントローラ対応

GX Works3は、モーションシステムを構築するMXコントローラに対応しています。

**MXコントローラを使用したシステムの関連項目**

→ MELSEC MXコントローラMX-Rモデルユーザーズマニュアル

→ MELSEC MXコントローラMX-Fモデルユーザーズマニュアル

→ MELSEC MXコントローラMX-Fモデル(EtherCAT対応)ユーザーズマニュアル

### MXコントローラの機能一覧

GX Works3には、MXコントローラを使用したシステムを構築および制御するための機能が備わっています。

| 機能名 | 内容 | 参照 |
|---|---|---|
| ネットワークラベル設定 | 下記のユニット間で使用する入出力のデータをラベルとして登録できます。 / MXコントローラ / MXコントローラ内蔵CC-Link IE TSNのデバイス局(リモート局) MXF100-ECの場合、下記のユニット間で使用する入出力のデータをネットワークラベルとして登録できます。 / MXコントローラ / MXコントローラ内蔵EtherCAT®のSubDevice | → ネットワークラベル設定 |
| アドオン管理 | MXコントローラの機能を拡張するアドオンを管理できます。 | → アドオン機能 |
| アドオンパラメータ | アドオンに関連するパラメータを設定できます。 | → アドオンパラメータ |
| パフォーマンスモニタ | MXコントローラのコアごとの演算周期の内訳や負荷状態を可視化できます。 | → コアの負荷状況を確認する |

**なお、MXコントローラの機能の関連項目**

→ MELSEC MXコントローラMX-Rモデルユーザーズマニュアル

→ MELSEC MXコントローラMX-Fモデルユーザーズマニュアル

→ MELSEC MXコントローラMX-Fモデル(EtherCAT対応)ユーザーズマニュアル

### モーションシステム対応機能一覧

GX Works3には、モーションシステムを構築および制御するための機能が備わっています。

| 機能名 | 内容 | 参照 |
|---|---|---|
| モーション設定 | MXコントローラを動作させるために必要な設定ができます。 | → モーション設定 |
| 軸モニタ | 実行中の運転軸の現在値およびエラーコードを一括で確認できます。 | → 軸モニタ |
| 位置データ履歴 | 軸の位置データ履歴を確認できます。 | → 位置データ履歴 |
| モーション共通モニタ | 軸および軸グループに依存しない演算周期などのCPUユニットの情報を確認できます。 | → モーション共通モニタ |

## MXコントローラの用語について

MXコントローラに関する記述では、「CPUユニット」を「コントローラ」と読み替えてください。

MXコントローラとそれ以外のCPUユニットでは、同じ用語でも意味や対象範囲が異なる場合があります。

CPUユニットによって内容が異なる用語は次のとおりです。

| 用語 | 対象CPU | 内容 |
|---|---|---|
| プログラムのステップ数 | MXコントローラ | プログラム容量の単位です。プログラム容量の単位にはバイトを使用します。 |
| | それ以外のCPUユニット | プログラム容量の単位です。プログラム容量の単位にはステップまたはバイトを使用します。1ステップはRCPUが4バイト、FX5CPUが2バイトです。 |
| ステップNo. | MXコントローラ | プログラムの位置を示す単位です。ステップNo.の数字は、プログラムの先頭から命令数を積算したものです。 |
| | それ以外のCPUユニット | プログラムの位置を示す単位です。ステップNo.の数字は、プログラムの先頭から命令ごとに設定されたステップ数を積算したものです。 |
| 実行プログラム | MXコントローラ | コントローラで実行可能なプログラムを示します。実行プログラムは、"変換後実行プログラムを作成"にチェックを入れて変換/全変換を実行すると作成されます。(→ 実行プログラムの作成) |
| | それ以外のCPUユニット | 変換済みのプログラム(CPUユニットで実行可能なプログラム)を示します。 |
