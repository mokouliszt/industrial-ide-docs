# 1.2 MXコントローラ対応

GX Works3は、モーションシステムを構築するMXコントローラに対応しています。

MXコントローラを使用したシステムの詳細は、下記を参照してください。

MELSEC MXコントローラ MX-Rモデルユーザーズマニュアル

MELSEC MXコントローラ MX-Fモデルユーザーズマニュアル

MELSEC MXコントローラ MX-Fモデル(EtherCAT対応)ユーザーズマニュアル

### MXコントローラの機能一覧

GX Works3には、MXコントローラを使用したシステムを構築および制御するための機能が備わっています。

| 機能名 | 内容 | 参照 |
|---|---|---|
| ネットワークラベル設定 | 下記のユニット間で使用する入出力のデータをラベルとして登録できます。 • MXコントローラ • MXコントローラ内蔵CC-Link IE TSNのデバイス局(リモート局) MXF100-ECの場合、下記のユニット間で使用する入出力のデータをネット ワークラベルとして登録できます。 • MXコントローラ • MXコントローラ内蔵EtherCATのSubDevice | → ネットワークラベル設定 |
| アドオン管理 | MXコントローラの機能を拡張するアドオンを管理できます。 | → アドオン機能 |
| アドオンパラメータ | アドオンに関連するパラメータを設定できます。 | → アドオンパラメータ |
| パフォーマンスモニタ | MXコントローラのコアごとの演算周期の内訳や負荷状態を可視化できます。 | → コアの負荷状況を確認す る |

なお、MXコントローラの機能の詳細は、下記を参照してください。

MELSEC MXコントローラ MX-Rモデルユーザーズマニュアル

MELSEC MXコントローラ MX-Fモデルユーザーズマニュアル

MELSEC MXコントローラ MX-Fモデル(EtherCAT対応)ユーザーズマニュアル

### モーションシステム対応機能一覧

GX Works3には、モーションシステムを構築および制御するための機能が備わっています。

| 機能名 | 内容 | 参照 |
|---|---|---|
| モーション設定 | MXコントローラを動作させるために必要な設定ができます。 | → モーション設定 |
| 軸モニタ | 実行中の運転軸の現在値およびエラーコードを一括で確認できます。 | → 軸モニタ |
| 位置データ履歴 | 軸の位置データ履歴を確認できます。 | → 位置データ履歴 |
| モーション共通モニタ | 軸および軸グループに依存しない演算周期などのCPUユニットの情報を確認 できます。 | → モーション共通モニタ |

## MXコントローラの用語について

MXコントローラご使用時は、本マニュアルをお読みになる場合、「CPUユニット」を「コントローラ」に読み替えてください。
