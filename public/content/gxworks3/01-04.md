# 1.4 RnSFCPU対応

GX Works3は、安全システムを構築するMELSEC iQ-RシリーズのRnSFCPUに対応しています。

## 安全システム対応

RnPSFCPUおよびRnSFCPU用のプロジェクトを「安全プロジェクト」、それ以外を「一般プロジェクト」と表記します。

安全プロジェクトでは、安全データと一般データがあります。

安全プロジェクト内には、安全プログラムと一般プログラムを作成できます。

| 名称 | | 内容 |
|---|---|---|
| データ | 安全データ | 安全プログラム、安全FB/FUN、安全グローバルラベル、下記CPUユニットのパラメータを示します。 / RnPSFCPU / RnSFCPU |
| データ | 一般データ | 一般プログラム、一般FB/FUN、一般グローバルラベル、一般/安全共有ラベル、下記CPUユニット以外のパラメータを示します。 / RnPSFCPU / RnSFCPU |
| プログラム、FB/FUN | 安全プログラム、安全FB/FUN | 安全制御を実行するためのプログラムです。 |
| プログラム、FB/FUN | 一般プログラム、一般FB/FUN | 一般制御を実行するためのプログラムです。 |
| ラベル | 安全グローバルラベル | 安全プログラムでのみ使用できるグローバルラベルです。 |
| ラベル | 一般グローバルラベル | 一般プログラムでのみ使用できるグローバルラベルです。 |
| ラベル | 一般/安全共有ラベル | 安全プログラム、一般プログラムで使用できるグローバルラベルです。 一般/安全共有ラベルを安全プログラムで使用する場合は、安全状態が確認できるようにプログラミングしてください。 |

### 使用可能なデバイス、ラベル、FB/FUN

安全プロジェクトと一般プロジェクトでは、それぞれ使用できるデバイス、ラベル、FB/FUNが異なります。

#### 使用可能なデバイス/ラベル

| プロジェクト | 安全デバイス | 安全グローバルラベル | 一般/安全共有ラベル | 一般デバイス | 一般グローバルラベル |
|---|---|---|---|---|---|
| 安全プログラム | ○ | ○ | ○ | × | × |
| 一般プログラム | × | × | ○ | ○ | ○ |

**各プロジェクトで使用可能なデバイスの関連項目**

→ GX Works3で編集可能なデバイス

#### 使用可能なFB/FUN

| プロジェクト | 安全FB | 安全FUN | 一般FB | 一般FUN |
|---|---|---|---|---|
| 安全プログラム | ○ | ○ | × | × |
| 一般プログラム | × | × | ○ | ○ |

## 安全システム対応機能一覧

GX Works3には、安全プロジェクトを作成するための機能が備わっています。

| 機能名 | 内容 | 参照 |
|---|---|---|
| 安全動作モード切替え | CPUユニットの安全動作モードを切り替えます。 | → 安全動作モード切替え |
| ユーザ認証 | プロジェクトを開く際にユーザ認証を行い、権限がないユーザによる不正なアクセスを防止します。 | → CPUユニットのプロジェクトへの不正なアクセスの防止(ユーザ認証) |
| | CPUユニットまたはCPUユニットに装着されたSDメモリカードへアクセスする場合、ユーザ認証を行うことで権限がないユーザによる不正なアクセスを防止します。 | |
| 安全データ同一性チェック | CPUユニット内のデータが、ユーザ自身によって書き込まれたデータであるかチェックします。 | → 安全データ同一性チェック |

### 安全動作モード

安全動作モードには、セーフティモードとテストモードがあります。

### 安全データの識別表示

安全データは、ナビゲーションウィンドウのアイコンおよびパラメータエディタのグリッド背景色が強調して表示されます。
