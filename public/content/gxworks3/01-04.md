# 1.4 RnSFCPU対応

GX Works3は、安全システムを構築するMELSEC iQ-RシリーズのRnSFCPUに対応しています。

## 安全システム対応

本マニュアル内では、RnPSFCPUおよびRnSFCPU用のプロジェクトを"安全プロジェクト"と呼び、それ以外のプロジェクトを"一般プロジェクト"と呼びます。

安全プロジェクトでは、安全データと一般データがあります。

安全プロジェクト内には、安全プログラムと一般プログラムを作成できます。

| 名称 | | 内容 |
|---|---|---|
| データ | 安全データ | 安全プログラム、安全FB/FUN、安全グローバルラベル、下記CPUユニットのパラメータを 示します。 • RnPSFCPU • RnSFCPU |
| | 一般データ | 一般プログラム、一般FB/FUN、一般グローバルラベル、一般/安全共有ラベル、下記CPU ユニット以外のパラメータを示します。 • RnPSFCPU • RnSFCPU |
| プログラム、FB/FUN | 安全プログラム、安全FB/FUN | 安全制御を実行するためのプログラムです。 |
| | 一般プログラム、一般FB/FUN | 一般制御を実行するためのプログラムです。 |
| ラベル | 安全グローバルラベル | 安全プログラムでのみ使用できるグローバルラベルです。 |
| | 一般グローバルラベル | 一般プログラムでのみ使用できるグローバルラベルです。 |
| | 一般/安全共有ラベル | 安全プログラム、一般プログラムで使用できるグローバルラベルです。 一般/安全共有ラベルを安全プログラムで使用する場合は、安全状態が確認できるように プログラミングしてください。 |

### 使用可能なデバイス、ラベル、FB/FUN

安全プロジェクトと一般プロジェクトでは、それぞれ使用できるデバイス、ラベル、FB/FUNが異なります。

#### ・使用可能なデバイス/ラベル

| プロジェクト | 安全デバイス | 安全グローバル ラベル | 一般/安全共有ラベル | 一般デバイス | 一般グローバル ラベル |
|---|---|---|---|---|---|
| 安全プログラム | ○ | ○ | ○ | × | × |
| 一般プログラム | × | × | ○ | ○ | ○ |

各プロジェクトで使用可能なデバイスは、下記を参照してください。

→ GX Works3で編集可能なデバイス

#### ・使用可能なFB/FUN

| プロジェクト | 安全FB | 安全FUN | 一般FB | 一般FUN |
|---|---|---|---|---|
| 安全プログラム | ○ | ○ | × | × |
| 一般プログラム | × | × | ○ | ○ |

## 安全システム対応機能一覧

GX Works3には、安全プロジェクトを作成するための機能が備わっています。

| 機能名 | 内容 | 参照 |
|---|---|---|
| | CPUユニットの安全動作モードを切り替えます。 | |
| | プロジェクトを開く際にユーザ認証を行い、権限がないユーザによる不正なア クセスを防止します。 | |
| | CPUユニットまたはCPUユニットに装着されたSDメモリカードへアクセスす る場合、ユーザ認証を行うことで権限がないユーザによる不正なアクセスを防 止します。 | |
| | CPUユニット内のデータが、ユーザ自身によって書き込まれたデータであるか チェックします。 | |

### 安全動作モード

安全動作モードには、セーフティモードとテストモードがあります。

### 安全データの識別表示

安全データは、ナビゲーションウィンドウのアイコンおよびパラメータエディタのグリッド背景色が強調して表示されます。

詳細は、下記を参照してください。

- ナビゲーションウィンドウ: → 安全データの識別表示

- パラメータエディタ: → 安全データの識別表示
