# 14.9 プログラムの処理時間を確認する

実行中のプログラムの処理時間を確認するには、プログラム一覧モニタを使用します。

MXコントローラ、FX5CPU、およびリモートヘッドユニットでは対応していません。

MXコントローラのプロジェクトで実行中のプログラムの処理時間を確認する場合は、パフォーマンスモニタを使用してください。(→ コアの負荷状況を確認する)

**画面の開き方**

[オンライン] ⇒ [モニタ] ⇒ [プログラム一覧モニタ]を選択します。

![図(元マニュアル p.793)](images/gxworks3/14-09-01.webp)

**画面項目**

| 項目 | | 内容 |
|---|---|---|
| 各プログラムの実行状態 | 実行時間(ms) | 実際のスキャンタイム(現在値)が表示されます。表示内容は、プログラムの実行タイプにより異なります。プログラム停止(待機)時: 0.000ms |
| | 実行回数 | CPUユニットをRUN状態にした時点を0回として実行した回数が表示されます。プログラム停止後も実行回数は保持されます。 |
| 全体の実行時間 | 監視時間(ms) | "CPUパラメータ"の"RAS設定"で設定したプログラム実行監視時間が表示されます。 |
| | 実行時間合計(ms) | CPUユニットが実行しているプログラムのスキャンタイム合計時間が表示されます。 |
| スキャン実行分の実行時間詳細 | プログラム処理時間(ms) | スキャン実行タイププログラムの実行時間の合計が表示されます。 |
| | END処理時間(ms) | END処理時間が表示されます。 |
| | コンスタント待ち時間(ms) | コンスタントスキャン設定時、コンスタントスキャンの待ち時間が表示されます。 |

プログラム起動：任意のプログラムを起動します。(→ プログラムの起動)

プログラム停止：任意のプログラムを停止します。(→ プログラムの停止)

### プログラムの起動

任意のプログラムを起動します。

CPUユニットがRUN状態の場合に、プログラム実行タイプが"待機"のプログラムを起動できます。

プログラム実行タイプが"待機"以外の場合は起動できません。

**操作手順**

1. [プログラム起動]ボタンをクリックします。

2. "プログラム起動"画面の各項目を設定し、[実行]ボタンをクリックします。

### プログラムの停止

任意のプログラムを停止します。

CPUユニットがRUN状態の場合に、プログラム実行タイプが"スキャン"のプログラムを停止できます。

プログラム実行タイプが"スキャン"以外の場合は停止できません。

**操作手順**

1. [プログラム停止]ボタンをクリックします。

2. "プログラム停止"画面の各項目を設定し、[実行]ボタンをクリックします。

"プログラム停止"画面の"停止モード"で選択した項目によって、プログラム停止時の動作が下記のとおり異なります。

- 停止後、出力停止: プログラムの次のスキャン時にOUT命令の出力がOFFになります。その次のスキャン以降、プログラム実行タイプが"待機"に変更されます。(POFF命令と同一の動作)

- 停止後、出力保持: プログラム実行タイプが"待機"に変更され、OUT命令の出力が保持されます。(PSTOP

命令と同一の動作)

**注意事項**

RnPCPU(二重化モード)のプログラムを停止した後に系切替えが発生した場合、停止したプログラムのプログラム実行タイプが系切替え後の制御系に引き継がれます。
