# 5.2 ラベルの登録

グローバルラベル/ローカルラベルの登録方法を説明します。

グローバルラベルは1ファイル内に20480個、1プロジェクト内に最大16384000個、ローカルラベルは1プログラムファイル内に最大5120個登録できます。

下記のパターンを除く名称を設定してください。

→ ラベル名

注意事項

ラベル名、定数、初期値には、Unicode基本多言語面の文字を使用してください。

なお、使用する言語とWindowsの地域設定で設定している言語が異なると機能が正常に動作しない場合があります。正常に動作しない場合、使用する言語とWindowsの地域設定の言語を統一してください。

基本多言語面以外の文字を指定したラベル名および定数値は、プログラムでは使用できません。

また、基本多言語面以外の文字を指定した初期値は、プログラムで正常に動作しない場合があります。

例グローバルラベル複製時に"ラベル初期値設定"画面で設定した初期値を複製する方法を説明します。

## 操作手順1.ナビゲーションウィンドウでグローバルラベル「Global」を複製し、「Global_2」を作成します。(→ コ

ピー&貼り付け) 2. 「Global」のグローバルラベルエディタから、"ラベル初期値設定"画面を開きます。(→ 初期値の設定) 3."ラベル初期値設定"画面で初期値を選択し、右クリックショートカットメニュー[コピー]を選択します。4."ラベル初期値設定"画面を閉じます。5. 「Global_2」のグローバルラベルエディタから、"ラベル初期値設定"画面を開きます。(→ 初期値の設定) 6.初期値入力エリアを選択し、右クリックショートカットメニュー[貼り付け]を選択してコピーした初期値を貼り付けます。

シングルクォーテーションおよびダブルクォーテーションはそのままコピー&貼り付けされます。
