# 5.4 タグFBの登録

RnPCPUおよびRnPSFCPUのプロジェクトにおいて、プロセス制御拡張を有効にしたFBD/LDプログラムでは、タグFBを使用できます。

タグFBを使用するには、事前にタグFB設定エディタでタグFBを登録する必要があります。

タグ名、タグFB型、コメントを入力し、[適用]ボタンをクリックしてください。

タグFBはプロジェクト内に最大で1000個登録できます。

タグFBの詳細は、下記を参照してください。

→ MELSEC iQ-R プログラミングマニュアル(プロセス制御FB/命令編)

注意事項

タグFBを使用する場合は、プログラムブロック、ファンクションブロック、ファンクション、グローバルラベル、構造体定義のデータを追加する前に、タグFB設定エディタにタグFBを登録し適用しておくことをお薦めします。

プロジェクトに上記のデータが多く登録されている場合、タグFB設定の適用に時間がかかることがあります。

## タグFB設定エディタの構成

タグFB設定エディタの画面構成を説明します。

画面表示

- [ツール] ⇒ [プロセス制御拡張] ⇒ [タグFB設定]を選択します。

- ツールバーの：(タグFB設定)をクリックします。

![図(元マニュアル p.349)](images/gxworks3/05-04-01.webp)

表示内容

| 項目 | 内容 |
|---|---|
| フィルタ | データをフィルタ表示します。フィルタ条件としてワイルドカード"*"、"?"を使用してもワイルドカードによるフィルタ結果は表示せず、"*"、"?"を使用している文字列を表示します。 |
| [最大タグ数]ボタン | 下記のオプションを確認および設定します。• [ツール] ⇒ [オプション] ⇒ "変換" ⇒ "プロセス制御拡張設定" ⇒ "タグFB" ⇒ "最大タグ数" |
| [チェック]ボタン | エラーを確認します。 |

#### ・設定項目○: 編集可、×: 編集不可

| 項目 | 編集 | 内容 |
|---|---|---|
| タグ名 | ○ | タグFBの名称を入力します。 |
| タグFB型 | ○ | タグFBインスタンスのタグFB型を選択します。 |
| タグタイプ | × | タグFBインスタンスのタグタイプが表示されます。 |
| 構造体型 | × | タグデータの構造体定義の名称が表示されます。 |
| FBインスタンス名 | × | タグFBインスタンスの名称が表示されます。 |
| 構造体ラベル名 | × | タグデータの名称が表示されます。 |
| 割付けデバイス | × | 下記のデバイスが割り付けられます。• システムリソースの先頭デバイス+3000+((行番号-1)*130)システムリソースは[ツール] ⇒ [オプション] ⇒ "変換" ⇒ "プロセス制御拡張設定" ⇒ "システムリソース" ⇒ "ファイルレジスタ: ZR"で設定します。 |
| コメント | ○ | タグFBインスタンスとタグデータのコメントに反映するコメントを入力します。 |

- [チェック]ボタンまたは[適用]ボタンをクリックすると、プロジェクトに使用されているタグデータおよびプロセス制御ファンクションブロックのバージョンが、プロジェクトを開いているGX Works3で対応しているかをチェックできます。対応するバージョンに更新したい場合は、表示される確認メッセージで[はい]ボタンをクリックしてください。

バージョンアップにともなうプロセス制御ファンクションブロックの追加/変更点は、下記マニュアルの「バージョンアップ」を参照してください。

→ MELSEC iQ-R プログラミングマニュアル(プロセス制御FB/命令編)

- タグデータおよびプロセス制御ファンクションブロックのバージョンを更新すると、全変換(再割付)が必要となる場合があります。

- タグデータおよびプロセス制御ファンクションブロックのバージョンを更新すると、FBプロパティの初期値がデフォルトに戻ります。バージョンを更新する前にFBプロパティ管理(オフライン)で初期値をエクスポートし、更新後にインポートすることで、FBプロパティの初期値を再設定できます。(→ 498ページ

FBプロパティ管理(オフライン))

### 行の編集

#### ・行の追加選択した行の上に行を追加します。(複数選択可)

行を追加すると、各タグFB設定の割付けデバイスが変更されます。

**操作手順**

[編集] ⇒ [行追加(1 つ前)](：)を選択します。

#### ・行の削除選択した行を削除します。(複数選択可)

行を削除すると、各タグFB設定の割付けデバイスが変更されます。

**操作手順**

[編集] ⇒ [行削除](：)を選択します。

#### ・行の複製行をコピーまたはカットし、貼り付けして行を追加したあとに、[適用]ボタンをクリックすると、コピー元のFBプロパティ

の初期値を引き継ぎます。ただし、貼り付け後にタグFB型を変更した場合は引き継ぎません。

- 有効行/無効行の設定[ツール] ⇒ [オプション] ⇒ "変換" ⇒ "プロセス制御拡張設定" ⇒ "タグFB" ⇒ "最大タグ数"で設定した行数が有効行になり、それ以降が無効行になります。

- 有効行: タグFB設定のチェックと適用の対象となります。

- 無効行: タグFB設定のチェックと適用の対象となりません。また、各情報の編集はできません。

なお、タグFB設定エディタの[最大タグ数]ボタンをクリックすると、該当のオプションを確認および設定できます。

## 情報の入力

### タグ名

下記のパターンを除く名称を設定してください。

- 先頭に半角数字を持つラベル名

- デバイスと同一名のラベル名

- 使用できない文字列(予約語)を含むラベル名(→ ラベル名)

フェースプレートによるモニタ、および割付情報データベースファイルのエクスポートを行う場合は、半角英数字と日本語文字のみ使用してください。

### タグFB型

"タグFB型"欄の[ ]ボタンをクリックすると表示される"タグFB型選択"画面から選択します。

メーカ定義の各タグFB型の詳細は、下記を参照してください。

→ MELSEC iQ-R プログラミングマニュアル(プロセス制御FB/命令編)

ユーザ定義のタグFBの詳細は、下記を参照してください。

→ ユーザ定義タグFBの作成

### コメント

フェースプレートによるモニタ、および割付情報データベースファイルのエクスポートを行う場合は、半角英数字と日本語文字のみ使用してください。

### 変換+RUN中書込み

タグFBへの変更操作に対する、変換+RUN中書込みの可否、および初期化の有無を下記に示します。

○: 変換+RUN中書込可、×: 変換+RUN中書込不可

| タグFBへの変更操作 | | 変換+RUN中書込み | 初期化 |
|---|---|---|---|
| FBプロパティ | 初期値の変更 | ○ | 初期化されません。 |
| タグFB設定 | タグFBの新規追加 | ○ | 初期化されます。 |
| | タグ名の変更*1 | ○ | 初期化されません。 |
| | タグFB型の変更*1 | ×*2 | ― |
| | タグタイプの変更*1 | ×*2 | ― |
| | 宣言位置の変更*1 | ×*2 | ― |
| | コメントの変更*1 | ○ | 初期化されません。 |
| | タグFB設定の削除*1 | ○ | 初期化されません。 |

*1：適用済みのタグFBに対する変更操作が対象です。*2：データを変更すると、全変換(再割付)が必要になります。

## タグFBの登録により追加されるデータ

タグFB設定エディタでタグ名とタグFB型を入力し、適用するとグローバルラベルの"M+PTAG"内にタグFBインスタンスとタグデータが作成されます。

タグFBの定義およびタグデータの構造体定義は、読み取り専用の定義としてナビゲーションウィンドウに追加されます。

例タグ名が"TIC001"、タグFB型が"M+M_PID"の場合

| 追加されるデータ、定義 | 名称 | 作成場所 |
|---|---|---|
| タグFBインスタンス | TIC001_FB | ナビゲーションウィンドウ ⇒ "ラベル" ⇒ "グローバルラベル" ⇒ "M+PTAG" |
| タグFBの定義 | M+M_PID | ナビゲーションウィンドウ ⇒ "FB/FUN" ⇒ "(FBファイル名)" |
| タグデータ | TIC001 | ナビゲーションウィンドウ ⇒ "ラベル" ⇒ "グローバルラベル" ⇒ "M+PTAG" |
| タグデータの構造体定義 | M+TM_PID | ナビゲーションウィンドウ ⇒ "ラベル" ⇒ "構造体" |
| タグFBが参照するファンクションブロックの定義 | M+P_MCHG M+P_IN M+P_PHPL M+P_PID M+P_OUT1 | ナビゲーションウィンドウ ⇒ "FB/FUN" ⇒ "(FBファイル名)" |

注意事項

タグFBの定義、タグデータの構造体定義、およびユーザ定義タグFBの定義をナビゲーションウィンドウから削除すると、プロジェクトが不正になる場合があります。

タグFBの定義とタグデータの構造体定義、およびユーザ定義タグFBの定義を削除する場合は、文字列検索機能で削除する

FB定義名と構造体名を検索し、プロジェクト内で使用していないことを確認してから、削除してください。

### "M+PTAG"に登録されたタグFBインスタンスおよびタグデータ

グローバルラベル設定"M+PTAG"に登録されたタグFBインスタンスは、初期値の設定のみ可能です。

画面表示

ナビゲーションウィンドウ ⇒ "ラベル" ⇒ "グローバルラベル" ⇒ "M+PTAG"を選択します。

![図(元マニュアル p.352)](images/gxworks3/05-04-02.webp)

注意事項

"M+PTAG"は、プロジェクトから削除しないでください。

### FBプロパティの表示/設定

タグFBインスタンスの初期値の設定は、"FBプロパティ"ウィンドウからも可能です。

"FBプロパティ"ウィンドウの表示内容はタグFB設定エディタの選択行に連動します。

"FBプロパティ"ウィンドウの詳細は、下記を参照してください。

→ FBプロパティの表示/設定

"FBプロパティ"ウィンドウにはタグFB設定エディタで適用済みのタグFBインスタンスの初期値が表示されます。適用されていないタグFB、無効行のタグFB、または適用後に設定内容が変更されているタグFBの場合は表示されません。
