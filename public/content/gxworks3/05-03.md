# 5.3 ユニットラベルの登録

ユニットラベルの登録方法を説明します。

### グローバルラベルへの登録

ユニットラベルは、下記の操作後に表示されるユニットラベルを追加するかどうかの確認画面にてユニットの設定が"ユニットラベル: 使用する"になっている場合に登録されます。

- ユニット構成図でユニットを配置し、パラメータを確定する

- ナビゲーションウィンドウでユニットを新規追加する登録したユニットラベルは、ナビゲーションウィンドウのグローバルラベルおよび部品選択ウィンドウの[ユニット]タブに表示されます。

プログラム内でユニットラベルを使用する場合はユニットラベル名を直接入力、または部品選択ウィンドウからドラッグ&

ドロップで挿入してください。

ユニットラベルを任意の名前で使用する場合は、ユニットラベルにエイリアスを設定して使用してください。

**注意事項**

- ユニットラベル登録時の注意事項登録したユニットラベルはすべてCPUユニットへ書き込まれるため、CPUユニットのメモリ容量をオーバすることがあります。その場合は下記を行ってください。

- グローバルラベルの書込み先をSDメモリカードにする

- FX5CPUではSDメモリカードに書き込む場合でも、書き込める容量はCPUユニットと同じです。

- ラベル名に"zReserve"が付くユニットラベルについてユニットラベルの"ラベル名"の中で先頭文字が"zReserve"となっているユニットラベルにデータを書き込まないでください。

本ラベルに対するデータを書き込むと、シーケンサシステムが誤動作する危険性があります。

例"インスタンス名"_"ユニット番号"."ラベル名" GF11_1.zReserveAreaSB00007

- モーション制御設定機能で設定した公開ラベルを登録する場合モーション制御設定機能で下記のメニューを選択すると、公開ラベルをユニットラベルとしてGX Works3のグローバルラベルに登録できます。

- [変換] ⇒ [公開ラベルの反映]

登録すると、GX Works3のグローバルラベル、およびグローバルラベルが使用されているプログラムが未変換状態になります。公開ラベルの詳細は、モーション制御設定機能のヘルプを参照してください。

### ユニットラベルの編集

グローバルラベルに登録されたユニットラベル(M+Global)をラベルエディタ上で編集する場合、下記の操作のみ行えます。

- 行を選択しての削除

- コメントの編集ラベルエディタ上でラベルの編集、追加はしないでください。

### 削除したユニットラベルの再登録

グローバルラベルから削除したユニットラベルを再度登録する場合、部品選択ウィンドウの[ユニット]タブ内に表示されているユニットの名前を選択し、右クリック ⇒ ショートカットメニュー[ユニットラベルの追加]を選択してください。

### 構造体の編集

ユニットラベルを登録するとユニットラベル内の構造体の定義がナビゲーションウィンドウの"構造体"に登録されます。

構造体定義はナビゲーションウィンドウ上でコピーできます。コピーされたユニットラベルの構造体定義は、定義名の"+"が"_"に変更されます。

### 注意事項

ユニットラベル使用時、そのユニットの先頭I/O No.を変更するとプログラムが未変換状態になり、全変換(再割付)が必要になります。
