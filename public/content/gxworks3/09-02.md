# 9.2 命令の検索/置換

プログラム中の命令を検索/置換します。

対応エディタ: ラダーエディタ、STエディタ、FBD/LDエディタ(LD部品、FBD部品(FB/FUN)のみ)、SFC図エディタラダープログラムで使用しているNOP命令は検索できません。

出力命令として、下記の応用命令も検索の対象です。

SET、RST、PLS、PLF、FF、SFT、SFTP、MC

画面表示

[検索/置換] ⇒ [命令検索]()/[命令置換]/[接点コイル検索]()を選択します。

![図(元マニュアル p.562)](images/gxworks3/09-02-01.webp)

(1): 回路記号選択欄

(2): デバイス命令入力欄操作手順各項目を設定し、[次を検索]ボタンをクリックします。

下記のメニューを使用することで、過去に検索/置換した位置にカーソルを戻したり進めたりできます。

- [検索/置換] ⇒ [前に戻る]()/[次に進む]()

"検索対象"の"FBインスタンスの入出力ラベルを含む"を選択すると、ラダープログラムおよびFBD/LDプログラムで使用しているFBインスタンスの入出力ラベルも検索/置換されます。

| 命令の検索例 | |
|---|---|
| 命令指定例*1 | 検索対象例 |
| MOV | MOV、MOVP |
| MOVP | MOVP |
| MOV D0 K4Y0 | MOV D0 K4Y0、MOVP D0Z1 K4Y0、MOV D0 K4Y0Z1、MOVP D0Z1 K4Y0Z1 |
| MOVP D0 J1¥W0 | MOVP D0 J1¥W0、MOVP D0Z1 J1¥W0、MOVP D0 J1Z1¥W0Z1、MOVP D0Z1 J1Z1¥W0Z1 |

*1引数を指定した検索はラダープログラムのみ可能です。
