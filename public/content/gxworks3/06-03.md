### ラダーエディタの分割

ラダーエディタを分割する手順を示します。

#### ・上下分割

**操作手順**

分割バー(横)を下へドラッグまたはダブルクリックします。

または、下記のメニューを使用します。

- [ウィンドウ] ⇒ [分割]

#### ・左右分割

**操作手順**

分割バー(縦)を左へドラッグまたはダブルクリックします。

![図(元マニュアル p.368)](images/gxworks3/06-03-03.webp)

ラダーエディタを分割すると、エディタごとにスクロールできるため、プログラム内の離れた位置にある回路を同時に表示できます。

![図(元マニュアル p.368)](images/gxworks3/06-03-04.webp)

下記のメニューを使用すると、左右のエディタを垂直方向へ同時にスクロールできます。(水平方向へは同時にスクロールしません。)

- [ウィンドウ] ⇒ [分割ウィンドウ操作] ⇒ [左右を同時にスクロール]

### 接点、コイル、命令、引数の入力

**操作手順**

- 回路入力ダイアログからの入力1.入力または編集するセルを選択後、ダブルクリックするか[Enter]を押します。

挿入モード([Insert]キーで上書きモードに切替え可)では、カーソル位置に命令が挿入されるため、選択しているセルにすでに命令が入力されていた場合、その命令セルは後ろに移動します。

2. 回路入力ダイアログで命令と引数を入力します。

回路入力ダイアログでは下記の操作を行えます。

- [Ctrl]+[←]/[→]/[↑]/[↓]を押すと、ラダーエディタ上のカーソルを移動できます。

- [Alt]+[←]/[→]を押すと、選択する命令/引数を切り替えられます。

- [Ctrl]+[Space]を押すと、命令/ラベルの候補が表示されます。

さらに、下記のオプションを"はい"に設定すると、デバイスコメント、ラベルコメント、および命令の説明を入力することで、それらのデータが設定されているデバイス、ラベル、および命令を候補として表示できます。

- [ツール] ⇒ [オプション] ⇒ "編集" ⇒ "命令/デバイス/ラベル候補表示" ⇒ "候補表示設定" ⇒ "デバイスコメント入力時に対応するデバイスを表示する"

- 説明付き"回路入力"画面からの入力1.回路入力ダイアログの[拡張表示]ボタンをクリックします。

2. "回路入力"画面で命令を選択し、引数を入力します。

"回路入力"画面では、よく使用する命令をお気に入りに追加できます。

命令を選択後、右クリック ⇒ ショートカットメニュー[お気に入りに追加]をクリックすると、選択した命令がお気に入りに追加されます。

"一覧"のプルダウンリストの1番下から"お気に入り"を選択することで、お気に入りに追加した命令を確認できます。

- メニュー/ツールバーからの挿入1.ラダーエディタ上の追加するセルを選択します。

2. [編集] ⇒ [回路記号] ⇒ [(回路)]を選択します。または、ツールバーの回路記号を選択します。

- 部品選択ウィンドウからの挿入1.部品選択ウィンドウに表示されている接点、コイル、または命令をラダーエディタにドラッグ&ドロップします。

### HTMLファイルへエクスポート

ラダープログラムの回路図を、HTMLファイルへエクスポートします。

なお、HTMLファイル内のデータは、ラダープログラムにインポートできません。

**操作手順**

1. ラダーエディタを開き、[編集] ⇒ [ファイルへエクスポート]を選択します。

2. "ファイルへエクスポート"画面でエクスポートするファイルのファイル名を入力します。

3. "ファイルの種類"に"HTML(*.html)"を指定し、[保存]ボタンをクリックします。

注意事項

プログラムに未変換回路がある場合、プログラムをエクスポートできません。

インラインSTボックスの表示行数(高さ)はHTMLファイルに反映されません。

また、下記項目はエクスポートされません。

- インラインSTプログラムの行番号

- 周辺ステートメント入力時に付加される"*"

- ファンクション/ファンクションブロックの引数のデータ型

- 動作環境動作環境を下記に示します。

| 項目 | | 内容 |

| OS | パソコン | Microsoft® Windows 11 |

| ブラウザ | パソコン | HTML5、CSS3に対応するブラウザ 下記のブラウザを推奨します。 Microsoft Edge® Internet Explorer® 11以降 Safari 8.0以降 Google Chrome™ 4.3以降 |

### テキストファイルへエクスポート

ラダープログラムの回路図を、テキストファイルへエクスポートします。

なお、テキストファイル内のデータは、ラダープログラムにインポートできません。

本機能は、プログラムサイズが最大260KBまでのプログラムをサポートしています。

**操作手順**

1. ラダーエディタを開き、[編集] ⇒ [ファイルへエクスポート]を選択します。

2. "ファイルへエクスポート"画面でエクスポートするファイルのファイル名を入力します。

3. "ファイルの種類"に"テキスト(*.txt)"を指定し、[保存]ボタンをクリックします。

注意事項

プログラムに未変換回路がある場合、プログラムをエクスポートできません。

インラインSTボックスの表示行数(高さ)はテキストファイルに反映されません。

また、下記項目はエクスポートされません。

- インラインSTプログラムの行番号

- 動作環境テキストエディタで文字が表示されない場合や、回路図の表示幅が均一にならない場合は動作環境を確認してください。

下記の場合、正常に表示されない場合があります。

- 日本語版以外のOSを使用している場合

- 日本語のフォントをインストールしていない場合

- シフトJISコードで表示できないフォントを使用している場合エクスポートしたテキストファイルを正常に表示するためには、下記の設定を行ってください。

- 日本語のフォントをインストールし、システムロケールを日本語に設定する。

- テキストエディタで、使用するフォントを等幅フォントに設定する。

- ファイルのフォーマット保存したテキストファイルをテキストエディタで開くと、下記のように表示されます。
