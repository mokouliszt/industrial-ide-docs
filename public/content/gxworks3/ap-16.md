# 付16 アドオン機能

GX Works3およびMXコントローラにアドオンとして提供されるファイルをインストールすることにより、機能を拡張できます。

MXコントローラのシミュレーション中は本機能を使用できません。

## アドオン管理(CPUユニット)

"アドオン管理"画面では、CPUユニットにアドオンをインストールできます。

また、インストールしたアドオンを有効化/無効化できます。

画面表示

[ツール] ⇒ [アドオン] ⇒ [CPUユニットアドオン管理]を選択します。

![図(元マニュアル p.1305)](images/gxworks3/ap-16-01.webp)

表示内容

| 項目 | 内容 |
|---|---|
| (1)[インストール/照合]ボタン | "アドオン管理インストール/照合"画面を表示します。 |
| (2)[有効/無効設定]ボタン | "アドオン管理有効/無効"画面を表示します。 |
| (3)アドオン一覧 | MXコントローラにインストールされているアドオンの一覧が表示されます。 |
| (4)[詳細情報]ボタン | "アドオン管理 アドオン詳細情報"画面を表示します。 |
| (5)空き容量 | MXコントローラの空き容量が表示されます。 |
| (6)[アンインストール]ボタン | アドオン一覧の"選択"列にチェックが入っているアドオンをMXコントローラからアンインストールします。 |

アドオンのインストール状態および有効/無効の設定は、CPUユニットごとに保存されます。

### アドオンのインストール

"アドオン管理インストール/照合"画面で、MXコントローラにアドオンをインストールする操作手順を示します。

MXコントローラには、最大64個までアドオンをインストールできます。

画面表示

"アドオン管理"画面の[インストール/照合]ボタンをクリックします。

![図(元マニュアル p.1306)](images/gxworks3/ap-16-02.webp)

**操作手順**

1. [ファイル選択]ボタンをクリックします。

2.表示された画面でアドオン(*.apk)を選択し、[開く]ボタンをクリックします。(複数選択可)

アドオンが"アドオン管理インストール/照合"画面に表示されます。

3.インストールするアドオンの"選択"列にチェックを入れます。(複数選択可) 4. [インストール]ボタンをクリックします。

### アドオンの有効化/無効化

"アドオン管理有効/無効"画面で、MXコントローラ内のアドオンを有効化/無効化する操作手順を示します。

画面表示

"アドオン管理"画面の[有効/無効設定]ボタンをクリックします。

![図(元マニュアル p.1307)](images/gxworks3/ap-16-03.webp)

**操作手順**

1.有効化するアドオンの"有効"列にチェックを入れ、無効化するアドオンの"有効"列からチェックをはずします。

2. [更新]ボタンをクリックします。

### アドオンの照合

MXコントローラにインストールしたアドオンとパソコン上のアドオンを照合することで、アドオンがMXコントローラに正常に書き込まれているかどうかを確認できます。

"アドオン管理インストール/照合"画面で、アドオンを照合する操作手順を示します。

画面表示

"アドオン管理"画面の[インストール/照合]ボタンをクリックします。

![図(元マニュアル p.1308)](images/gxworks3/ap-16-04.webp)

**操作手順**

1. [ファイル選択]ボタンをクリックします。

2.表示された画面でアドオン(*.apk)を選択し、[開く]ボタンをクリックします。(複数選択可)

アドオンが"アドオン管理インストール/照合"画面に表示されます。

3.照合するアドオンの"選択"列にチェックを入れます。(複数選択可) 4. [照合]ボタンをクリックします。

MXコントローラ内のアドオンの有効/無効にかかわらず照合できます。

### アドオンのアンインストール

"アドオン管理"画面で、MXコントローラからアドオンをアンインストールする操作手順を示します。

**操作手順**

1."アドオン管理"画面で、アンインストールするアドオンの"選択"列にチェックを入れます。(複数選択可) 2. [アンインストール]ボタンをクリックします。

### アドオン一覧のエクスポート

"アドオン管理 アドオン詳細情報"画面で、MXコントローラにインストールされているアドオンの情報をCSVファイルにエクスポートする操作手順を示します。

画面表示

"アドオン管理"画面の[詳細情報]ボタンをクリックします。

![図(元マニュアル p.1309)](images/gxworks3/ap-16-05.webp)

**操作手順**

1. [エクスポート]ボタンをクリックします。

2.表示された画面でファイル名を入力します。

3. [保存]ボタンをクリックします。

## アドオン管理(GX Works3)

"アドオン管理(GX Works3)"画面では、GX Works3にアドオンをインストールできます。

また、インストール済みのアドオンを一覧で表示できます。

画面表示1.開いているすべてのプロジェクトを閉じます。(プロジェクトを開いていない場合は操作不要です。) 2. [ツール] ⇒ [アドオン] ⇒ [GX Works3アドオン管理]を選択します。

![図(元マニュアル p.1310)](images/gxworks3/ap-16-06.webp)

表示内容

| 項目 | 内容 |
|---|---|
| (1)[インストール]ボタン | "アドオン管理(GX Works3)インストール"画面を表示します。 |
| (2)アドオン一覧 | GX Works3にインストールされているアドオンの一覧が表示されます。 |
| (3)[詳細情報]ボタン | "アドオン管理(GX Works3)アドオン詳細情報"画面を表示します。 |
| (4)[アンインストール]ボタン | アドオン一覧の"選択"列にチェックが入っているアドオンをGX Works3からアンインストールします。 |

- アドオンのインストール状態は、パソコンごとに保存されます。

- アドオンの有効/無効の設定は、プロジェクトごとに保存されます。

### アドオンのインストール

"アドオン管理(GX Works3)インストール"画面で、GX Works3にアドオンをインストールする操作手順を示します。

画面表示

"アドオン管理(GX Works3)"画面の[インストール]ボタンをクリックします。

![図(元マニュアル p.1311)](images/gxworks3/ap-16-07.webp)

**操作手順**

1. [ファイル選択]ボタンをクリックします。

2.表示された画面でアドオン(*.gx3apk)を選択し、[開く]ボタンをクリックします。(複数選択可)

アドオンが"アドオン管理(GX Works3)インストール"画面に表示されます。

3.インストールするアドオンの"選択"列にチェックを入れます。(複数選択可) 4. [インストール]ボタンをクリックします。

### アドオンの有効化/無効化

GX Works3にインストールされているアドオンは、"アドオン管理(GX Works3)有効/無効"画面で有効化/無効化できます。

画面表示1.アドオンを有効化/無効化したいプロジェクトを開きます。

2. [プロジェクト] ⇒ [アドオン有効無効設定]を選択します。

![図(元マニュアル p.1312)](images/gxworks3/ap-16-08.webp)

**操作手順**

1.有効化するアドオンの"有効"列にチェックを入れ、無効化するアドオンの"有効"列からチェックをはずします。

2. [更新]ボタンをクリックします。

注意事項

アドオンを有効化/無効化するとき、プロジェクトで有効なアドオンのバージョンがGX Works3にインストールされたアドオンのバージョンより古い場合、プロジェクトで有効なアドオンをバージョンアップします。

バージョンアップ後、CPUユニットと通信するには下記の操作を行ってください。

- プロジェクトで有効なアドオンをCPUユニットにインストールし、有効化する。

- CPUユニットに書きこまれているデータを削除する。

また、アドオンをバージョンアップした後のプロジェクトは、古いバージョンのアドオンがインストールされたGX Works3

では使用できません。

使用するにはプロジェクトで有効なアドオンと同じバージョン、または新しいバージョンのアドオンをGX Works3にインストールしてください。
