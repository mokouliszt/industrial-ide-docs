# 15.8 不正な読出し/書込みの防止(ファイルパスワード)

CPUユニットおよびSDメモリカード内のファイルの不正な読出し/書込みを防止するには、ファイルパスワード機能を使用します。

MXコントローラでは対応していません。

## ファイルパスワード機能

本機能を使用してCPUユニットおよびSDメモリカード内のファイルにパスワードを登録することで、ファイルを読み出す/

書き込む操作を禁止できます。

読出しと書込みのどちらかの操作のみを禁止することも、両方の操作を禁止することもできます。

### ファイル

パスワードを登録できるファイルを示します。

○: 登録可、×: 登録不可

| ファイル名 | 登録可否 |
|---|---|
| システムパラメータ、CPUパラメータ、ユニットパラメータ、ユニット拡張パラメータ、メモリカードパラメータ | ○ |
| リモートパスワード | × |
| グローバルラベル設定 | ○ |
| グローバルラベル割付情報 | × |
| グローバルラベル初期値 | × |
| ローカルラベル初期値 | × |
| プログラムファイル | ○ |
| プログラム部品(FBファイル/FUNファイル) | ○ |
| ファイルレジスタ | × |
| デバイス初期値 | ○ |
| 共通デバイスコメント | ○ |
| プログラム別デバイスコメント | ○ |
| ファームウェアアップデート禁止ファイル | ○*1 |
| 異常データベース確認ファイル | × |
| CPUユニットのバックアップ用システムファイル | × |

*1：パソコンに装着されたSDメモリカード内のファイルにはパスワードを登録できません。

### パスワード認証

ファイルにパスワードを登録すると、オンライン操作を行うときにパスワードを認証する必要があります。

パスワード認証が必要なオンライン操作を示します。

○: パスワード認証が必要、―: パスワード認証が不要

| オンライン操作 | 認証要否 | 対応するパスワードの種類 |
|---|---|---|
| シーケンサからの読出し | ○ | 読出し禁止パスワード |
| シーケンサへの書込み | ○ | 書込み禁止パスワード |
| RUN中の書込み(変換+RUN中書込み、ファイル一括RUN中書込み) | ○ | 書込み禁止パスワード |
| シーケンサとの照合 | ○ | 読出し禁止パスワード |
| ファイルパスワード設定(登録/変更/認証/削除) | ○ | 読出し禁止パスワード、書込み禁止パスワード |
| シーケンサのデータ削除 | ○ | 書込み禁止パスワード |
| CPUメモリ操作(初期化) | ― | ― |

注意事項

パスワード認証を一定回数失敗した場合、一定時間パスワード認証が拒否(ロックアウト)されます。

### 外部からのアクセス

GX Works3以外の機能(FTPサーバ機能およびMCプロトコル)を使用してパスワードが登録されたRCPU内のファイルにアクセスする場合、パスワードの認証が必要です。詳細は、下記を参照してください。

| 機能 | 参照 |
|---|---|
| FTPサーバ | → MELSEC iQ-R Ethernetユーザーズマニュアル(応用編) |
| MCプロトコル | → MELSEC iQ-R シリアルコミュニケーションユニットユーザーズマニュアル(応用編) |

### 注意事項

- パスワードは大切に保管してください。

パスワードを紛失した場合、または"解除できないプロテクト"を設定したファイルの保護を解除する場合、CPUユニットを初期化し(→ メモリ使用状況の確認)、プロジェクトを新たにCPUユニットに書き込んでください。

## ファイルパスワード機能の使用手順

### セキュリティを有効にする手順

#### ・CPUユニット/CPUユニットに装着されたSDメモリカード

**操作手順**

1.パソコンとCPUユニットを接続します。(→ 接続先指定について) 2.ファイルにパスワードを登録します。(→ ファイルパスワードの設定)

#### ・パソコンに装着されたSDメモリカード

**操作手順**

ファイルにパスワードを登録します。(→ ファイルパスワードの設定)

### セキュリティでロックされているファイルを読み書き可能にする手順

#### ・CPUユニット/CPUユニットに装着されたSDメモリカード

**操作手順**

1.パソコンとCPUユニットを接続します。(→ 接続先指定について) 2.ファイルに登録されているパスワードを認証します。(→ ファイルパスワードの設定)

#### ・パソコンに装着されたSDメモリカード

**操作手順**

ファイルに登録されているパスワードを認証します。(→ ファイルパスワードの設定)

## ファイルパスワードの設定

ファイルにパスワードを登録、登録したパスワードを変更、認証、および削除します。

![図(元マニュアル p.861)](images/gxworks3/15-08-01.webp)

画面表示

- CPUユニット/CPUユニットに装着されたSDメモリカード[プロジェクト] ⇒ [セキュリティ] ⇒ [ファイルパスワード設定]を選択します。

![図(元マニュアル p.861)](images/gxworks3/15-08-02.webp)

- パソコンに装着されたSDメモリカード"メモリカード操作"画面の[設定] ⇒ [ファイルパスワード設定]を選択します。

![図(元マニュアル p.861)](images/gxworks3/15-08-03.webp)

**操作手順**

1.ファイルを選択し、[登録]/[変更]/[認証]/[削除]ボタンをクリックします。

2.表示された画面で各項目を設定し、[完了]ボタンをクリックします。

3."ファイルパスワード設定"画面の[設定]ボタンをクリックします。

- 解除できないプロテクト(FX5CPU) FX5CPU内のファイルは、パスワードを登録せずに読出し/書込みを禁止できます。ただし、一度禁止すると解除できません。

禁止するには、"パスワード登録"画面の"解除できないプロテクト"にチェックを入れてください。

設定できるファイルは、下記を参照してください。

→ ファイル

注意事項

- FX5CPUを設定したプロジェクトを開いている場合、パソコンに装着したSDメモリカード内のファイルにはパスワードを登録できません。また、パスワードの変更、削除、および認証もできません。

- CPUユニットまたはCPUユニットに装着されたSDメモリカード内のファイルにパスワードを登録する場合、あらかじめパソコンとCPUユニットを接続してください。

- CPUユニットの安全動作モードによって、行える操作が異なります。

テストモード: すべての操作セーフティモード: 認証のみ

- "解除できないプロテクト"にチェックを入れたファイルには、読出しと書込みの両方の操作を禁止できますが、すでにどちらか一方のみが設定されている場合は、後からもう一方の操作を禁止することはできません。
