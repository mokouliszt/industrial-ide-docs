# 12.6 GOT経由(GOTトランスペアレント機能への対応)

GOTトランスペアレント機能を使用して、パソコンからCPUユニットにアクセスするときの設定方法を説明します。

**注意事項**

- パソコンからのオンライン操作実行時GOTトランスペアレント機能を使用してパソコンからCPUユニットへのオンライン操作中は、GT Designer2またはGT

Designer3からGOTへのオンライン操作(プロジェクトデータのダウンロードなど)を行わないでください。

- GOTが正常にモニタしていない場合下記の場合には、トランスペアレント機能は使用できません。

- CPUユニットの異常、またはCPUユニットとGOT間の通信状態の異常により、GOTが正常にモニタしていない場合

- CPUユニットまたはGOTの電源ONもしくはリセットからGOTのモニタが開始するまでの間

GOTが正常にモニタしていない場合には、下記を確認してください。

| 項目 | 参照 |
|---|---|
| CPUユニットが正常に動作しているか | → ユニットの診断 |
| CPUユニットとGOTが正常に接続されているか | → 使用するGOTのマニュアル |

## GOT経由

GOTを経由してパソコン(GX Works3)からCPUユニットへアクセスできます。

![図(元マニュアル p.689)](images/gxworks3/12-06-01.webp)

GOTのシリーズ、およびパソコンとGOT間の接続形態により接続可否、接続先指定方法が異なります。

**関連項目**

→ GOT3000シリーズ本体取扱説明書(接続編)

→ GOT2000シリーズ接続マニュアル(三菱電機機器接続編) GT Works3 Version1対応

→ GOT1000シリーズ接続マニュアル(三菱電機機器接続編) GT Works3対応

## GOTとユニット経由

GOTとユニット(シリアルコミュニケーションユニット、CC-Link IEコントローラネットワークユニット、CC-Link IEフィールドネットワークユニット、Ethernetユニット、およびCC-Link IE TSNユニット)を経由して、パソコンからCPUユニットへアクセスできます。

GOTのシリーズ、およびパソコンとGOT間の接続形態により接続可否、接続先指定方法が異なります。

**関連項目**

→ GOT3000シリーズ本体取扱説明書(接続編)

→ GOT2000シリーズ接続マニュアル(三菱電機機器接続編) GT Works3 Version1対応

→ GOT1000シリーズ接続マニュアル(三菱電機機器接続編) GT Works3対応
