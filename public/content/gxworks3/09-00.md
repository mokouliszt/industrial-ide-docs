- "検索と置換"ウィンドウは、[表示][ドッキングウィンドウ][検索/置換]()から開くこともできます。

- 文字の半角/全角は区別して検索します。

- 検索/置換種類の切換えは、"検索と置換"ウィンドウのプルダウンリストからも行えます。

- ラベルを置換する場合、置換できるデバイスは1点分です。

- インラインSTボックス内の検索開始位置は、カーソル位置にかかわらず、インラインSTボックスの先頭からです。

注意事項

- 置換後は、未変換状態になります。

- ラベル名を置換する場合は、置換後の名前がすでに使用されていないかを確認してから行ってください。

- モニタ中は置換できません。モニタを終了してから行ってください。ただし、ラダーエディタのモニタ書込みモードの場合は置換できます。

- 読出し禁止のデータは、検索/置換の対象外です。セキュリティを解除するなど、編集可能な状態にして操作してください。

### エディタでの範囲の指定について

ラダーエディタ、FBD/LDエディタ、およびデバイスコメントエディタにおいて、すべて検索およびすべて置換を実行する場合の範囲を指定できます。

指定できる範囲はエディタ上での操作によって異なります。

| 操作 | 範囲 |
|---|---|
| • ラダーエディタ/デバイスコメントエディタ: 1つのセルのみを選択 • FBD/LDエディタ: 1つの部品のみを選択 | エディタ内のすべてのセルまたは部品 |
| • ラダーエディタ: 複数のセルを選択 • FBD/LDエディタ: 複数の部品を選択 • デバイスコメントエディタ: 行または複数のセルを選択 | 選択したセル、部品、または行 |
| • デバイスコメントエディタ: 列を選択 | 選択した列(表示されていないビット指定情報を含む) |
| • デバイスコメントエディタ: 全選択(+) | すべて(表示されていない複数コメントの列を除く) |

- を押しながらセルまたは部品を選択すると、離れている複数のセルまたは部品を選択できます。

- デバイスコメントエディタで複数のセルをドラッグして選択した場合、エディタ上に表示されていないビット指定情報も検索または置換されます。

### 結果とエラーログの表示について

すべて検索/すべて置換をした場合に、結果とエラーログが表示されます。

画面表示

"検索と置換"ウィンドウで[すべて検索]/[すべて置換]ボタンをクリックします。

![図(元マニュアル p.558)](images/gxworks3/09-00-01.webp)

表示内容

| 項目 | 内容 |
|---|---|
| (1)ステータス表示 | 検索/置換を行った結果のログ数、検索エラーが表示されます。 |
| (2)検索/置換結果表示内容 | 検索/置換を行った場所を特定する位置情報が表示されます。 |
| (3)サブステータスバー | 検索/置換を行った結果が表示されます。 |
| (4)エラーログ表示内容 | 検索/置換を行った結果のエラー内容が表示されます。 |

検索/置換結果またはエラーログの任意の行から、該当文字列へジャンプできます。

ジャンプする場合、任意の行を選択後、右クリックショートカットメニュー[選択場所にジャンプ]/[前の場所にジャンプ]/[次の場所にジャンプ]を選択、またはダブルクリックしてください。

また、"検索結果"画面に表示されている任意の行をハイライト表示できます。

→ ハイライト表示
