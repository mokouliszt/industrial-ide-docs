# 16.2 センサ・機器の状態の確認

ネットワークに接続されているセンサ・機器の状態および現在値を表示します。

対応ネットワーク: Ethernet(CPUユニット)*1、AnyWireASLINK、CC-Link IEフィールドネットワーク、CC-Link

*1 MXコントローラでは対応していません。

画面表示1. [診断] ⇒ [センサ・機器モニタ]を選択します。2."ユニット選択(センサ・機器モニタ)"画面で、マスタユニットまたはマスタ局を選択し、[OK]ボタンをクリックします。

CC-Link IEフィールドネットワークのマスタ局を選択した場合の画面例です。

![図(元マニュアル p.876)](images/gxworks3/16-02-01.webp)

操作手順

センサ・機器モニタ画面の局一覧(1)または機器構成図(2)で、モニタ対象のiQSS対応機器を選択します。

"モニタ情報"ウィンドウ(3)に、選択したiQSS対応機器の状態が表示されます。

- iQSS対応機器のエラー状態は、ネットワークの診断機能で確認してください。

- センサ・機器モニタ機能は、1度に多くの情報をCPUユニットから読み出します。そのため、設定した通信経路によってはセンサ・機器モニタの処理速度が低下する場合があります。

注意事項

通信経路にMELSEC-Q/Lシリーズのネットワークユニットが含まれる場合、下記のユニットはセンサ・機器モニタを起動できません。

- Ethernet内蔵形CPU

- CC-Link IEフィールドネットワークユニット

### ブリッジユニットに接続しているセンサ・機器の状態の確認

CC-Link IE TSN―AnyWireASLINKブリッジユニットに接続しているセンサ・機器の状態を確認する場合、下記の操作でセンサ・機器モニタ画面を表示できます。

FX5CPUでは対応していません。

操作手順1. [診断] ⇒ [CC-Link IE TSN/CC-Link IE Field診断]を選択します。2."CC-Link IE TSN/CC-Link IE Field診断"画面でCC-Link IE TSN―AnyWireASLINKブリッジユニットを選択します。3.右クリック ⇒ ショートカットメニュー[AnyWireASLINKセンサ・機器モニタ]を選択します。

詳細は、下記を参照してください。

→ iQ Sensor Solution リファレンスマニュアル

### ブリッジユニットに接続している特小無線機器の状態の確認

ブリッジユニット(NZ2WM-GFB)に接続している特小無線機器は、下記の操作でセンサ・機器モニタ画面を表示できます。

詳細は、下記を参照してください。

→ CC-Link IEフィールドネットワーク-特小無線ブリッジユニットユーザーズマニュアル(応用編)

操作手順1.センサ・機器モニタ画面の局一覧(1)または機器構成図(2)で、ブリッジユニット(NZ2WM-GFB)を選択します。2. [センサ・機器モニタ] ⇒ [システム構成を開く] ⇒ [特小無線構成を開く]を選択します。

"モニタ情報"ウィンドウ(3)に、選択した特小無線機器の状態が表示されます。
