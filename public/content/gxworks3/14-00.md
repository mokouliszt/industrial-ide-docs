# 14.0 プログラムの動作確認(機能概要)

## 14 プログラムの動作確認

パソコンとCPUユニットを接続し、CPUユニットおよびインテリジェント機能ユニットの実行状態を確認する方法を説明します。

実行状態を確認するための機能を示します。

| 目的 | 機能名 | 参照 |
|---|---|---|
| 実行中のプログラムの状況をプログラムエディタ上で確認 する。 | モニタ | → 実行プログラムをプログラムエディタ上で 確認する |
| デバイス、バッファメモリの現在値を一括で確認する。 | デバイス/バッファメモリ一括モニタ | → デバイス/バッファメモリを一括で確認す る |
| デバイス、ラベルを登録して現在値を確認する。 | ウォッチ | → デバイス/ラベルを登録して現在値を確認 する |
| CPUユニットに登録されている強制的にON/OFFする入出 力デバイスを確認する。 | 強制入出力登録/解除*1 | → 強制的に入出力デバイスをON/OFFする |
| タイミングを指定してデバイス/ラベルの値を変更し、プロ グラムの動作を確認する。 | 実行条件付きデバイステスト*2 | → 条件を設定してデバイス/ラベルの値を変 更する |
| FBプロパティの現在値を初期値に反映する。 | FBプロパティ管理(オンライン) | → FBプロパティの現在値を初期値に反映する |
| タグデータを確認する。 | フェースプレート*3 | → タグデータを計器ウィンドウ(フェースプ レート)で確認する |
| | PX Developerモニタツール連携*3 | → PX Developerモニタツール連携 |
| プログラムの実行中に、プロセス制御ファンクションブ ロックを一時停止または再開する。 | FBの停止/再開*3 | → ファンクションブロックを一時停止/再開 する |
| プログラムの任意の区間でスキャンタイムを測定する。 | スキャンタイム測定*4 | → プログラムのスキャンタイムを測定する |
| 実行中のプログラムの処理時間を確認する。 | プログラム一覧モニタ*2 | → プログラムの処理時間を確認する |
| プログラムで使用されている割込みプログラムの実行回数 を確認する。 | 割込みプログラム一覧モニタ*2 | → 割込みプログラムの実行回数を確認する |
| MXコントローラのコアごとの演算周期の内訳や負荷状態 を確認する。 | パフォーマンスモニタ*5 | → コアの負荷状況を確認する |
| インテリジェント機能ユニットのユニット情報を登録し て、入出力信号、バッファメモリの現在値を確認する。 | インテリジェント機能ユニットモニタ | → インテリジェント機能ユニットの現在値を 確認する |
| CPUユニットで収集した現在のロギングデータを確認する。 | リアルタイムモニタ | → 現在のロギングデータを確認する |
| 軸および軸グループに依存しない演算周期などのCPUユ ニットの情報を確認する。 | モーション共通モニタ*5 | → モーション共通モニタ |
| 実行中の運転軸の現在値およびエラーコードを一括で確認 する。 | 軸モニタ*5 | → 軸モニタ |
| 軸の位置データ履歴を確認する。 | 位置データ履歴*5 | → 位置データ履歴 |

*1 MXコントローラでは使用できません。*2 MXコントローラおよびFX5CPUでは使用できません。*3プロセス制御拡張を有効にしているRnPCPU(プロセスモード/二重化モード)およびRnPSFCPUを設定したプロジェクトで使用できます。*4 RnCPU、RnENCPU、およびRnSFCPUで使用できます。*5 MXコントローラでのみ使用できます。

### 現在値の変更

デバイス、ラベル、およびバッファメモリの現在値を変更して動作を確認するには、現在値変更機能を使用します。

CPUユニットのビットデバイスを強制的にON/OFFします。また、ワードデバイス/バッファメモリの現在値を強制的に変更します。

現在値の変更は下記の画面で行います。

- プログラムエディタ

- "デバイス/バッファメモリ一括モニタ"ウィンドウ

- ウォッチウィンドウ

- インテリジェント機能ユニットモニタウィンドウただし、安全デバイス/ラベルおよび一般/安全共有ラベルは、CPUユニットの安全動作モードがテストモードの場合にのみ変更可能です。

- "CPUパラメータ"の"デバイス・ラベルアクセスサービス処理設定"に"プログラム間とEND処理で行う"を設定している場合、プログラムの実行後またはEND処理実行時に現在値が変更されます。

- 指定プログラムモニタ機能でプログラムを選択している場合、選択したプログラムを実行後に現在値が変更されます。指定プログラムモニタ機能の詳細は下記を参照してください。

→ 指定したプログラムのデバイス/ラベルをモニタする場合注意事項

CPUユニットの外部からのデバイス書込み禁止機能を使用している場合、書込みを禁止しているデバイスの現在値は変更できません。

外部からのデバイス書込み禁止機能の詳細は、下記を参照してください。
