# 9.3 文字列の検索/置換

プロジェクト内のデータの文字列を検索/置換します。

| 対象エディタ | 検索対象の文字列 | 置換可能な文字列 |
|---|---|---|
| ラダーエディタ | 下記を除く文字列*1 / エディタ上に表示された割付デバイス*2 | 命令名を除く文字列 |
| STエディタ | 下記を除く文字列 / モニタ部(モニタ中に画面右側に表示) / 折りたたみ表示の簡易表示部分(例: IF END IF) _ | 命令名を除く文字列 |
| FBD/LDエディタ | 下記を除く文字列 / エディタ上に表示された割付デバイス*2 / インラインST部品のプレビュー表示エリアに表示されるインラインSTプログラム | 下記を除く文字列 / ファンクションのデータ型 / ファンクションブロックのデータ型 / FB/FUNの入出力ラベル / リターン部品の"RETURN"の文字列 / オプション*2により、エディタ上に表示された割付デバイス / インラインST部品のプレビュー表示エリアに表示されるインラインSTプログラム |
| SFC図エディタ | 下記を除く文字列 / エディタ上に表示された割付デバイス*2 | 下記を除く文字列 / クオリファイア / ステップNo./移行条件No.のコメント / ジャンプ先に指定されたステップ名 |
| ラベルエディタ | クラス名以外の文字列 | 下記を除く文字列 / クラス名、データ型、デバイス名 |
| デバイスコメントエディタ | コメント列の文字列 | 下記を除く文字列 / クラス名、データ型、デバイス名 |
| ラベルコメントウィンドウ | コメント列の文字列 | |
| モーション入出力データの"プロパティ"画面 | "ソース"の文字列 | ― |

*1ラダープログラムで使用しているNOP命令は検索できません。*2下記のメニューにより、割付デバイスの表示/非表示が切り替わります。[表示] ⇒ [デバイス表示]

**画面の開き方**

[検索/置換] ⇒ [文字列検索]/[文字列置換]を選択します。

![図(元マニュアル p.563)](images/gxworks3/09-03-01.webp)

**操作手順**

各項目を設定し、[次を検索]ボタンをクリックします。

下記のメニューを使用することで、過去に検索/置換した位置にカーソルを戻したり進めたりできます。

- [検索/置換] ⇒ [前に戻る]/[次に進む]

ラダーエディタ上のノート、Pステートメント、およびIステートメントは、これらのデータが付加されているコイル、命令、ポインタ、および割込みポインタを選択した場合のみ、検索または置換できます。

"@STB"または"@IST"を入力すると、インラインSTボックス/インラインST部品を検索できます。

**注意事項**

文字列検索でプログラム中のコメントを検索する場合、結果の表示に時間がかかる場合があります。プログラム中のコメントを検索しない場合は、下記項目にチェックを入れてください。

- "検索/置換オプション" ⇒ "プログラム中のコメントを検索しない"

### 単語単位の検索/置換

単語とは、区切り文字で区切られた文字列です。

"検索文字列"に入力された文字列と完全に一致する文字列のみを検索します。

区切り文字は、下記の文字列のことを指します。

- 全角スペース

- 半角スペース

- タブ

- 改行

- STエディタ内の演算子記号例デバイスコメントabc；defに対して下記の検索文字での検索

| 検索文字 | 検索結果 | |
|---|---|---|
| 検索文字 | チェックなし | チェックあり |
| a | abc； def | 該当なし |
| abc | abc； def | abc； def |
| bc | abc； def | 該当なし |
| abc；def | abc； def | abc； def |
