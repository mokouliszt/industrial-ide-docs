# 12.7 QCPU(Qモード)経由

パソコンからQCPU(Qモード)を経由してCPUユニットにアクセスするときの設定方法を説明します。

MX-Fモデル、LHCPU、およびFX5CPUへはQCPU(Qモード)を経由したアクセスができません。

シーケンサ側I/Fとして使用できるQCPU(Qモード)は、下記のとおりです。

項目使用可能QCPU(Qモード)

パソコン側I/F シリアル通信(USB) USBポートを内蔵するQCPU(Qモード)ただし、リモートI/Oユニット(QJ72LP25、QJ72BR15)を除くパソコン側I/F EthernetボードEthernet内蔵形CPU

パソコンからQCPU(Qモード)を経由してRCPUにアクセスする場合の設定例を示します。

| QCPU(Qモード) RCPU (3) (1) (2) CC-Link IEフィールドネットワーク | | | | |
|---|---|---|---|---|
| No. | 項目 | 選択項目 | 内部設定 | 設定内容 |
| (1) | パソコン側I/F | シリアル/USB | USB | |
| (2) | シーケンサ側I/F | CPUユニット | CPUモード | QCPU(Qモード) |
| | 他局指定 | 他局(単一ネットワーク) | 交信タイムチェック | 30秒 |
| | | | リトライ回数 | 0回 |
| (3) | ネットワーク通信経路 | CC IE TSN CC IE Field | ネットワークNo. | 1 |
| | | | 局番 | 0 |

注意事項

- MELSECNET/10ネットワークユニットまたはMELSEC-QシリーズのMELSECNET/Hネットワークユニットを経由して、

RCPUにアクセスすることはできません。

- QCPU(Qモード)のEthernetポートに設定されたリモートパスワードは、GX Works3で解除できません。あらかじめGX

Works2を使用してリモートパスワードを削除しておいてください。
