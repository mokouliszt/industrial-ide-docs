## パソコンとCPUユニット間の接続構成

パソコンとCPUユニット/リモートヘッドユニットを接続するときの構成を示します。

### USBポートからの接続

![図(元マニュアル p.53)](images/gxworks3/01-10-01.webp)

使用可能なUSBケーブルを示します。

- USBケーブル(USB Aタイプ ― USB miniBタイプ)

- USBケーブル(USB Aタイプ ― USB Cタイプ)

- USBケーブル(USB Cタイプ ― USB Cタイプ)

- USBアダプタ(USB Bタイプ ― USB miniBタイプ)

USBケーブルを初めて使用する場合は、USBドライバをインストールしてください。

詳細は、下記を参照してください。

→ USBドライバのインストール手順各CPUで使用可能なUSBケーブルの詳細は下記を参照してください。

| シリーズ | 参照先 |
|---|---|
| MELSEC MXコントローラ MX-Rモデル | → 周辺機器用ケーブル・変換器の紹介品、接続可能品一覧( FA-D-0036) |
| MELSEC iQ-R | |
| MELSEC MXコントローラ MX-Fモデル | → MELSEC MXコントローラ MX-Fモデルユーザーズマニュアル |
| MELSEC iQ-F | → MELSEC iQ-F FX5S/FX5UJ/FX5U/FX5UCユーザーズマニュアル(ハードウェア編) |

CPUユニットにアクセスする場合の注意事項は、下記を参照してください。

→ USBケーブルを使用してCPUユニットと通信する場合

### I/Fボードからの接続

![図(元マニュアル p.55)](images/gxworks3/01-10-04.webp)

下記のI/Fボードに対応しています。

- Ethernetボード: パソコン内蔵/市販のEthernetボード

- CC-Link IEコントローラネットワークインタフェースボード*1、CC-Link IEフィールドネットワークインタフェースボード

*1、CC-Link Ver.2ボード*1、CC-Link IE TSNインタフェースボード*1: 詳細は各I/Fボードのマニュアルを参照してください。

*1 MXコントローラ、LHCPU、およびFX5CPUでは対応していません。
