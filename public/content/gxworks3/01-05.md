# 1.5 RnPSFCPU対応

GX Works3は、計装システム、安全システム、および二重化システムを構築するMELSEC iQ-RシリーズのRnPSFCPUに対応しています。

RnPSFCPUを使用したシステムの詳細は、下記を参照してください。

→ MELSEC iQ-R SIL2プロセスCPUユニットユーザーズマニュアル

## 計装システム対応

RnPSFCPUのプロジェクトは、安全プログラムと一般プログラムがあります。

プロセス制御プログラムは、一般プログラムでのみ作成できます。

計装システムの概要は、下記を参照してください。

→ 計装システム対応

## 安全システム対応

安全システムの概要は下記を参照してください。

→ 安全システム対応

### RnPSFCPUを使用した安全システム対応機能一覧

RnSFCPUを使用した安全システムと共通している機能は、下記を参照してください。

→ 安全システム対応機能一覧

RnPSFCPUを使用したシステムのみ、下記の機能を使用できます。

| 機能名 | 内容 | 参照 |
|---|---|---|
| ユーザ認証 | ユーザ認証を行ったCPUユニットからログオフします。 | → CPUユニットからのログオフ |
| | 両系のユーザ情報を一致させるために、他系へユーザ情報をコピーします。 | → 他系へのユーザ情報のコピー |
| 安全ユニット操作 | RnPSFCPUを使用したシステムで使用するユニットに対して、安全通信機能を有効化するかどうか操作します。 | → 安全ユニット操作 |

## 二重化システム対応

RnPSFCPUの動作モードは、二重化モードのみ対応しています。

二重化システムの概要は下記を参照してください。

→ 二重化システム対応

### RnPSFCPUの運転モード

RnPSFCPUの運転モードは、バックアップモードのみ対応しています。

バックアップモードでは、オンライン操作が両系に対して実行されます。
