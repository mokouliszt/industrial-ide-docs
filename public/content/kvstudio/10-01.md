# 10-1 運転記録/リプレイの各機能について

【運転記録/リプレイ(R)】メニューに含まれる各機能について説明します。

ポイント

【運転記録/リプレイ(R)】はKV STUDIO Ver.10以降で対応機種にKV-8000(A)を選択している場合にのみ表示されるメニューです。

## 運転記録設定

運転記録/リプレイ(R)：運転記録設定(P)

メニューから【運転記録/リプレイ(R)】【運転記録設定(P)】を選択します。運転記録機能を使用するときに設定が必要になります。運転記録機能の詳細は、『KV-8000シリーズ　ユーザーズマニュアル』を参照してください。

## CPUユニット内の運転記録を読出

運転記録/リプレイ(R)：CPUユニット内の運転記録を読出(D)

メニューから【運転記録/リプレイ(R)】【CPUユニット内の運転記録を読出(D)】を選択します。CPU

ユニットに保存されている運転記録をPCに読み出します。

![図(元マニュアル p.408)](images/kvstudio/10-01-01.webp)

| 項目 | 説明 |
|---|---|
| 選択 | PCに保存する対象の運転記録を選択します。 |
| 〔全チェックON〕 | すべての運転記録を選択します。 |
| 〔全チェックOFF〕 | すべての運転記録の選択を解除します。 |
| 〔削除〕 | 選択している運転記録をCPUユニットから削除します。 |
| 分割保存 | チェックすると、指定した容量に運転記録を分割して保存します。 |
| 〔保存〕 | クリックすると保存先フォルダを選択するダイアログが表示され、選択した フォルダに運転記録を保存します。 |

## CPUユニット内の運転記録を読出→リプレイモード

運転記録/リプレイ(R)：CPUユニット内の運転記録を読出→リプレイモード(O)

メニューから【運転記録/リプレイ(R)】【CPUユニット内の運転記録を読出→リプレイモード(O)】を選択します。CPUユニットに保存されている運転記録をPCに読み出し、KV STUDIOをリプレイモードに移行します。

![図(元マニュアル p.409)](images/kvstudio/10-01-02.webp)

| 項目 | 説明 |
|---|---|
| 〔更新〕 | CPUユニットに存在する運転記録を確認し、新たな運転記録が保存されている 場合は追加で表示します。 |
| 読出成功時、読み出した CPUユニット内の運転記 録を削除する | チェックすると、読み出しに成功した運転記録をCPUユニットから削除しま す。 |
| 〔選択〕 | 選択している運転記録をPCに保存し、KV STUDIOをリプレイモードに移行し ます。 |

## リプレイモード

運転記録/リプレイ(R)：リプレイモード(R)

メニューから【運転記録/リプレイ(R)】【リプレイモード(R)】を選択します。KV STUDIOをリプレイモードに移行します。リプレイの詳細については、「リプレイ機能」（10-8ページ）を参照してください。

運転記録/リプレイ(R)：運転記録モニタ(M)

## 運転記録の保存

運転記録/リプレイ(R)：運転記録の保存(L)

メニューから【運転記録/リプレイ(R)】【運転記録の保存(L)】を選択します。リプレイで再生している運転記録を分割保存/切出保存します。

**分割保存**

運転記録を指定したサイズに分割して保存します。

![図(元マニュアル p.411)](images/kvstudio/10-01-05.webp)

**切出保存**

運転記録を指定した時間、データだけ選択して切り出して保存します。〔詳細設定〕をクリックすると、保存対象のデータを選択できます。

![図(元マニュアル p.411)](images/kvstudio/10-01-06.webp)

![図(元マニュアル p.411)](images/kvstudio/10-01-07.webp)

## リプレイVTビューワ

運転記録/リプレイ(R)：リプレイVTビューワ(I)

メニューから【運転記録/リプレイ(R)】【リプレイVTビューワ(I)】を選択します。VT STUDIOと連携して運転記録のデバイス/変数データをタッチパネルで表示した場合の状態をパソコン上でシミュレートできます。リプレイVTビューワの詳細は 「リプレイVTビューワ」（10-12ページ）を参照してください。

《KV-8000》シリーズで保存した運転記録を再生できる、リプレイ機能について説明します。

ポイント

リプレイ機能はKV STUDIO Ver.10以降で対応機種がKV-8000(A)の場合にのみ使用できます。

リプレイ機能の概要リプレイとは、《KV-8000》シリーズで保存した運転記録を再生する機能です。運転記録とは、《KV-8000》

シリーズで収集したデバイス/変数のデータやPLCのイベント履歴、プロジェクトデータ、カメラ入力ユニット《KV-CA02》で撮影した映像データのことを表します。リプレイ機能で運転記録を再生すると、以下のことができます。

- トラブル発生時のラダープログラムの状態をカメラで撮影した映像とともに再生できます※1。

- イベントやエラー履歴、デバイス値変更イベントから運転記録を再生する位置が指定できるため、問題発生時の状態を把握できます。

- リアルタイムチャートモニタと連動して、トラブル発生時のデバイス/ 変数データをタイムチャート形式で表示できます。

- 《VT STUDIO》と連動して、トラブル発生時のタッチパネルの様子を再現できます※2。

※1カメラ入力ユニット《KV-CA02》が必要です。

※2VT STUDIO Ver.7.10以降が必要です。

リプレイでの制限

## リプレイモード

はあらかじめ保存されている運転記録を再生するモードです。リプレイでは以下の制限があります。

- ラダープログラムは編集できません。

- 登録モニタウィンドウ、一括モニタウィンドウからデバイス/変数の値を書き換えられません。

- リプレイモードで使用できるモニタ機能は下記のとおりです。下記以外のモニタ機能はリプレイモード中は使用できません。

- リアルタイムチャートモニタ

- 登録モニタ/一括モニタ

- ウォッチウィンドウ

- 内蔵機能モニタ

- 命令モニタ

- イベント/エラーモニタ

- ユニットモニタ

- サイクリック通信エラーモニタ

- センサI/Oモニタ

- センサ専用モニタ

- KVソケット情報モニタ
