# 10-1 運転記録/リプレイの各機能について

【運転記録/リプレイ(R)】メニューに含まれる各機能をまとめます。

**ポイント**

【運転記録/リプレイ(R)】はKV STUDIO Ver.10以降で対応機種にKV-8000(A)を選択している場合にのみ表示されるメニューです。

## 運転記録設定

運転記録/リプレイ(R)：運転記録設定(P)

メニューから【運転記録/リプレイ(R)】【運転記録設定(P)】を選択します。運転記録機能を使用するときに設定が必要になります。運転記録機能の詳細は、『KV-8000シリーズユーザーズマニュアル』を参照してください。

## CPUユニット内の運転記録を読出

運転記録/リプレイ(R)：CPUユニット内の運転記録を読出(D)

メニューから【運転記録/リプレイ(R)】【CPUユニット内の運転記録を読出(D)】を選択します。CPU

ユニットに保存されている運転記録をPCに読み出します。

![図(元マニュアル p.408)](images/kvstudio/10-01-01.webp)

| 項目 | 説明 |
|---|---|
| 選択 | PCに保存する対象の運転記録を選択します。 |
| 〔全チェックON〕 | すべての運転記録を選択します。 |
| 〔全チェックOFF〕 | すべての運転記録の選択を解除します。 |
| 〔削除〕 | 選択している運転記録をCPUユニットから削除します。 |
| 分割保存 | チェックすると、指定した容量に運転記録を分割して保存します。 |
| 〔保存〕 | クリックすると保存先フォルダを選択するダイアログが表示され、選択したフォルダに運転記録を保存します。 |

## CPUユニット内の運転記録を読出→リプレイモード

運転記録/リプレイ(R)：CPUユニット内の運転記録を読出→リプレイモード(O)

メニューから【運転記録/リプレイ(R)】【CPUユニット内の運転記録を読出→リプレイモード(O)】を選択します。CPUユニットに保存されている運転記録をPCに読み出し、KV STUDIOをリプレイモードに移行します。

![図(元マニュアル p.409)](images/kvstudio/10-01-02.webp)

| 項目 | 説明 |
|---|---|
| 〔更新〕 | CPUユニットに存在する運転記録を確認し、新たな運転記録が保存されている場合は追加で表示します。 |
| 読出成功時、読み出したCPUユニット内の運転記録を削除する | チェックすると、読み出しに成功した運転記録をCPUユニットから削除します。 |
| 〔選択〕 | 選択している運転記録をPCに保存し、KV STUDIOをリプレイモードに移行します。 |

## リプレイモード

運転記録/リプレイ(R)：リプレイモード(R)

メニューから【運転記録/リプレイ(R)】【リプレイモード(R)】を選択します。KV STUDIOをリプレイモードに移行します。リプレイの詳細については、「リプレイ機能」（10-8ページ）を参照してください。

## 運転記録モニタ

運転記録/リプレイ(R)：運転記録モニタ(M)

メニューから【運転記録/リプレイ(R)】【運転記録モニタ(M)】を選択します。運転記録機能の状態をモニタできます。

## 運転記録モニタ

運転記録モニタはKV STUDIOがモニタ、オンラインエディットモードの場合にのみ選択できるメニューです。

![図(元マニュアル p.410)](images/kvstudio/10-01-04.webp)

![図(元マニュアル p.410)](images/kvstudio/10-01-03.webp)

| 項目 | | 説明 |
|---|---|---|
| | 状態 | 運転記録機能の状態を表示します。 |
| | 収集状況 | 運転記録設定で設定した収集時間と、最大収集期間を表示します。 |
| 運転記録モニタ | 〔詳細〕 | クリックすると、内蔵機能モニタを表示します。 |
| | 〔運転記録開始〕 | クリックすると、運転記録データの収集、保存を実行します。 |
| | 〔運転記録読出〕 | CPUユニットから運転記録を読み出します。 |
| 内蔵機能モニタ | | 運転記録機能に関するCR、CＭが登録されたモニタです。 |

## 運転記録の保存

運転記録/リプレイ(R)：運転記録の保存(L)

メニューから【運転記録/リプレイ(R)】【運転記録の保存(L)】を選択します。リプレイで再生している運転記録を分割保存/切出保存します。

**分割保存**

運転記録を指定したサイズに分割して保存します。

![図(元マニュアル p.411)](images/kvstudio/10-01-05.webp)

**切出保存**

運転記録を指定した時間、データだけ選択して切り出して保存します。〔詳細設定〕をクリックすると、保存対象のデータを選択できます。

![図(元マニュアル p.411)](images/kvstudio/10-01-06.webp)

![図(元マニュアル p.411)](images/kvstudio/10-01-07.webp)

## 操作バー

運転記録/リプレイ(R)：操作バー(Q)

メニューから【運転記録/リプレイ(R)】【操作バー(Q)】を選択します。リプレイモード時にリプレイの再生や停止を行うための操作バーを表示します。

## イベント/エラー履歴ウィンドウ

運転記録/リプレイ(R)：イベント/エラー履歴ウィンドウ(E)

メニューから【運転記録/リプレイ(R)】【イベント/エラー履歴ウィンドウ(E)】を選択します。リプレイモード時にPLCで発生したイベントやエラーの一覧を表示します。

**ポイント**

運転記録の保存、操作バー、イベント/エラー履歴ウィンドウはKV STUDIOがリプレイモードの場合にのみ選択できるメニューです。

## デバイス変化点検索

運転記録/リプレイ(R)：デバイス変化点検索(C)

メニューから【運転記録/リプレイ(R)】【デバイス変化点検索(C)】を選択します。リプレイモード時に指定したデバイス/変数が変化したタイミングを検索し、リプレイ位置を変更できます。

ポイント：【デバイス変化点検索(C)】はKV STUDIOがリプレイモードの場合にのみ選択できるメ

ニューです。

![図(元マニュアル p.412)](images/kvstudio/10-01-08.webp)

| 項目 | | 説明 | |
|---|---|---|---|
| デバイス | | 検索対象を入力します。プルダウンでモジュール名を選択すると、ローカル変数/ ローカルラベルを検索の対象にできます。 | |
| データ形式 | | 2進数1BIT/10進数16BIT/10進数32BIT/±10進数16BIT/±10進数32BIT/ FLOAT/DOUBLE FLOATから選択します。 | |
| 条件 | 「データ形式」で 「2進数1BIT」を選択した場合 | ↑ | デバイス/変数が立ち上がったデータ点を検索します。 |
| 条件 | 「データ形式」で 「2進数1BIT」を選択した場合 | ↓ | デバイス/変数が立ち下がったデータ点を検索します。 |
| 条件 | 「データ形式」で 「2進数1BIT」を選択した場合 | ↑↓ | デバイス/変数が立ち上がった、または立ち下がったデータ点を検索します。 |
| 条件 | 「データ形式」で 「2進数1BIT」を選択した場合 | On※1 | デバイス/変数がONしているデータ点をすべて検索します。 |
| 条件 | 「データ形式」で 「2進数1BIT」を選択した場合 | Off※1 | デバイス/変数がOFFしているデータ点をすべて検索します。 |
| 条件 | 「データ形式」で 「2進数1BIT」以外を選択した場合 | >成立時※1 | デバイス/変数の値が比較対象より大きいデータ点をすべて検索します。 |
| 条件 | 「データ形式」で 「2進数1BIT」以外を選択した場合 | >=成立時※1 | デバイス/変数の値が比較対象以上のデータ点をすべて検索します。 |
| 条件 | 「データ形式」で 「2進数1BIT」以外を選択した場合 | =成立時※1 | デバイス/変数の値が比較対象と等しいデータ点をすべて検索します。 |
| 条件 | 「データ形式」で 「2進数1BIT」以外を選択した場合 | !=成立時※1 | デバイス/変数の値が比較対象と等しくないデータ点をすべて検索します。 |
| 条件 | 「データ形式」で 「2進数1BIT」以外を選択した場合 | <成立時※1 | デバイス/変数の値が比較対象より小さいデータ点をすべて検索します。 |
| 条件 | 「データ形式」で 「2進数1BIT」以外を選択した場合 | <=成立時※1 | デバイス/変数の値が比較対象以下のデータ点をすべて検索します。 |
| 条件 | 「データ形式」で 「2進数1BIT」以外を選択した場合 | 極小値 | デバイス/変数の値が極小となっているデータ点をすべて検索します。 |
| 条件 | 「データ形式」で 「2進数1BIT」以外を選択した場合 | 極大値 | デバイス/変数の値が極大となっているデータ点をすべて検索します。 |
| 条件 | 「データ形式」で 「2進数1BIT」以外を選択した場合 | 最小値 | デバイス/変数の値が最小となっているデータ点を1点検索します。 |
| 条件 | 「データ形式」で 「2進数1BIT」以外を選択した場合 | 最大値 | デバイス/変数の値が最大となっているデータ点を1点検索します。 |

## リプレイVTビューワ

運転記録/リプレイ(R)：リプレイVTビューワ(I)

メニューから【運転記録/リプレイ(R)】【リプレイVTビューワ(I)】を選択します。VT STUDIOと連携して運転記録のデバイス/変数データをタッチパネルで表示した場合の状態をパソコン上でシミュレートできます。リプレイVTビューワの詳細は 「リプレイVTビューワ」（10-12ページ）を参照してください。
