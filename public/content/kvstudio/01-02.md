# 1-2 データの互換性について

KV STUDIO Ver.12は、CPUユニットKVシリーズ※に対応します。

過去作成したラダープログラムファイルを自動変換し読み込む機能があるので、プログラミング環境の移行がスムーズにおこなえます。

※対応PLCは、KV-8000(A)、KV-7500、KV-7300、KV-5500、KV-5000、KV-3000、KV-1000、KV-700/700＋M、

KV Nanoシリーズ、KV-10/16/24/40、KV-P16です。

## KV BUILDERで作成したラダープログラム

**読み込み時の注意**

KV BUILDERで作成したKV-700/700＋M、KV-10/16/24/40、KV-P16、KZ-300/350用のラダープログラムファイル(＊.ldc)をKV STUDIO Ver.12で読み込む場合、以下の注意が必要です。

- KV STUDIO Ver.12でKV-700/700＋M、KV-10/16/24/40、KV-P16用のラダープログラムファイル

(＊.ldc)を読み込む場合、読み込み時にKV STUDIOのファイル形式(＊.kpr)にコンバートされます。

KV STUDIO Ver.12で変換したラダープログラムファイル(＊.kpr)を、元のフォルダ(＊.ldcが保存されていたフォルダ)に上書き保存すると、KV BUILDERでプロジェクトが開けなくなります。別のフォルダに保存してください。

- KV STUDIO Ver.12でKZ-300/350用ラダープログラム(＊.ldc)を読み込む場合、対応機種はKV-700として読み込まれます。

- 「コメント1」「コメント2」「ラベル」は読み込めますが、「コメント3」は読み込みません。

- 回路ライブラリには対応しません。

- ラベルを使用している場合、以下の文字列が使われていないことを確認してください。

ラベル名使用禁止文字列

- ラベル名の先頭に使用できない文字：_0123456789

- 使用できない文字列：グローバル※1、(命令語と同じ名前)、(デバイス名)、

- アルファベット1文字＋R、またはM＋数字

- KVスクリプトで規定された以下のキーワードIF, THEN, ELSE, END, SELECT, CASE, IS, TO, MC, MCR, FOR, NEXT, WHILE, DO, UNTIL, TYPE

- ラベル名の中に、2つ以上連続した”_”が現れてはならない。”a_b_c”はできますが、”a__bc”は使用できません。

- 大文字と小文字の区別はしません。”aBc”と”abC”は同じラベル名と見なします。

ポイント

- 対応機種に「KV-1000、KV-700/700 ＋M、KV-10(16)、KV-24(40)、KV-P16」を選択した場合、一部の文字列は使用できます。

- ラベル名使用禁止文字列を含むラダープログラムを読み込んだ場合、禁止文字列を使用したラベル は読み込まれず、デバイス名に置き換わって表示されます。

- KV-700/700＋M、KV-10/16/24/40、KV-P16にラダープログラムを転送する場合、使用しているラベル情報は転送されません(ラベル情報記憶領域がないため)。そのため、KV STUDIO Ver.12を使ってKV-700/700＋M、KV-10/16/24/40、KV-P16からラダープログラムを読み出した場合、ラベル使用箇所がデバイス名に置き換わって表示されます。

重 要

- 機能によっては、ラダープログラムでの設定から命令語になったものや、ラダープログラムでの設定から《KV STUDIO》での設定になったものなどがあります。また、外部入出力リレー(R)、コントロールリレー(CR)やコントロールメモリ(CM)についても追加・変更がありますので、使用する各CPUのマニュアルも合わせて確認してください。特に注意が必要な項目

- ロギング機能･ ･ ･ ラダープログラムから《KV STUDIO》でのロギング設定への置き換え

- 高速カウンタ、周波数カウンタ、カムスイッチ機能 ･ ･ 外部入力リレーの変更

- KV-8000(A)/7500/7300/5500/5000/3000/1000ではKV-L20は使用できません。ユニットエディタで、KV-L2*Vに設定してください。《KV-L21V》はVTマルチリンクモードに対応していません。VT2マルチリンクをご使用ください。また、KV-L20用に作成していたラダープログラムはKV-L21V用に作成し直してください。

- KV-8000(A)/7500/7300/5500/5000/3000/1000、KV NanoシリーズではQLは使用できません。QL用に作成していたラダープログラムは削除してください。

- KV-8000(A)/7500/7300ではKV-H20は使用できません。

**機種変換**

KV STUDIO Ver.12では、ラダープログラムの対応機種を変換する機能があります。

詳細は、「機種変更」（4-20ページ）を参照してください。

## KZ用ラダープログラム

KV STUDIOには、KZ-300/350、KZ-A500のラダープログラムを読み込む機能があります。KZ-300/350用ラダープログラムファイル(＊.ldc)およびKZ-A500用ラダープログラムファイル(＊.ldx)を読み込む場合、読み込み時にKV STUDIOのファイル形式(＊.kpr)にコンバートされます。

パソコンのハードディスクに《KV STUDIO》をインストールする方法について簡単に説明します。

インストールの準備インストールする前に次のことを確認してください。

**ハードディスクの空き容量**

《KV STUDIO》は、ハードディスクにのみインストールできます。インストール先のハードディスクに空き容量が6400MB必要です。空きがない場合は、前もって空けておいてください。

「動作環境とシステム構成」（1-2ページ）

### ■Windows環境とインストール先

《KV STUDIO》は、Windows上で動作するソフトウェアで、インストールもWindows上から実行します。

お使いのパソコンにWindowsがインストールされ、正しく動作しているかどうか確認してください。

### ■USB(USB ポート)

パソコンからUSB ポートを使って《KV-8000(A)/7500/7300/5500/5000/3000/1000/700》《KV Nano シリーズ》にプログラムなどを転送したり、モニタ機能を使うには、パソコンのUSBポートを使える状態にしておきます。設定方法については、お手持ちのパソコンのマニュアルをご覧ください。

### ■イーサネット(イーサネットポート)

パソコンからイーサネットポートを使って《KV-8000(A)/7500/5500/5000》《KV Nano シリーズ》にプログラムなどを転送したり、モニタ機能を使うには、パソコンのイーサネットポート(LAN機能)を使える状態にしておきます。設定方法については、お手持ちのパソコンのマニュアルをご覧ください。

### ■RS-232C(シリアルポート)

パソコンからRS-232Cポートを使って《KVシリーズ》にプログラムなどを転送したり、モニタ機能を使うには、パソコンのRS-232C(シリアルポート)を使える状態にしておきます。設定方法については、お手持ちのパソコンのマニュアルをご覧ください。

### ■Bluetooth通信

パソコンから《Bluetoothユニット　KV-BT1》を使って《KV-5500/5000/3000》にプログラムを転送したりモニタ機能を使うには、Bluetooth接続用USBアダプタ(別売)をパソコンに取り付け、使える状態にしておきます。設定方法については、Bluetooth接続用USBアダプタの取扱説明書を参照してください。
