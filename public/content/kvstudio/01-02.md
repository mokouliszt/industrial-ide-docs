# 1-2 データの互換性について

KV STUDIO Ver.12には、KV BUILDERやKZシリーズ向けに作成したラダープログラムを変換して読み込む機能があります。移行時に保持されない情報や、機種ごとの制限を先に確認してください。

## KV BUILDERのプロジェクトを読み込む場合

KV-700/700＋M、KV-10/16/24/40、KV-P16、KZ-300/350用の`*.ldc`ファイルを開くと、KV STUDIOの`*.kpr`形式へ変換されます。

**重要**

変換後の`*.kpr`を元の`*.ldc`と同じ名前・場所へ上書きすると、KV BUILDERから元プロジェクトを開けなくなる可能性があります。移行確認が終わるまでは別フォルダへ保存してください。

読み込み時の主な扱いは次のとおりです。

| 項目 | 読み込み結果 |
|---|---|
| KZ-300/350用`*.ldc` | 対応機種をKV-700として読み込む |
| コメント1、コメント2、ラベル | 読み込み対象 |
| コメント3 | 読み込まない |
| 回路ライブラリ | 非対応 |

## ラベル名の互換性

旧プロジェクトのラベル名がKV STUDIOの命名規則に合わない場合、そのラベルは読み込まれず、使用箇所がデバイス名へ置き換わります。

### 使用できない文字・形式

- 記号: ``! " # $ % & ' ( ) * + , - . / : ; < = > ? @ [ ¥ ] ^ ` { | } ~``と空白
- 先頭文字: `_`または数字
- 2個以上連続するアンダースコア（例: `a__bc`）
- `グローバル`、命令語名、デバイス名と同じ名前
- アルファベット1文字＋`R`、または`M`＋数字
- アルファベット1文字＋数字
- `X`または`Y`＋`A`～`F`

ラベル名の大文字・小文字は区別されません。たとえば`aBc`と`abC`は同じ名前として扱われます。

### 予約語の主なグループ

- KVスクリプト: `IF`、`THEN`、`ELSE`、`END`、`SELECT`、`CASE`、`FOR`、`NEXT`、`WHILE`、`DO`、`UNTIL`、`TYPE`など
- 真偽値・日時型: `TRUE`、`FALSE`、`TIME`、`DATE`、`TIME_OF_DAY`、`TOD`、`DATE_AND_TIME`、`DT`
- IEC 61131-3のデータ型: `BOOL`、各種整数型、`REAL`、`LREAL`、`STRING`、`WSTRING`、`BYTE`、`WORD`、`DWORD`、`LWORD`など
- IEC 61131-3の構文: `FUNCTION`、`FUNCTION_BLOCK`、`PROGRAM`、`STRUCT`、`TASK`、`VAR`、`VAR_INPUT`、`VAR_OUTPUT`、`END_VAR`など

対象機種がKV-1000、KV-700/700＋M、KV-10/16/24/40、KV-P16の場合は、上記の一部を使用できることがあります。

## ラベル情報を保持しない機種

KV-700/700＋M、KV-10/16/24/40、KV-P16にはラベル情報の記憶領域がありません。これらのPLCへ転送したプログラムを再度読み出すと、ラベル使用箇所はデバイス名で表示されます。

## 機種移行時の注意点

| 対象 | 確認事項 |
|---|---|
| ロギング | ラダープログラム内の設定ではなく、KV STUDIOのロギング設定へ移行する場合がある |
| 高速カウンタ、周波数カウンタ、カムスイッチ | 外部入力リレーの割り付け変更を確認する |
| KV-L20 | KV-8000(A)/7500/7300/5500/5000/3000/1000では使用不可。ユニットエディタでKV-L2*Vへ変更する |
| KV-L21V | VTマルチリンクモードには非対応。必要に応じてVT2マルチリンクへ変更する |
| QL | KV-8000(A)/7500/7300/5500/5000/3000/1000およびKV Nanoシリーズでは使用不可 |
| KV-H20 | KV-8000(A)/7500/7300では使用不可 |

KV STUDIOの［機種変更］機能で対象PLCを変換できます。変換後は、対象CPUのユーザーズマニュアルと照合し、外部入出力リレー、コントロールリレー（CR）、コントロールメモリ（CM）を含めて動作を確認してください。

## KZ用ラダープログラム

KZ-300/350用の`*.ldc`とKZ-A500用の`*.ldx`は、読み込み時にKV STUDIOの`*.kpr`形式へ変換されます。
