# 6-2 CPUシステム設定ダイアログボックス

CPUユニットの各種パラメータの設定をおこないます。

［CPUシステム設定］ダイアログボックスの表示方法は、 「CPUシステム設定の表示」（6-44ページ）

を参照してください。

ポイント

［CPUシステム設定］は、対応機種が《KV-8000(A)/7500/7300/5500/5000/3000/1000》

《KV Nanoシリーズ》の場合のみ、表示されます。

## プログラムの設定

ここでは、モジュールの実行順やスキャンタイムの設定など、プログラム全般に関係する設定をします。

### モジュール実行順設定

### モジュール実行順設定は、［CPUシステム設定］ダイアログボックスの「モジュール実行順設定」を選択

します。

メニューからの表示方法については、 「モジュール実行順設定」（7-9ページ）を参照してください。

![図(元マニュアル p.230)](images/kvstudio/06-02-01.webp)

モジュールが複数ある場合、実行する順番が設定できます。設定したNo.順にモジュールが実行されます。

| 項目 | 説明 |
|---|---|
| No. | モジュールの実行順です。 |
| モジュール名 | プロジェクトに登録されている各モジュールのモジュール名です。 |
| 種類 | モジュールの種類を表示します。 |
| 無効化 | チェックボックスをONにすると、モジュールを無効化します。無効化指定されたモジュールは、実行しません。 |
| 上へ | 選択したモジュールの実行順番を1つ上に移動します。 |
| 下へ | 選択したモジュールの実行順番を1つ下に移動します。 |

ポイント

モジュールの種類は必ず、「ユニット間同期モジュール」→「定周期モジュール」→「初期化モジュール」→「毎スキャンモジュール/待機モジュール」の実行順に設定されます。

### 定周期実行とユーザ割り込み設定(KV-8000/7000シリーズ)

定周期実行とユーザ割り込み設定は、［CPUシステム設定］ダイアログボックスの「定周期実行とユーザ割り込み設定」を選択します。

ポイント

「定周期実行とユーザ割り込み設定」は、対応機種が「KV-8000/7000 シリーズ」と「KV-

5000/3000シリーズ」とで設定項目が異なります。

「KV-5000/3000シリーズ」の場合 ：「定周期実行とユーザ割り込み設定(KV-5000/3000

シリーズ)」（6-55ページ）を参照してください。

対応機種が「KV Nanoシリーズ」の場合 ：「ユーザ割り込み設定(KV Nanoシリーズ)」

（6-57ページ）が選択できます。

**割り込み優先順位の設定**

複数の割り込みが発生した場合の優先順位を設定します。

割り込みの詳細は、 『KV-8000シリーズ ユーザーズマニュアル』 『KV-7000シリーズ ユーザーズマニュアル』「4-7 割り込み」を参照してください。

| 項目 | 説明 |
|---|---|
| 項目 | 割り込み処理できる項目です。定周期モジュール/ユニット間同期モジュールを登録している場合、モジュール名が表示されます。ユニット構成にユニット割り込み対応ユニットが含まれる場合、最下行の空欄部分から設定するユニットを選択します。 |
| 割り込み要因 | 各割り込み項目の割り込み要因を設定します。定周期モジュールの場合、空欄になります。ユニット割り込みの場合、設定する割り込み要因を選択します。 |
| 優先度 | すべての割り込み項目での優先順位を設定します。優先順位は3段階で、「高」が最も優先順位が高く、「低」が最も優先順位が低くなります。 |

### スキャンタイム関連の設定

### スキャンタイム関連の設定は、［CPUシステム設定］ダイアログボックスの「スキャンタイム関連の設

定」を選択します。

![図(元マニュアル p.237)](images/kvstudio/06-02-05.webp)

| | 項目 | 説明 |
|---|---|---|
| ユニ | ニット間同期する※1 | ユニット間同期が有効になります。同期するユニットと周期は、ユニットエディタで設定します。 |
| | ユニット間同期モジュールを同期する | 各ユニットの動作とユニット間同期モジュールのラダープログラム実行を同期します。 |
| | 毎スキャンモジュール・待機モジュールを同期する | 各ユニットの動作と毎スキャンモジュール・待機モジュールのラダープログラム実行を同期します。 |
| 一定 | 定スキャンタイム運転を行う | 一定スキャンタイム運転を行います。スキャンタイム設定値を設定してください。「ユニット間同期する」にチェックして「毎スキャンモジュール・待機モジュールを同期する」を選択している場合、または「END処理時間の設定を有効にする」にチェックしている場合、グレーアウトして設定できません。 |
| | スキャンタイム設定値 | スキャンタイムを任意の時間に設定します。設定値は、実測スキャンタイムよりも大きな値を設定してください。実測スキャンタイムより小さい値を設定すると、一定スキャンタイムでは運転できません。 |

| | END処理時間設定値 | END処理時間を任意の時間に設定します。ロギング処理や通信処理など、ラダープログラム以外の要因でスキャンタイムが大きく変動する場合に設定します。 |

END処理時間の詳細は、使用しているユニットのユーザーズマニュアルを参照してください。

### アクセスウィンドウ設定

アクセスウィンドウ設定は、［CPUシステム設定］ダイアログボックスの「アクセスウィンドウ設定」を選択します。アクセスウィンドウの詳細については、 『KV-8000 シリーズユーザーズマニュアル』、および『KV-7000シリーズユーザーズマニュアル』「2-1 アクセスウィンドウ」を参照してください。

ポイント

「アクセスウィンドウ設定」は、対応機種がKV-8000(A)/7500/7300の場合のみ選択できます。

![図(元マニュアル p.254)](images/kvstudio/06-02-28.webp)

| 項目 | 内容 |
|---|---|
| 表示言語設定 | 電源投入時に表示する言語を「PLCの設定に従う」「日本語」「英語」「簡体字中国語」から設定します。「PLCの設定に従う」に設定した場合、前回の電源断時の言語で表示します。 |
| 表示コメント番号設定 | 以下を表示させる際のコメントセット番号(メッセージセット番号)を設定できます。• デバイスモニタのコメント表示• アラームモニタのコメント表示• ユーザメッセージ「PLCの設定に従う」以外に設定した場合、電源投入時の表示コメントセット番号はKV STUDIOの設定に従います。 |
| バックライトOFF時間 | アクセスウィンドウのキー操作が一定時間なかった場合にバックライトを消灯する時間を設定します。「常時点灯」に設定すると、キー操作が無くてもバックライトは消灯しません。 |
| 電源投入時にメッセージ表示 | 電源投入時に指定IDのユーザメッセージを表示します。 |
| ENTキーでメッセージ表示解除 | ユーザメッセージ表示時に「ENTER」キーを押すとユーザメッセージを解除するかどうかを設定します。 |
| メッセージエディタ | ユーザメッセージの編集ダイアログを表示します。 |

- メッセージエディタ
