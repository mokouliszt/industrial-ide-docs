# 2-1 プロジェクト管理

プロジェクト管理について説明します。

## プロジェクトフォルダについて

KV STUDIO は、様々なデータファイルを扱います。プロジェクトフォルダごとにデータファイルを一括管理します。

一つのプロジェクトフォルダには、次のサポートツール用データがあります。

- KV DATALINK+設定データ

- ユニットエディタデータ

- KV-XH設定データ

- MOTION BUILDER/位置決めユニット・シンプル設定データ

- 通信型位置決めパラメータ設定データ

- MV LINK STUDIOデータ

- KV-XL設定データ

- KV-CL20設定データ

- FTPクライアント設定データ

- デバイス初期値データ

- リアルタイムチャートモニタデータ

- パネル設定データ

- FL設定ツールデータ

- シリアルPLCリンク設定データ

- OPC UAサーバ設定データ

- KV STUDIOデータ

- CPU位置決めパラメータ設定データ

- KV MOTION+データ

- KV-SH設定データ

- PROTOCOL STUDIOデータ

- KV-XLE設定データ

- KV-XCM02設定データ

- KV-DN20設定データ

- KV-XD02設定データ

- KV-CA設定データ

- メール設定データ

- 簡易PLCリンク設定データ

- ファイルレジスタ設定データ

- 高速タイムチャートモニタデータ

- ロギング/トレースデータ

![図(元マニュアル p.52)](images/kvstudio/02-01-01.webp)

KV STUDIO上でプロジェクトを作成すると、プロジェクトフォルダとKV STUDIO用データファイルが自動的に作成されます。ユニットエディタ、CPU 位置決めパラメータ設定、MOTION BUILDER、PROTOCOL

STUDIO、MV LINK STUDIO、KV MOTION＋の各サポートツールのデータについては、各ツール起動時に作成されます。

## プロジェクトフォルダの作成

プロジェクトフォルダは、デフォルトでは次の場所に作成されます。プロジェクトフォルダは、KV STUDIOの「PROJECT」内に生成されます。インストール先のフォルダを変更せず、インストールした場合

C:¥Users¥<ユーザ名>¥Documents¥KEYENCE¥KVS□□¥KVS11¥KVS¥PROJECT

となります。

例

「USER」というプロジェクトフォルダをKV STUDIOのインストール先のサブフォルダ「PROJECT」内に作成した場合

C:¥Users¥<ユーザ名>¥Documents¥KEYENCE¥KVS□□¥KVS11¥KVS¥PROJECT¥USER

になり、「USER」フォルダの下にKV STUDIO用データファイルなど、ユーザデータファイルが格納されます。

プロジェクトフォルダのファイル操作

**新規にプロジェクトを作成する**

新規にプロジェクトを作成して、ラダープログラムを作成します。

新規にプロジェクトを作成する方法については、「ファイルの各機能について」（4-2ページ）を参照してください。

**プロジェクトフォルダ名を変更する1メニューから【ファイル(F)】【プロジェクトを開く(O)】を選択します。**

- ボタンをクリック別手順

## •キー＋キーを押す2プロジェクト名を変更するフォルダを選択し、右クリックメニューから[名前の変更]を選択し、フォ

Ctrl O

ルダ名を入力します。

フォルダ名が変更されます。

ポイント

「プロジェクト名を変更する」で変更できる名称は、プロジェクトフォルダのみになります。プロジェクトフォルダ名を変更しても、フォルダ内のデータは変更されません。フォルダ内のデータファイル名も変更する場合は編集画面から「プロジェクトを別名保存」の操作をします。

「プロジェクトを別名保存」（4-3ページ）

**プロジェクトを削除する**

Delete

プロジェクトを削除するには、削除するプロジェクト名のフォルダを選択してキーを押すか、右クリックメニューから「削除」を選択します。

ポイント

プロジェクトフォルダを削除すると、プロジェクトフォルダ内のデータもすべて削除されます。

**プロジェクトをコピーする**

Ctrl C

プロジェクトをコピーするには、コピーするプロジェクト名のフォルダを選択してから、キー＋

Ctrl V

キーを押してコピーしキー＋キーを押して貼り付けるか、右クリックメニューから「コピー」、「貼り付け」を順に選択します。

**パスワードを設定する**

パスワード設定機能とは、《KV-8000(A)/7500/7300/5500/5000/3000/1000/700》《KV Nanoシリーズ》に記憶されているユニット設定情報、プログラム、コメント、ロギング設定の読み込みを禁止するための機能です。

パスワードを設定しておくと、正しいパスワードを入れないと読み込みができません。

プログラムを無断で《KV-8000(A)/7500/7300/5500/5000/3000/1000/700》《KV Nanoシリーズ》から読み出されては困るような場合に設定しておきます。

パスワード設定の詳細は、「パスワード設定」（4-5ページ）を参照してください。

ポイント

- 対応機種にKV-P16、KV-10(16)、KV-24(40)を選択した場合、パスワードは設定できません。

- リードプロテクトが設定されている場合、パスワード確認ダイアログも表示されません。
