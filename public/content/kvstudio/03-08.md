# 3-8 環境設定

ユニットエディタの環境設定を変更する方法について説明します。

## ユニットエディタの環境設定について

オプション(O)オプション設定(S)

[オプション設定]ダイアログボックスを表示させ、ユニットエディタの環境設定をします。

メニューから【オプション(O)】【オプション設定(S)】を選択すると[オプション設定]ダイアログボックスが表示します。

次の内容が設定できます。

| タブの種類 | 内 容 |
|---|---|
| 表示設定 | ユニット配置エリアおよび子機配置エリアで表示されるユニットの表示倍 率・フォントサイズ・カーソル移動の単位 |
| 色の設定 | ユニットエディタ上の文字色や背景色 |
| デバイス割付の設定 | リレー/DM の自動割り付けルール |
| EtherNet/IP設定 | 先頭デバイス、デバイス割付の設定や、センサ設定ファイルの機器一覧への 自動登録設定 |
| ユニット色の設定 | ユニット配置エリアおよび子機配置エリアで表示される各ユニットの色 |
| その他の設定 | 《KL マスタユニットKL-N20V 》の手動アドレス設定の可否など |
| ユニット間同期リフレッシュ 設定 | ユニット間同期リフレッシュデバイスに割り付けるデバイスの設定 |

変更した設定は、KV STUDIO に保存され、設定後すぐに反映されます。

### 色の設定

色を変更する項目の〔色〕のボタンをクリックして、色を選択します。

![図(元マニュアル p.99)](images/kvstudio/03-08-03.webp)

- 設定した色は《KV STUDIO》に保存されるため、次回《KV STUDIO》を起動したときでも参 考有効です。

- 保存した色を初期設定に戻すには、〔初期化〕ボタンをクリックします。

### EtherNet/IP設定

EtherNet/IPに対応したユニットがサイクリック通信で使用するデバイスの割り付けルールや、センサ設定ファイルの機器一覧への自動登録設定を設定します。

![図(元マニュアル p.101)](images/kvstudio/03-08-06.webp)

| 項目 | | 説明 | 初期値 |
|---|---|---|---|
| 先頭デバイスの設定 | 先頭ビットデバイス | 先頭ビットデバイスを入力します。 | B00 |
| | 先頭ワードデバイス | 先頭ワードデバイスを入力します。 | W00 |
| デバイス割付の設定 | ビットデバイスを10ch単位 で割り付ける | チェックすると、EtherNet/IP機器 に割り付けるデバイスに、10ch単 位でビットデバイスを割り付けま す。 | あり |
| | ワードデバイスを100単位で 割り付ける | チェックすると、EtherNet/IP機器 に割り付けるデバイスに、100単位 でワードデバイスを割り付けま す。 | あり |
| センサ設定ファイルの機器 一覧への自動登録設定 | 登録するかしないかを問い 合わせる | 登録するかしないかを問い合わせ るダイアログボックスを表示しま す。 | 登録する かしない かを問い 合わせる |
| | 常に何もしない | 登録しません。 | |
| | 常にスキャンリストのセン サ設定ファイルを登録する | 自動的にKV STUDIOのセンサ設 定ファイルを更新します。 | |

### ユニット色の設定

ここで設定した色が、ユニット配置エリアのボックス表示されたユニット色と、ラダー図上のそのユニットのリレー、DM番号表示のときの文字色になります。

色を変更する項目の〔色〕のボタンをクリックして、色を選択します。

![図(元マニュアル p.102)](images/kvstudio/03-08-07.webp)

- 設定した色は《KV STUDIO》に保存されるため、次回《KV STUDIO》を起動したときでも参 考有効です。

- 保存した色を初期設定に戻すには、〔初期化〕ボタンをクリックします。
