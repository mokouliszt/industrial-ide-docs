# 3-1 ユニットエディタの概要

ユニットエディタの機能の概要と、使用する上で注意することをまとめます。

対応機種にKV-10(16)、KV-24(40)、KV-P16を選択した場合、ユニットエディタは使用できません。

## 機能の概要

KV-8000(A)/7500/7300/5500/5000/3000/1000/700は、ラダープログラムを実行する前に接続ユニットをCPUに登録しておく必要があります。

ユニットエディタで接続ユニットを選択してユニット構成を作成し、作成したユニット構成を転送します。また、各ユニットの機能設定や、KLの子機も設定します。

従来の機種では、ディップスイッチでの設定によるハード設定が各ユニットに必要でしたが、

KV-8000(A)/7500/7300/5500/5000/3000/1000/700では基本的に不要になり、ユニットエディタから簡単に設定できます。システム構築後でも、ユニットを取り外すことなく設定を変更できます。

各ユニットの設定情報

ラダープログラムポイント

- ユニットエディタは単独で起動できません。必ずラダーファイルを開き、KV STUDIOから起動します。

- KV Nanoシリーズで、特殊ユニットを使用しない場合、CPUに登録せずに使用できます。

## ユニットエディタでの設定内容

ユニットエディタでは、次の内容を設定します。

KV-8000(A)/7500/7300/5500/5000/3000/1000/700/KV Nanoシリーズの、システムのユニット構成情報の作成

KV-8000/7500/7300/5500/5000/3000/1000/700/KV Nanoシリーズのシステムに接続されているユニット構成の一覧を作成し、システム動作時に実際に接続されているユニットが正常に動作しているかをチェックする、ユニット構成テーブルを作成します。

また、ユニットエディタで各ユニットのリレー/DM番号を変更すると、自動的にラダープログラム上のリレー/DM番号も変更する「デバイス自動置換機能」を装備しています。ラダーとユニット構成情報が一体化しているため、デバイスの割り付けミスがありません。

### ユニットのリレー/DM割り付けの設定

KV-8000(A)/7500/7300/5500/5000/3000/1000/700/KV Nanoシリーズでは、CPUユニット以外のリレー/DMを割り付けるとき、以下の範囲で設定します。

リレー/DMの割り付けには、手動設定と自動設定があります。

| | リレー番号 | DM番号 |
|---|---|---|
| KV-8000(A) | R000～R199915 | DM0～DM65534 |
| KV-7500 | R000～R199915※ | DM0～DM65534 |
| KV-7300 | R000～R199915※ | DM0～DM65534 |
| KV-5500 | R1000～R99915 | DM0～DM65534 |
| KV-5000 | R1000～R99915 | DM0～DM65534 |
| KV-3000 | R1000～R99915 | DM0～DM65534 |
| KV-1000 | R1000～R59915 | DM0～DM65534 |
| KV-700＋M | R1000～R59915 | DM0～DM39999 |
| KV-700 | R1000～R59915 | DM0～DM19999 |
| KV Nanoシリーズ | R1000～R59915 | DM0～DM32767 |

※CPU機能バージョン2.2以前のKV-7500/7300の場合は、R99915までになります。
