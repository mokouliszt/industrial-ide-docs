# 12-1 ツールの各機能について

【ツール(T)】メニューに含まれる各機能について説明します。

## ユニットエディタ

ツール(T)ユニットエディタ(U)＋

Ctrl U

《KV-8000(A)/7500/7300/5500/5000/3000/1000/700》《KV Nanoシリーズ》では、ラダープログラムを実行する前に接続ユニットをCPUに登録しておく必要があります。ユニットエディタで接続ユニットを選択してユニット構成を作成し、作成したユニット構成を転送します。ユニットエディタの詳細については、「3章 ユニットエディタ」（3-1ページ）を参照してください。

![図(元マニュアル p.448)](images/kvstudio/12-01-01.webp)

《KV Nanoシリーズ》で、基本ユニットと拡張I/Oユニットのみを使用する場合は、ユニット参 考構成を設定しなくても動作できます。

## CPU位置決めパラメータ設定

ツール(T) CPU位置決めパラメータ設定(I)

## CPU位置決めパラメータ設定とは《KV-5500/5000/3000/1000/700》《KV Nanoシリーズ》には、位置/速度制御のためのパルス出力機能があります。この機能を使えば、ステッピングモータやパルスタイプのAC サーボモータなどを制御できます。「CPU位置決めパラメータの編集」を使用して台形制御のための起動速度、運転速度、加減速時間などのパラメータを設定します。パラメータには、ポイントパラメータとシステムパラメータがあります。

ポイントパラメータ････移動先のポイント( 位置) の設定、運転/ 起動速度の設定、加減速時間の設定など、台形制御の基本となるパラメータです。システムパラメータ････センサ入力や、パルス出力の設定、コンパレータの設定、原点復帰/JOG運転のためのパラメータです。

主な機能

KV-5500/5000/3000/1000/700の場合

- 《KV STUDIO》で32点分のポイントパラメータをあらかじめ設定できます。ポイントパラメータが32

点を超える場合は、ラダープログラムでコントロールメモリCMの内容を書きかえます。

- 制御方法として、位置制御と速度制御が選べます。また、目標値/移動量には、相対値と絶対値のどちらでも設定できます。

- 2軸分のパルスが出力でき、それぞれ50Hz～100000Hzまで対応しています。

- パルスの出力方式は、1 パルス方式、2 パルス方式のどちらにも対応しているので、接続するモータドライバの方式に合わせて切り換えることができます。

KV Nanoシリーズの場合

- 《KV STUDIO》で各軸20点分のポイントパラメータをあらかじめ設定できます。ポイントパラメータが20 点を超える場合は、ラダープログラムでコントロールメモリCM の内容を書きかえます(KV

Nanoシリーズでは停電保持するポイントパラメータの範囲をCPUシステム設定で設定する必要があります(初期値はポイント1～10を停電保持))。

- 制御方法として、位置制御と速度制御が選べます。また、目標値/移動量には、相対値と絶対値のどちらでも設定できます。

- 機種により2軸～4軸分のパルスが出力でき、それぞれ1Hz～100000Hzまで対応しています。

- パルスの出力方式は、1 パルス方式、2 パルス方式のどちらにも対応しているので、接続するモータドライバの方式に合わせて切り換えることができます。

次の手順で、CPU位置決めパラメータ設定を起動します。(以下はKV-5500/5000/3000/1000/700の場合) 1メニューから【ツール(T)】【CPU 位置決めパラメータ設定(I)】を選択します。

「CPU位置決めパラメータ設定」が起動します。

![図(元マニュアル p.457)](images/kvstudio/12-01-09.webp)

ポイントパラメータ、システムパラメータなどを設定します。

| 設定するパラメータ | | 内 容 |
|---|---|---|
| ポイントパラメータ | | 各ポイントごとの動作内容を設定します。 |
| システムパラメータ | 入出力設定 | センサやZ相などの入力設定やパルス出力方式を設定します。 |
| | コンパレータ設定 | コンパレータ0～2の動作設定をします。 |

## 翻訳動作について

同じ単語で違う訳が辞書登録されている場合など、翻訳動作の詳細に関して説明します。翻訳時には以下のルールに従って翻訳が行われます。

- 辞書に翻訳元言語の文言が登録されているが、翻訳先言語に対応する文言が登録されていない（空白）場合は翻訳しません。

- 大文字小文字、半角・全角（スペース文字含む）は区別されません。

- 翻訳先言語が英語等のスペースで分割する言語の場合、文言間に半角スペースを付加して翻訳します。

- 翻訳元言語の文言に辞書の文言が複数ヒットする場合は以下の優先順位で翻訳します。

① 辞書への登録順辞書への登録順（上に登録されているもの）に優先して翻訳します。

② 一致する文字列の長さ翻訳元言語の一致する文字列の長さが長い単語を優先して翻訳します。例：

製品Work番号No製品番号Product Numberなどが登録されている状態で" 製品番号" を翻訳すると、もっとも一致する文字列が長い" 製品番号" が最初に一致し、"Product Number"と翻訳します。

③ 翻訳先言語の文字列の長さ翻訳元言語の一致する文字列の長さが同じで「省略語を優先」の場合には、省略語（翻訳先言語の文字列の長さが短い文字列）が優先されます。例：

製品番号Product Number製品番号ProductNum製品番号ProductNoと登録されていた場合、通常は一番上の文言に翻訳されますが「、省略語を優先」が選択されていた場合、"ProductNo″に翻訳されます。

④ 辞書の種類

KV STUDIOインストール時にもともと登録されている用語と、追加で辞書登録された用語が重複した場合、追加で登録した用語を優先して使用します。

## PLCクリア

ツール(T) PLCクリア(1)

接続しているCPUユニット、基本ユニットに対して、KV STUDIOからRAMクリア、ユーザメモリクリア、オールクリア、または初期化を実行します。

KV STUDIOを起動し、プロジェクトを起動していない状態でも、PLCクリアは実行できま参 考す。

ポイント

- KV-8000/7000シリーズ、KV Nanoシリーズのみ実行できます。

- ユーザメモリクリアは、KV-8000/7000シリーズのみ表示されます。

- 初期化はKV-8000シリーズのみ表示されます。

- モニタモード中、オンラインエディット中は実行できません。エディタモード中に実行してください。

- KV-8000/7000シリーズ、KV Nanoシリーズ以外の場合はアクセスウィンドウからSRAMクリア、またはオールクリアを実行してください。

(E)〕ボタンをクリックします。
