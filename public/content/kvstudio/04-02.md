# 4-2 印刷

※1CPU機能バージョン2.0以降のKV-8000(A)のみ印刷可能です。※2CPU機能バージョン2.7以降のKV-8000(A)のみ印刷可能です。

行コメントに<print-break>( 半角)と記入した場合、ラダー図の印刷時にその次の行から改印刷するには、事前に印刷する内容を設定しておきます。1メニューから【ファイル(F)】：【印刷(P)】を選択します。

[印刷設定]ダイアログボックスが表示されます。

2. 印刷する内容を設定します。印刷したい項目のチェックボックスをチェックします。

![図(元マニュアル p.129)](images/kvstudio/04-02-01.webp)

印刷したい項目をチェックします。

変更できる設定項目の場合は、クリックすることで設定内容を指定できます。

**参考**: 印刷に使用するフォントやフォントサイズが指定できます。設定項目をクリックして、表

示されるドロップダウンリストから指定します。

3. 〔ページ設定〕ボタンをクリックすると、［ページ設定］ダイアログボックスが表示されます。

![図(元マニュアル p.129)](images/kvstudio/04-02-02.webp)

| 項目 | 説明 |
|---|---|
| ヘッダー | チェックボックスにチェックを入れると、各ページの左上にプロジェクトをプリントアウトした日時を印刷します。 |
| フッター | チェックボックスにチェックを入れると、各ページ中下に「印刷ページ/総ページ数」 を印刷します。 |
| 余白プレビュー | 出力紙の余白を設定します。 |
| 余白プレビュー | 余白値変更後の印刷位置が表示されます。 |
| 初期化〕 | ［ページ設定］ダイアログボックスのすべての設定値を初期状態に戻します。 |

4. 〔印刷〕ボタンをクリックすると、［印刷］ダイアログが表示されます。

［印刷］ダイアログボックスの〔OK〕を押すと印刷が開始されます。

**参考**: 〔印刷プレビュー〕ボタンをクリックすると、［Print Preview］ウィンドウが表示されます。

**ポイント**

設定を変更して、〔キャンセル〕ボタンをクリックすると、変更した内容が無効になります。

印刷プレビュー

ファイル(F)： 印刷プレビュー(V)

印刷するイメージを表示します。1メニューから【ファイル(F)】：【印刷プレビュー(V)】を選択します。

[印刷設定]ダイアログボックスが表示されます。

2. 印刷する内容を設定します。印刷したい項目のチェックボックスをチェックして、〔印刷プレビュー〕

ボタンをクリックするとプレビューが表示されます。

![図(元マニュアル p.130)](images/kvstudio/04-02-03.webp)

印刷したい項目をチェックします。

変更できる設定項目の場合は、クリックすることで設定内容を指定できます。

**参考**: 印刷に使用するフォントやフォントサイズが指定できます。設定項目をクリックして、表

示されるドロップダウンリストから指定します。

プリンタの設定

ファイル(F)：プリンタの設定(W)

印刷する用紙サイズや、縦や横の印字方向などの設定をします。

**ポイント**

表示される内容は、お使いのプリンタによって異なります。

**参考**: 設定方法については、プリンタの取扱説明書を参照してください。
