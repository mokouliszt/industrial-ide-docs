# 1-5 基本操作について

KV STUDIOでプロジェクトを作成し、実機またはシミュレータで確認するまでの基本的な流れをまとめます。

## プログラム作成の流れ

| 手順 | 作業 | 主に使用する機能・モード |
|---|---|---|
| 1 | 設計図を基にユニット構成、ユニット機能、KLシステム、リレー／DM割り付けを設定する | ユニットエディタ |
| 2 | 制御内容に応じてモジュールやマクロを作成し、実行順序を設定する | エディタ |
| 3 | ラダープログラムを対象PLC用のコードへ変換し、PLCへ転送する | エディタ、変換、転送 |
| 4 | デバイス状態やエラー箇所を確認し、プログラムをデバッグする | モニタ、シミュレータ、リプレイ |
| 5 | デバッグ結果を反映してプログラムを修正し、完成版をPLCへ再転送する | エディタ、オンラインエディット、転送 |

## モードの使い分け

| モード | 用途 |
|---|---|
| エディタ | オフラインでラダー、モジュール、マクロ、コメント、ラベルなどを作成・編集する |
| モニタ | PLCを接続し、プログラムやデバイスの現在状態を確認する |
| オンラインエディット | モニタ中のPLCに対してラダープログラムを変更する |
| シミュレータ | PLCを接続せずにプログラムの動作を確認する |
| シミュレータエディット | シミュレーションを続けながらラダーを修正する |
| リプレイ | 保存された運転記録を再生し、トラブル発生時の状態を調べる |

シミュレータエディットはKV-8000/7000シリーズ、リプレイはKV-8000シリーズを対象とする場合に使用できます。

## 画面の扱い

各モードの画面は相互に連携します。タイトルバーをクリックして現在の操作対象にしたウィンドウを「カレントウィンドウ」と呼び、メニューやショートカットの処理対象になります。複数の編集・モニタ画面を開いている場合は、操作前にカレントウィンドウを確認してください。

## ヘルプと関連資料

［ヘルプ］メニューまたはツールバーから、HTMLヘルプ、PDFマニュアル、命令語ヘルプ、ショートカットキー一覧などを開けます。表示される資料は、プロジェクトで選択しているPLC機種によって異なります。

主な資料の種類は次のとおりです。

- KV STUDIOユーザーズマニュアル
- CPUユニット、通信、アナログ、位置決め、モーションなど各機能・ユニットのマニュアル
- 命令語リファレンス
- ST／スクリプトのプログラミングマニュアルと関数ヘルプ
- システムマクロ、デバイスマップなどの補助資料
- ショートカットキー一覧
- 参考プログラムや技術資料を探す「お役立ちナビ」

機種固有の設定や制限を確認するときは、KV STUDIOのヘルプだけでなく、対象CPU・ユニットの資料も併せて確認してください。
