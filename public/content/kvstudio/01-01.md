# 1-1 KV STUDIOの概要

KV STUDIO Ver.12の動作環境、対応PLC、主要機能をまとめます。

## 動作環境

| 項目 | 要件 |
|---|---|
| パソコン | Windows 11またはWindows 10がプリインストールされ、USB、RS-232C、Ethernetのいずれか必要なインターフェースを備えた機器 |
| CPU | 使用するWindowsが推奨するプロセッサ |
| メモリ | 使用するWindowsが推奨する容量 |
| ディスプレイ | 1024×768ピクセル、High Color（16ビット）以上 |
| プリンタ | パソコンに接続できるWindows対応プリンタ（300dpi以上、印刷する場合） |
| 対応OS | Windows 11、Windows 10 |
| ディスク空き容量 | 6,400MB以上 |

マニュアルのヘルプファイルをインストールする場合は、1言語あたり700MBが別途必要です。インストール中は、上記に加えて一時的に1,000MBを使用します。

## 対応PLC

KV STUDIO Ver.12は、次のPLCシリーズに対応します。

- KV-8000 / KV-8000A
- KV-7500 / KV-7300
- KV-5500 / KV-5000 / KV-3000
- KV-1000
- KV-700 / KV-700＋M
- KV Nanoシリーズ
- KV-P16
- KV-10 / KV-16 / KV-24 / KV-40

KV Nanoシリーズには、KV-N14、KV-N24、KV-N40、KV-N60の各タイプとKV-NC32Tが含まれます。型式末尾や入出力仕様は機種ごとに異なります。

**ポイント**

KV-X500/X300シリーズを対象にすると、専用の「KV STUDIO for KV-X500/X300シリーズ」が起動します。

## 主な機能

### ユニットエディタ

- PLCのユニット構成を作成する
- 各ユニットの機能、KLシステム、リレー／DM割り付けを設定する
- ユニット構成図と設定一覧を印刷する

### ラダーエディタ

- 命令語を使ってラダープログラムを作成する
- モジュール、ファンクションブロック、マクロ、ローカルデバイスでプログラムを構造化する
- コメントやラベルを管理し、PLCへ転送する
- プログラムを対象PLC用のコードへ変換する
- ラダー、ニモニック、コメント、ラベル、デバイス使用状況を表示・印刷する

### シミュレータ

実機を接続せずにラダープログラムを実行し、連続スキャン、1スキャン、1ステップ、逆ステップなどで動作を確認できます。シミュレーション中のラダー編集は、KV-8000(A)、KV-7500、KV-7300を対象とする場合に使用できます。

### モニタ

実機上の接点やデバイスの状態をオンラインで確認します。タイムチャート表示とオンラインエディットにも対応します。

### リプレイ

運転記録、カメラ映像、イベント・エラー履歴、デバイスデータを組み合わせ、トラブル発生時の状態を追跡します。VT STUDIOとの連携や、KV-XD02の分析レポート表示にも対応します。リプレイ機能の対象はKV-8000(A)です。
