## KV STUDIOの機能

《KV STUDIO》には、次のような機能があります。

**ユニットエディタ機能**

- 《KV-8000(A)/7500/7300/5500/5000/3000/1000/700》

![図(元マニュアル p.25)](images/kvstudio/01-01-01.webp)

《KV Nanoシリーズ》のシステムを構築します。

- 《KV-8000(A)/7500/7300/5500/5000/3000/1000/700》

《KV Nanoシリーズ》の各ユニットの設定や、リレー/DMを割り付けます。

- ユニット構成図および設定一覧表の印刷ができます。

**エディタ機能**

- ラダー言語の各種命令語を使って、ラダープログラムを作成

![図(元マニュアル p.25)](images/kvstudio/01-01-02.webp)

できます。

- モジュール・ファンクションブロック・マクロおよびローカルデバイスを使用してプログラムを構造化できます。

- デバイスにコメント・ラベルを登録したり、PLC本体に転送できます。

- 作成したラダープログラムを各KVシリーズが読み込めるコードに変換します。

- 作成したラダープログラムや、ニモニックリスト、コメント・ラベル、デバイスの使用状況などの一覧を画面上に表示、印刷できます。

**シミュレータ機能**

- 作成したラダープログラムの動作を、各KVシリーズを接続し

![図(元マニュアル p.25)](images/kvstudio/01-01-03.webp)

ない状態でシミュレートでき、作成したプログラムをデバッグできます。

- 実行モードとして連続スキャン実行、1スキャン実行、1ステップ実行などを選ぶことができ、問題が発生している箇所を確実に見つけだせます。

- 逆ステップ実行も可能です。

- シミュレートしながらのラダー編集（シミュレータエディット）が可能です。

シミュレータエディット機能は、対応機種がKV-8000(A)/7500/7300のときのみ使用できます。参 考

**モニタ機能**

- 作成したラダープログラムを使って、接点のON/OFF の状態

![図(元マニュアル p.26)](images/kvstudio/01-01-04.webp)

をリアルタイムでオンラインモニタできます。

- タイムチャートの表示ができます。

- オンラインエディットが可能です。

**リプレイ機能**

- トラブル発生時のラダープログラムの状態をカメラで撮影

![図(元マニュアル p.26)](images/kvstudio/01-01-05.webp)

した映像とともに再生できます。

- 巻き戻しやコマ送り機能に加えて、イベントやエラー履歴からも再生位置を変更できるため、問題発生時の状態を把握できます。

- リアルタイムチャートモニタと連動して、トラブル発生時のデバイスデータをタイムチャート形式で表示できます。

- 《VT STUDIO》と連動して、トラブル発生時のデバイスデータをリプレイVTビューワ上で確認できます。

- KV-XD02 のいつもと違う分析で作成された分析レポートを、分析レポートビューワで表示できます。

リプレイ機能は、対応機種がKV-8000(A)のときのみ使用できます。参 考

KV STUDIO Ver.12は、CPUユニットKVシリーズ※に対応します。

過去作成したラダープログラムファイルを自動変換し読み込む機能があるので、プログラミング環境の移行がスムーズにおこなえます。

※対応PLCは、KV-8000(A)、KV-7500、KV-7300、KV-5500、KV-5000、KV-3000、KV-1000、KV-700/700＋M、

## KV BUILDERで作成したラダープログラム

**読み込み時の注意**

KV BUILDERで作成したKV-700/700＋M、KV-10/16/24/40、KV-P16、KZ-300/350用のラダープログラムファイル(＊.ldc)をKV STUDIO Ver.12で読み込む場合、以下の注意が必要です。

- KV STUDIO Ver.12でKV-700/700＋M、KV-10/16/24/40、KV-P16用のラダープログラムファイル

(＊.ldc)を読み込む場合、読み込み時にKV STUDIOのファイル形式(＊.kpr)にコンバートされます。

KV STUDIO Ver.12で変換したラダープログラムファイル(＊.kpr)を、元のフォルダ(＊.ldcが保存されていたフォルダ)に上書き保存すると、KV BUILDERでプロジェクトが開けなくなります。別のフォルダに保存してください。

- KV STUDIO Ver.12でKZ-300/350用ラダープログラム(＊.ldc)を読み込む場合、対応機種はKV-700として読み込まれます。

- 「コメント1」「コメント2」「ラベル」は読み込めますが、「コメント3」は読み込みません。

- 回路ライブラリには対応しません。

- ラベルを使用している場合、以下の文字列が使われていないことを確認してください。

ラベル名使用禁止文字列

- ! " # $ % & ' ( ) * + , - . / : ; < = > ? @ [ ¥ ] ^ ` { } ~ (スペース)

- ラベル名の先頭に使用できない文字：_0123456789

- 使用できない文字列：グローバル※1、(命令語と同じ名前)、(デバイス名)、

- アルファベット1文字＋R、またはM＋数字

- アルファベット1文字＋数字

- X＋(A～F)

- Y＋(A～F)

- KVスクリプトで規定された以下のキーワードIF, THEN, ELSE, END, SELECT, CASE, IS, TO, MC, MCR, FOR, NEXT, WHILE, DO, UNTIL, TYPE
