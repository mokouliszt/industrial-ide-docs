# 9-8 ユニットモニタ

ユニットモニタは、登録モニタ、一括モニタと違い、リレーやDMの値をモニタするユニットにあわせて最適に表示します。ここでは、ユニットモニタの見かたや操作の方法をまとめます。

## ユニットモニタの特長

ユニットモニタには次のような特長があります。

- ユニット毎に割り付けられているDM、リレーを最初から登録した状態で表示できます。

- ユニット毎に表示内容やレイアウトが雛形として用意されているため、モニタが簡単です。

- リレーやDMの割付状況を気にしなくても、モニタしたいユニットを選択するだけで、選択したユニットのモニタができます。

## ユニットモニタの表示

モニタ/シミュレータ(N)：ユニットモニタ(U)

- メニューから【モニタ/シュミレータ(N)】【ユニットモニタ(U)】を選択し、モニタしたいユニットを選択して〔表示〕ボタンをクリックします。

- ワークスペースのユニット設定で、モニタしたいユニットを選択し、右クリックメニューから[ ユニットモニタ]を選択します。

![図(元マニュアル p.380)](images/kvstudio/09-08-01.webp)

**ポイント**

ユニットモニタは、モニタ/オンラインエディット/リプレイ中にのみメニューから選択できます。

## ユニットモニタウィンドウについて

ここでは、KV-SIR32XTのユニットモニタ画面について説明します。他ユニットの機能の詳細や設定方法については、各ユニットのユーザーズマニュアルを参照してください。

### 16点、32点、64点ユニットモニタ

![図(元マニュアル p.381)](images/kvstudio/09-08-02.webp)

⑨入力時定数

⑥リレー番号

⑧〔表示形式切替〕ボタン

| 名称 | 機能 |
|---|---|
| ①入出力種別 | 入力リレーか出力リレーかを表示します。 |
| ②先頭リレー番号 | 選択されたユニットに割り付けられたリレーの先頭番号を表示します。番号は、 ユニットエディタで設定した内容を反映します。 リレーの先頭番号は、各チャンネルごとに(16点ごと)表示します。 |
| ③ON/OFF状態表示 | リレーのON/OFF状態を表示します。 OFFのとき：グレーONのとき：入力ユニットの場合は青色、出力ユニットの場合は緑色で表示されます。 |
| ④コメント表示エリア | リレーのデバイスコメントを表示します。表示するコメントは先頭に登録されているコメントセットです。 |
| ⑤ユニット情報 | 選択されたユニット情報を表示します。ユニット接続番号、機種名を表示します。 |
| ⑥リレー番号 | 「①先頭リレー番号」の先頭番号から数えて何番目のリレー番号かを表示します。 |
| ⑦ワード表示値 | 各チャンネルのリレーのワード表示での値を表示するエリアです。 |
| ⑧〔表示形式切替〕ボタン | 「ワード表示値」の表示形態を「10進(符号なし)」「10進(符号あり)」「16進」の3種類から選択します。 |
| ⑨入力時定数 | チェックすると、ユニットエディタで設定した各入力リレーの入力時定数を表示します。 |
| ⑩ch選択タブ | モニタするチャンネルを切り替えます。「0～31」は、64点ユニットの下位2ch分、 32～63は、上位2ch分を表示します。 |

特殊ユニットのユニットモニタについては、各特殊ユニットのマニュアルを参照してください。

**ポイント**

- ユニットモニタ上では、「⑨入力時定数」の設定は、変更できません。

- アクセスウィンドウでモニタ中に「⑨入力時定数」の設定を変更した場合、ユニットモニタに表示される内容は反映されません。表示を反映する場合は、ユニットモニタウィンドウを開きなおしてください。

- 割り込みプログラム中において、内蔵の出力リレー(例：R500)をダイレクト出力するプログラムを、オンラインエディット機能で変更する場合は、「DR(例：DR500)」で表記してください。「R」で表記した場合は、正しく動作しないことがあります。
