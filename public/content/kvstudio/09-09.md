# 9-9 デバイス値一括変更/読出

デバイス値一括変更/読出は、PLC本体のデバイス値を一括で書き替えたり、ファイルに保存したりするものです。ここでは、デバイス値一括変更/読出の操作方法について説明します。

### デバイス値一括変更/読出の制限事項

デバイス値一括変更/読出には以下の制限事項があります。

- インデックスおよび間接参照デバイスは使用できません。

- タイマ(T)、カウンタ(C)、高速カウンタ(CTH)は、現在値のみ使用できます(接点および設定値は使用できません)。

- 高速カウンタコンパレータ(CTC)は設定値のみ使用できます(接点は使用できません)。

- デバイスは、編集中の機種で存在するものだけが対象になります。

- DM、EM、FM、W、ZF、C、T、CTH、CTC、CMの各デバイスは、2進数1BITでアクセスしないでください。

また、表示形式を32ビットにする場合、デバイス番号は偶数番号を指定してください。

モニタ中に《PLC》のラダープログラムを変更できる、オンラインエディット機能について説明します。

機能の概要

オンラインエディット機能は、《KV STUDIO》のモニタ画面上で直接ラダープログラムを変更できます。プログラム作成時の最終デバッグでの現場調整や簡単なプログラムを変更するときに便利な機能です。

オンラインエディットの特長オンラインエディットの特長は、以下のようになります。• オンラインエディット中もモニタ画面と同様に各デバイスのON/OFF 状態および値の変化がリアルタイムに確認できます。リアルタイムに装置の動作状況を確認しながらのデバッグが可能です。

- 編集箇所の背景色が変化し(水色)、どのセルに修正を加えているかが一目で確認できます。変更の結果、装置がうまく動かなかった場合などに再変更をおこない易くなります。

- オンラインエディット中も全画面の編集が可能です。複数回路にインターロックを設定する場合でも、編集回路の切り換えなしにエディタモードと同じ感覚で編集が可能です。

オンラインエディット機能使用時の注意事項

| | PLC がRUNの状態でオンラインエディット機能を使用した場合、使用方法を誤ると装 置及び作業者に多大な被害を与える可能性があります。次に記載する注意事項をよくご 確認いただき、正しく使用してください。 • オンラインエディットを使用する場合は、装置周辺をよく確認の上、不測の事態が発 生しても人的・物的被害が出ないような状態で使用してください。 • プログラム転送後直ちに動作をおこなう回路(実行条件がB接点の回路など)を追加し ないでください。機器が暴走するおそれがあります。 • ON している出力は削除しないでください。ON している出力を削除すると、出力がON 状態で保持されます。 |

重 要

- オンラインエディット中または、ラダープログラム転送中にパソコンの異常でモニタが強制終了した場合は、本体へ転送がおこなわれていないラダープログラムは、破棄されます。

- オンラインエディット中にモニタされている各デバイスのON/OFF状態および値は、オンラインエディット開始直前のラダープログラム(現在PLC内で動作しているプログラム)をもとに表示されています。転送するまでは、オンラインエディット中のプログラム状態を反映した動きではありません。オンラインエディットで追加・変更したプログラムで動作させるには、転送をおこなう必要があります。

- オンラインエディット中にマクロ命令の追加・削除をおこなうと、RUN 中書き込みできません。

- オンラインエディット中はローカルデバイスの割付設定ができません。エディタに戻って割り付け直してください。

- 割り込みプログラム中で、内蔵の出力リレー(例：R500) をダイレクト出力するプログラムをオンラインエディット機能で変更する場合は、「DR」で表記してください。「R」で指定した場合、正しく動作しないことがあります。
