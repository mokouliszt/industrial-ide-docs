# 3-3 子機設定画面のメニュー一覧

子機設定画面でのメニューとアイコンを説明します。各機能の詳しい説明については、それぞれのページを参照してください。

## ファイル

ファイル操作、印刷関係の項目をまとめています。

| 項目 | アイコン | 機種 | 参照頁 |
|---|---|---|---|
| 子機情報のインポート | | 他のプロジェクトから、子機設定情報をコピーします。 | 3-39 |
| 子機設定画面を閉じる | | 現在編集中の子機設定ウィンドウを閉じます。 | 3-36 |

## 編集

ユニット情報を編集するときに使用する項目をまとめています。

| 項目 | アイコン | 機種 | 参照頁 |
|---|---|---|---|
| 切り取り | | 選択範囲を切り取ります。 | 3-38 |
| コピー | | 選択範囲をコピーします。 | 3-38 |
| 貼り付け | | カーソル位置に切り取ったデータを貼り付けます。 | 3-38 |
| 削除 | － | 選択範囲を削除します。 | 3-37 |
| ユニット/子機コメント編集 | | [ユニットコメント]ダイアログボックスを表示し、ユ ニットおよび子機コメントを編集します。 | 3-40 |
| ユニット/子機予約 | | ユニット(子機)の予約および予約の解除をします。 | 3-40 |

## 変換

子機アドレスの割付に関係した項目をまとめています。

| 項目 | アイコン | 機種 | 参照頁 |
|---|---|---|---|
| エラー行ジャンプ | | メッセージウィンドウに表示されている、エラーの該当 ユニットの位置にジャンプします。 | 3-40 |
| アドレス自動割付 | | 子機のアドレスを自動割り付けします。 | 3-38 |
| アドレスチェック | | 子機のアドレスがユニット間で重複していないか確認 します。 | 3-24 |

| ツールバー | － | ツールバーの表示/非表示を選択します。 | － |
| ステータスバー | － | ステータスバーの表示/非表示を選択します。 | － |
| メッセージエリア | － | メッセージエリアの表示/非表示を選択します。 | － |
| ユニット/子機設定内容 | － | 現在カーソルが選択されている子機のユニット設定 ウィンドウを開きます。 | － |
