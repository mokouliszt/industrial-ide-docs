# 3-5 ユニットエディタの基本操作

ユニットエディタの使用方法をまとめます。

## システム作成の流れ

例として、次のようなユニット構成で、ユニットエディタを使用して新規にシステムを作成するまでの手順を説明します。

| ＫＶ\|ＰＵ１ | ＫＶ\|８０００ | ＫＶ\|ＳＩＲ３２ＸＴ | ＫＶ\|ＳＡＤ０４ | バス接続ユニット | | | | |
|---|---|---|---|---|---|---|---|---|
| ＫＶ\|ＰＵ１ | ＫＶ\|８０００ | ＫＶ\|ＳＩＲ３２ＸＴ | ＫＶ\|ＳＡＤ０４ | バス接続ユニット | ＫＶ\|Ｌ２１Ｖ | ＫＶ\|Ｈ２０Ｓ | ＫＶ\|ＡＤ４０Ｖ | エンドユニット |

1. KV STUDIO編集画面のメニューから【ファイル(F)】 【プロジェクトの新規作成(N)】を選択します。

[プロジェクト新規作成]ダイアログボックスが表示されます。

2. 新規に作成するプロジェクト名を入力し、対応機種を選択して、〔OK〕ボタンをクリックします。保存位置を変更する場合は、〔参照〕ボタンをクリックし、保存場所を選択して〔OK〕ボタンをクリックします。

［ユニット構成設定の確認］ダイアログボックスが表示されます。

〔はい〕を選択すると、ユニットエディタが起動します。

![図(元マニュアル p.71)](images/kvstudio/03-05-01.webp)

| 項目 | 説明 |
|---|---|
| 〔はい〕 | ユニットエディタが起動します。ユニット構成設定をおこなってください。 |
| 〔いいえ〕 | KV STUDIOのエディタ画面を表示します。 ユニット構成設定は、別途ユニットエディタを起動しておこないます。 |
| 〔ユニット構成読出〕 | CPUユニットからユニット構成情報を読み出します。 あらかじめパソコンとCPUユニットを通信可能な状態にしておいてください。 |

**参考**: 対応機種がKV Nanoシリーズの場合、[ユニット構成設定の確認]ダイアログは表示されません。基本ユニットと拡張I/Oユニットのみ使用する場合は、占有デバイスが自動で割り付くので、以下の手順は不要です。

3. [ユニット選択(1)]タブをクリックします。

4. ユニット選択ウィンドウから、接続するユニットを選択して、ユニット配置エリアにドラッグ&ドロップします。

編集画面上に表示されます。

ここでは例として、前述のユニット構成のようにKV-PU1→KV-SIR32XT→･･･→KV-AD40の順にドラッグ&ドロップします。

![図(元マニュアル p.72)](images/kvstudio/03-05-02.webp)

CPUユニット・バス接続ユニット・エンドユニットは、自動的に配置されます。

**参考**:

- KV-8000(A)とKV-8000/7000シリーズ拡張ユニットだけの場合、バス接続ユニット・エンドユニットは不要です。

- KV-7500/7300とKV-7000シリーズ拡張ユニットだけの場合、バス接続ユニット・エンドユニットは不要です。

- KV Nanoシリーズの場合、エンドユニットはありません。

5. メニューから【変換(P)】－【リレー/DM自動割付(A)】を選択します。

リレー番号が自動的に割り当てられ、メッセージウィンドウに結果が表示されます。

ここでは、リレー/DMを自動的に割り付ける手順について説明します。リレー番号を手動で割り付ける場合は、「配置しているユニットのリレー/DMを手動で割り付ける」（3-22ページ）を参照してください。

6. ユニット配置エリア上で詳細設定するユニットをダブルクリックし、［ユニット設定］タブで設定します。

![図(元マニュアル p.72)](images/kvstudio/03-05-03.webp)

**参考**: 各ユニットの設定方法および内容については、各ユニットのマニュアルを参照してください。

7. すべてのユニットの設定が完了したら、〔OK〕ボタンをクリックします。

設定したシステムを保存して、KV STUDIOのエディタ画面を表示します。
