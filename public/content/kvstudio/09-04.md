# 9-4 一括モニタ

一括モニタは、デバイスの種類ごとに全デバイスの現在値、接点のON/OFF状態、登録されているコメントを一括して表示します。ここでは、一括モニタで操作できることをまとめます。

## 一括モニタの概要

一括モニタには、次の機能があります。

- デバイスの種類ごとに、すべてのデバイスの現在値・設定値・接点(タイマ・カウンタの接点の状態)・コメントを一括表示

- デバイスの現在値、設定値の変更一括モニタウィンドウの表示の方法は：「一括モニタウィンドウ」（9-46ページ）を参照してください。

## 表示するデバイスの変更

すでに表示している一括モニタの表示デバイスを変更できます。

デバイスを変更する方法は以下の2通りあります。

**ラダーモニタウィンドウからドラッグして変更する**

ラダーモニタで表示するデバイスを選択(マウスーソルの形が変わります。)し、一括モニタウィンドウの上にドラッグ&ドロップします。

![図(元マニュアル p.333)](images/kvstudio/09-04-01.webp)

![図(元マニュアル p.333)](images/kvstudio/09-04-02.webp)

一括モニタウィンドウに、選んだデバイスと同じ種類のデバイスのリストが表示されます。

ラダーモニタウィンドウ上で右クリックメニューから［一括モニタ］を選択すると、一括モニタウィンドウを複数起動できます。

**デバイスを指定して変更する**

一括モニタウィンドウにキーボードから直接入力します。

操作は、次項の：「現在値などの変更」と同じです。

以下のモニタから、デバイスを選択して右クリックし、【一括モニタウィンドウ】を選択すると、選択したデバイスを先頭に一括モニタを表示できます。

- ラダーモニタ

- ウォッチウィンドウ

## 現在値などの変更

一括モニタウィンドウでデバイスを選択して、デバイスのON/OFF状態や現在値などを変更できます。

![図(元マニュアル p.334)](images/kvstudio/09-04-03.webp)

![図(元マニュアル p.334)](images/kvstudio/09-04-04.webp)

変更したい項目をダブルクリックし、数値を入力します。変更できない項目は背景色がグレーで表示されています。

| 項目 | 説明 |
|---|---|
| デバイス(D)※1プログラム※2デバイス | ラジオボタンをONにすると、グローバルデバイス/ローカルデバイスの一括モニタを表示します。 |
| デバイス(D)※1プログラム※2デバイス | グローバルデバイスを一括モニタする場合は、「グローバル」を選択します。 ローカルデバイスを一括モニタする場合は、モニタするプログラムを選択します。 |
| デバイス(D)※1プログラム※2デバイス | モニタ中のデバイスを表示します。先頭行のセルで、デバイスが変更できます。 |
| 拡張ユニットバッファメモリ(G)※1ユニット※1オフセット※1 | ラジオボタンをONにすると、拡張ユニットバッファメモリの一括モニタを表示します。 拡張ユニットバッファメモリがモニタ可能なユニットが接続されていない場合、この項目はグレーアウトされます。 |
| 拡張ユニットバッファメモリ(G)※1ユニット※1オフセット※1 | モニタ中のユニットを表示します。先頭のセルで、一括モニタするユニットが変更できます。 |
| 拡張ユニットバッファメモリ(G)※1ユニット※1オフセット※1 | 拡張ユニットバッファメモリの先頭アドレスからのオフセットを表示します。 先頭のセルで、表示する拡張ユニットバッファメモリの先頭アドレスを変更できます。 |
| 拡張ユニット内部デバイス（Ｕ） ユニットデバイス | ラジオボタンをONにすると、拡張ユニット内部デバイスの一括モニタを表示します。 拡張ユニット内部デバイスがモニタ可能なユニットが接続されていない場合、この項目はグレーアウトされます。 |
| 拡張ユニット内部デバイス（Ｕ） ユニットデバイス | モニタ中のユニットを表示します。先頭のセルで、一括モニタするユニットが変更できます。 |
| 拡張ユニット内部デバイス（Ｕ） ユニットデバイス | モニタ中のデバイスを表示します。先頭行のセルで、デバイスが変更できます。 |
| 現在値 | 現在値を表示します。(CTC選択時は、設定値を表示します。)項目をダブルクリックすると、キーボードから現在値を変更できます。 ビットデバイスの場合は、ダブルクリックするごとに、ON/OFFが切り替わります。 |
| 表示形式 | 一括モニタ中の値の表示形式が表示されます。 先頭のセルで、表示形式が変更できます。 |
| 設定値 | 設定値を表示します。値が表示されているセルで、設定値が変更できます。 |
| 接点 | タイマ・カウンタの接点の状態を表示します。 |
| コメント | コメントを表示します。 |

※1対応機種にKV-8000(A)/7500/7300/5500/5000/3000KV Nanoシリーズを選択している場合のみ表示されます。

※2対応機種にKV-8000(A)/7500/7300/5500/5000/3000/1000KV Nanoシリーズを選択している場合のみ表示されます。

**クロスリファレンスの作成**

［一括モニタ］ウィンドウ内の右クリックメニューから、［クロスリファレンス］を選択します。

右クリックしたデバイスのクロスリファレンスが作成されます。

「拡張ユニットバッファメモリ」を選択している場合、クロスリファレンスは作成できません。

**コメントの表示を切り替える**

［一括モニタ］ウィンドウ内の右クリックメニューから、［コメント表示］を選択し、表示なし、もしくは各コメントセットから表示内容を選択します。

チェックマークが付いた項目が有効になります。

「拡張ユニットバッファメモリ」を選択している場合、表示なし、もしくは表示ありから選択します。

**現在値を初期値に戻す**

［一括モニタ］ウィンドウ内の右クリックメニューから、［リセット］を選択します。

右クリックしたデバイスの現在値が初期値に戻ります。

「デバイス初期値設定」（12-5ページ）

### 強制セット/リセットを登録する

［一括モニタ］ウィンドウ内の右クリックメニューから、［強制セット/リセット登録］を選択し、[セット登録]/[リセット登録]を選択します。

右クリックしたデバイスの強制セット/リセットの登録と解除をおこないます。

「強制セット/リセット登録」（11-12ページ）
