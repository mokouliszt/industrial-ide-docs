# A-1 USBケーブルで直接接続する場合のドライバのインストール方法

パソコンを初めてコントローラ、通信カプラユニットまたはNA シリーズ PT のUSB ポートに接続する場合の手順を、Windows 7 の場合を例に示します。パソコンにはあらかじめSysmac Studio がインストールされているものとします。

## A-1-1 USB ドライバのインストール方法

1. コントローラの電源、通信カプラユニットまたは表示器の電源を入れ、USB ポートとパソコンをUSB ケーブルで接続します。接続後、自動的にUSB ドライバがインストールされます。なお、NA5-££W££££-V2 をご使用の場合、USB ドライバは自動的にはインストールされません。詳細は「A-1-3 NA5-££W££££-V2 のUSB ドライバをインストールする」を参照してください。

参考

USB ドライバのインストール後、Windows の再起動が必要になることがあります。Windowsの表示に従ってWindows を再起動してください。

## A-1-2 USB ドライバを指定してインストールする

USB ドライバを自動的にインストールできない場合は、以下の手順でドライバを指定してインストールします。

1. コントロールパネルから［デバイスマネージャー］を起動し、［他のデバイス］から［不明なデバイス］をダブルクリックします。［不明なデバイスのプロパティ］ダイアログボックスが表示されます。

![図(元マニュアル p.875)](images/sysmac/ap-01-01.webp)

2. ［ドライバーの更新］ボタンをクリックします。［ドライバーソフトウェアの更新］ダイアログボックスが表示されます。

![図(元マニュアル p.876)](images/sysmac/ap-01-02.webp)

3. ［コンピューターを参照してドライバーソフトウェアを検索します］をクリックします。ドライバーソフトウェアの参照先を指定するダイアログボックスが表示されます。

![図(元マニュアル p.876)](images/sysmac/ap-01-03.webp)

- 4. ［参照］ボタンをクリックし、USB ドライバを選択します。USB ドライバの場所は以下のとおりです。コントローラまたは表示器（形NA5-££££££）の場合のフォルダを示します。通信カプラユニットの場合は、「Controller」を「Coupler」に読み替えてください。

- Windows 7 32 ビット版、Windows 8.1 32 ビット版、Windows 10 32 ビット版C:\Program Files\OMRON\Communications Middleware\USB\Controller\7\32bit

- Windows 7 64 ビット版、Windows 8.1 64 ビット版、Windows 10 64 ビット版、Windows 11 64 ビット版C:\Program Files (x86)\OMRON\Communications Middleware\USB\Controller\7\64bit表示器（NA5-££W££££-V1）の場合のフォルダを示します。

- Windows 7 32 ビット版C:\Program Files\OMRON\Communications Middleware\USB\NA5-V1\7\32bit

- Windows 8.1 32 ビット版C:\Program Files\OMRON\Communications Middleware\USB\NA5-V1\8\32bit

- Windows 10 32 ビット版C:\Program Files\OMRON\Communications Middleware\USB\NA5-V1\10\32bit

- Windows 7 64 ビット版C:\Program Files(x86)\OMRON\Communications Middleware\USB\NA5-V1\7\64bit

- Windows 8.1 64 ビット版C:\Program Files(x86)\OMRON\Communications Middleware\USB\NA5-V1\8\64bit

- Windows 10 64 ビット版、Windows 11 64 ビット版C:\Program Files(x86)\OMRON\Communications Middleware\USB\NA5-V1\10\64bit

5. ［次へ］ボタンをクリックします。［Windows セキュリティ］ダイアログボックスが表示されます。

![図(元マニュアル p.877)](images/sysmac/ap-01-04.webp)

6. ［インストール］ボタンをクリックします。USB ドライバのインストールが完了し、ダイアログボックスが表示されます。

![図(元マニュアル p.877)](images/sysmac/ap-01-05.webp)

## A-1-3 NA5-££W££££-V2 のUSB ドライバをインストールする

NA5-££W££££-V2 のUSB ドライバは、以下の手順でインストールします。

1. Microsoft®Update カタログを開き、「USB RNDIS Gadget」を検索します。複数のドライバが見つかるため、以下のドライバを選択してください。［Acer Incorporated. - Other hardware - USB Ethernet/RNDIS Gadget］

Microsoft®Update カタログについては、https://www.catalog.update.microsoft.com/Home.aspx を参照してください。

2. お客様の環境にあったドライバをダウンロードします。［Windows 7, Windows 8, Windows 8.1 and later drivers］のダウンロードボタンをクリックしてください。ダウンロードボタンをクリックするとダウンロードのウインドウが表示されるため、cab ファイルのリンクをクリックして、cab ファイルをダウンロードしてください。

参考

32 ビット版のWindows をご使用の場合、Sysmac Studio からのシステムプログラムの転送には対応していません。システムプログラムを更新するには、システムリカバリーを実行する必要があります。システムリカバリーの詳細は『プログラマブルターミナル NA シリーズ ユーザーズマニュアル ハードウェア(-V2)編(SBSA-561)』を参照してください。

3. ダウンロードしたcab ファイルを解凍します。Cab ファイル上で右クリックして［開く］を選択してください。

![図(元マニュアル p.878)](images/sysmac/ap-01-06.webp)

2 つのファイルが表示されるため、これらのファイルを選択し、右クリックして［展開］を選択してください。

![図(元マニュアル p.878)](images/sysmac/ap-01-07.webp)

［宛先の選択］ダイアログボックスで展開先のフォルダを選択し、［展開］ボタンをクリックしてください。

![図(元マニュアル p.878)](images/sysmac/ap-01-08.webp)

［ファイルのダウンロード–セキュリティの警告］ダイアログボックスが表示される場合は、［保存］ボタンをクリックしてファイルを保存してください。

![図(元マニュアル p.878)](images/sysmac/ap-01-09.webp)

4. 解凍したフォルダを開き、以下のファイルがあることを確認します。［セキュリティカタログ］ファイルと［セットアップ情報］ファイルが存在することを確認してください。

![図(元マニュアル p.879)](images/sysmac/ap-01-10.webp)

5. コントロールパネルから［デバイスマネージャー］を起動し、［他のデバイス］から［RNDIS/Ethernet Gadget］をダブルクリックします。［RNDIS/Ethernet Gadget のプロパティ］ダイアログボックスが表示されます。

![図(元マニュアル p.879)](images/sysmac/ap-01-11.webp)

参考

デバイスマネージャーを起動したときに［他のデバイス］以外のデバイスとして認識される場合があります（例: ［USB シリアル デバイス］等）。USB ケーブルをNA5-££W££££-V2に対して抜き差しすることでデバイスマネージャ上で認識されたデバイスが更新されます。該当のデバイスに対してプロパティダイアログボックスを表示させてください。

6. ［ドライバーの更新］ボタンをクリックします。［ドライバーソフトウェアの更新］ダイアログボックスが表示されます。

![図(元マニュアル p.879)](images/sysmac/ap-01-12.webp)

7. ［コンピューターを参照してドライバーソフトウェアを検索します］をクリックします。ドライバーソフトウェアの参照先を指定するダイアログボックスが表示されます。

![図(元マニュアル p.880)](images/sysmac/ap-01-13.webp)

8. ［参照］ボタンをクリックし、3 で解凍したファイルがあるフォルダを選択してください。

![図(元マニュアル p.880)](images/sysmac/ap-01-14.webp)

9. ［次へ］ボタンをクリックします。［Windows セキュリティ］インストール警告メッセージが表示される場合は［インストール］

ボタンをクリックし、インストールを進めてください。USB ドライバのインストールが完了し、ダイアログボックスが表示されます。

![図(元マニュアル p.880)](images/sysmac/ap-01-15.webp)

## A-1-4 インストール後の確認方法

- 操作手順

1. パソコンとコントローラ、通信カプラユニットまたは表示器をUSB ケーブルで接続します。

2. パソコンでデバイスマネージャを表示します。

3. デバイスマネージャを確認し、次の名称が表示されていることを確認してください。コントローラまたは表示器（形NA5-££££££の場合）の場合: OMRON USB DirectLine通信カプラユニットの場合: OMRON USB AbstractControl表示器（形NA5-£W££££-V1）の場合: NA5 V1 USBClient表示器（形NA5-£W££££-V2）の場合: USB Ethernet/RNDIS Gadget
