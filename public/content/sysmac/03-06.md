# 3-6 基本的な編集操作

マルチビューエクスプローラやエディットウィンドウなどで行う編集操作の一覧を以下に示します。編集対象によっては存在しない操作もあります。

| 編集操作 | ボタン | Menu | コンテキストメニュー | マウス操作 | ショートカットキー |
|---|---|---|---|---|---|
| 挿入する | － | － | 追加 | － | [R]、[Shift]+[R] |
| 切り取る | | 編集｜切り取り | 切り取り | ドラッグ | [Ctrl]+[X] |
| コピーする | | 編集｜コピー | コピー | [Ctrl]+ドラッグ | [Ctrl]+[C] |
| 貼り付ける | | 編集｜貼り付け | 貼り付け | ドロップ | [Ctrl]+[V] |
| 削除する | | 編集｜削除 | 削除 | － | [DEL] |
| 左側を削除する | － | － | － | － | [BackSpace] |
| 名前を変更する | － | － | 名前の変更 | － | － |
| 行を選択する | － | － | すべてを選択 | 項目を囲む | [Ctrl]+[↑] |
| もとに戻す | | 編集｜元に戻す | 元に戻す | － | [Ctrl]+[Z] |
| やり直す | | 編集｜やり直す | やり直す | － | [Ctrl]+[Y] |
| 編集モードにする | － | － | 編集 | ダブルクリック | [F2] |
| 編集モードを抜ける | － | － | － | クリック | [Enter] |
| カーソルを移動する | － | － | － | － | [↑]、[↓]、[←]、[→] |
| カーソルを次、前へ移動 | － | － | － | － | [TAB]、[Shift]+[TAB] |
| カーソルを先頭へ移動 | － | － | － | － | [Home] |
| カーソルを行末へ移動 | － | － | － | － | [END] |
| 次ページに移動 | － | － | － | － | [PageDown] |

| 回路コメント一覧表示 | － | － | － | － | [L] |

（注）1. 編集対象により、行／項目の編集操作は異なります。（注）2. 他操作のショートカットは「A-6 ショートカットキー」を参照してください。

コンテキストメニューウィンドウに表示されている編集可能なデータ項目には、操作が簡単に行えるようにコンテキストメニューがあります。データ項目を選択した状態で右クリックすることで、［追加］［名前の変更］［コピー］［貼り付け］などの基本的な操作をメニューから選択することができます。

入力支援機能

以下の時、キーボードで入力した文字により、使用できる命令・パラメータを自動的に絞り込み、選択肢を表示します。

- 変数名、データ型の入力時

- ラダーエディタ、ST エディタでの変数名の入力時

- ラダーエディタ、ST エディタでのファンクションブロック／ファンクションのデータ型の入力時

- 変数テーブルでのデータ型の入力時例：ラダーエディタで変数名を入力するときに、マウスでクリックし、[Ctrl]+[Space]キーを押すか先頭の文字を入力すると、入力可能な変数名がドロップダウンリストで一覧表示されます。

![図(元マニュアル p.130)](images/sysmac/03-06-01.webp)

- ST エディタでの入力時

- インラインST を含むST エディタにて、先頭文字を入力すると、先頭文字に一致する文字列にカーソルが移動して一覧表示されます。

![図(元マニュアル p.130)](images/sysmac/03-06-02.webp)

- ST エディタにて、キーワードを入力（ここではIF を入力）した直後に［Tab］キーを押すと、ST 構文を構成するキーワードが自動的に入力されます。

![図(元マニュアル p.131)](images/sysmac/03-06-03.webp)

- ST エディタにて、ツールボックスのStructured Text Tools からST 構文（ここではIF 文）をドラッグ＆ドロップすると、ST 構文を構成するキーワードが自動的に入力されます。

![図(元マニュアル p.131)](images/sysmac/03-06-04.webp)
