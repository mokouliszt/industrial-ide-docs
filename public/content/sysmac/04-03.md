# 4-3 データ型の登録

- データ型の登録とは既存のデータ型を組み合わせて新たに任意のデータ型を定義することです。データ型エディタで登録したデータ型は、変数エディタのデータ型から選択して使用することができます。定義できるデータ型には構造体型、共用体型、列挙型があります。各データ型の詳細は、『NJ/NXシリーズCPUユニットユーザーズマニュアルソフトウェア編（SBCA-467）』または『NYシリーズ産業用パネル型PC／産業用ボックス型PCユーザーズマニュアルソフトウェア編（SBCA-436）』を参照してください。

- データ型登録画面の起動

1. マルチビューエクスプローラの［プログラミング］–［データ］–［データ型］をダブルクリックします。または、［プログラミング］–［データ］–［データ型］を右クリックして［編集］を選択します。

![図(元マニュアル p.244)](images/sysmac/04-03-01.webp)

データ型エディタが表示されます。

![図(元マニュアル p.244)](images/sysmac/04-03-02.webp)

- 構造体型の新規登録データ型を登録したあと、メンバを追加します。

1. データ型エディタのサイドタブ［構造体型］をクリックします。

![図(元マニュアル p.244)](images/sysmac/04-03-03.webp)

構造体型のデータ型エディタが表示されます。

![図(元マニュアル p.245)](images/sysmac/04-03-04.webp)

2. この例のようにデータ型が1つも登録されていない場合は、データ型エディタをクリックします（データ型の登録行が表示されます）。

![図(元マニュアル p.245)](images/sysmac/04-03-05.webp)

既にデータ型が登録されている場合に、データ型を追加登録するときは、データ型エディタ内で[Insert]キーを押します。または右クリックして［データ型新規作成］を選択します。

![図(元マニュアル p.245)](images/sysmac/04-03-06.webp)

3. 各項目を入力または選択して、[Enter]キーを押して確定します。

![図(元マニュアル p.245)](images/sysmac/04-03-07.webp)

構造体型のデータ型が登録されます。各属性の入力方法は、「データ型の属性項目の入力方法」を参照してください。

4. 次に、メンバを追加します。登録されたデータ型を選択後、右クリックして［メンバ新規作成］を選択します。

![図(元マニュアル p.246)](images/sysmac/04-03-08.webp)

メンバが新規作成されます。

5. 各項目を入力または選択して、[Enter]キーを押して確定します。メンバが構造体型データ型の下に追加されます。

![図(元マニュアル p.246)](images/sysmac/04-03-09.webp)

各属性の入力方法は、「データ型の属性項目の入力方法」を参照してください。

6. 手順4～5を実行して必要なメンバを追加します。

既にデータ型が登録されている場合に、データ型を追加登録するときは、データ型エディタ内で[Insert]キーを押します。または右クリックして［データ型新規作成］を選択します。

![図(元マニュアル p.247)](images/sysmac/04-03-12.webp)

![図(元マニュアル p.247)](images/sysmac/04-03-13.webp)

6. 手順4～5を実行して必要なメンバを追加します。

1. 削除するデータ型またはメンバ行の任意のセルをクリックして行全体を選択します。2 [Delete]キーを押します。または、右クリックメニューから［削除］を選択します。

3. [Ctrl]+[V]キーを押します。または、右クリックして［貼り付け］を選択します。

![図(元マニュアル p.251)](images/sysmac/04-03-27.webp)

**使用上の注意**

構造体型のメンバに別の構造体型を使用するとき、再帰または循環するような指定はできません。プログラムチェック時にエラーとなります。

**参考**

表計算ソフトなどからデータ型またはメンバをコピーして貼り付ける際に、データ型名またはメンバ名が重複している場合、コピー元と貼り付け先のデータ型/メンバ一覧がダイアログボックスで表示されます。上書きするデータ型またはメンバを選択して貼り付けを適用します。

- データ型の属性項目の入力方法データ型の属性項目の入力方法を説明します。各属性の詳細、および入力可能な数値の範囲や文字についての詳細は『NJ/NXシリーズCPUユニットユーザーズマニュアルソフトウェア編（SBCA-467）』または『NYシリーズ産業用パネル型PC／産業用ボックス型PCユーザーズマニュアルソフトウェア編（SBCA-436）』を参照してください。

| データ型の属性項目 | 説明 |
|---|---|
| 名称 | 任意の文字列を入力します。 |
| データ型 | データ型を直接入力するか、ドロップダウンリストから選択します。ただし構造体データ型はSTRUCT、共用体データ型はUNION固定です。 |
| オフセット種別 | 構造体メンバのオフセットの種類を選択します。 詳細は後述の「構造体型メンバのオフセット指定」 を参照してください。 |
| バイトオフセット | 構造体型先頭からのオフセットを表示・設定します。 オフセット種別が「任意」のときに入力することができます。 |
| ビットオフセット | ［バイトオフセット］で指定したバイトのビット0からのオフセットを表示・設定します。 オフセット種別が「任意」のときに入力することができます。 |
| 列挙値 | 列挙子に対応する整数値を入力します。 |
| コメント | 任意のコメントを入力します。 |

- 構造体型メンバのオフセット指定構造体型のメンバのオフセットを指定することで、メンバのメモリ配置を設定することができます。CJシリーズCPUユニットや外部機器など、メモリ配置が異なる機器とタグデータリンクを行うときに、構造体型のメンバのメモリ配置を一致させることができます。詳細は『NJ/NXシリーズCPUユニットユーザーズマニュアルソフトウェア編（SBCA-467）』または『NYシリーズ産業用パネル型PC／産業用ボックス型PCユーザーズマニュアルソフトウェア編（SBCA-436）』を参照してください。

メモリ配置は、目的に合わせて［オフセット種別］で設定します。

| オフセット種別 | 説明 |
|---|---|
| NJ | NJ/NX/NYシリーズコントローラに最適なメモリ配置になり、最も高速に動作します。通常は変更する必要はありません。 |
| CJ | CJシリーズCPUユニットに合わせたメモリ配置になります。バイトオフセット、ビットオフセットに、適切な値が自動的に設定されます。 |
| 任意 | バイトオフセット、ビットオフセットを任意に設定することができます。外部機器に合わせて、 バイトオフセット、ビットオフセットに任意の値を設定します。 |

バージョン関連情報構造体型メンバのオフセット指定は、CPUユニットVer.1.01以降とSysmac Studio Ver 1.02以降の組み合わせで使用可能です。

**使用上の注意**

オフセット指定したメンバを持つ構造体型を使用した変数では、メンバの初期値を指定することはできません。初期値を指定するとエラーとなります。

**参考**

- 表示するコメントを一括で切り替えることができます。また、表示コメントと切替用コメントは、変数テーブルから同時に編集することが可能です。詳細な手順は、「8-17変数・データ型コメント切り替え」を参照してください。

- 日本語でコメントを入力する場合、オプション設定で［IME自動起動］を有効にしておくと、コメント入力欄にフォーカスが移動したときに自動的に日本語入力がONになります。詳細は「8-26 Sysmac Studioオプション設定」を参照してください。
