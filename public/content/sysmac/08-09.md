# 8-9 ファームウェアアップデート履歴

ファームウェアのアップデート履歴をコントローラから読み出し、確認することができます。ファームウェアのアップデートについての詳細は、当社営業担当者までお問い合わせください。

- ファームウェアアップデート履歴の表示手順この機能はオンライン接続時のみ使用できます。

1. ［コントローラ］–［ファームウェアアップデート履歴］を選択します。［ファームウェアアップデート履歴］ダイアログボックスが表示されます。

| 記号 | 項目名 | 説明 |
|---|---|---|
| (a) | インデックス | アップデート順序を表示します。最も新しい履歴が「1」となります。 |
| (b) | 実施時刻(UTC) | ファームウェアアップデートを実施した時刻を表示します。時刻はUTC（協定世界時）で記録されます。 |
| (c) | ユニットバージョン | ファームウェアのバージョンを表示します。 |
| (d) | 識別子 | ファームウェアアップデータに格納されているシステム情報を表示します。 |
| (e) | アップデート種別 | ファームウェアのアップデート種別を表示します。 |
| (f) | 実行結果 | ファームウェアアップデートの実行結果を表示します。 |
| (g) | ［クリップボードへコピー］ボタン | ファームウェアアップデート履歴の内容をクリップボードへコピーします。 |
| (h) | ［閉じる］ボタン | ［ファームウェアアップデート履歴］を閉じます。 |

項目の詳細については、『NJ/NXシリーズCPUユニットユーザーズマニュアルソフトウェア編（SBCA-467）』を参照してください。
