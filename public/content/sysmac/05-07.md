# 5-7 モーション制御設定

- モーション制御設定とはモーション命令で利用する軸を登録し、軸が利用するサーボドライブ、エンコーダを軸に関連付け、軸のパラメータを設定する一連の設定をモーション制御設定と呼びます。

## 5-7-1 軸の設定

1. マルチビューエクスプローラの［構成・設定］–［モーション制御設定］–［軸設定］を右クリックし、メニューから［追加］–［モーション制御軸］、または［単軸位置制御軸］*1 を選択します。

![図(元マニュアル p.446)](images/sysmac/05-07-01.webp)

![図(元マニュアル p.446)](images/sysmac/05-07-02.webp)

「MC_Axis000(0)」のかっこ内は軸番号を表します。

NX/NY シリーズコントローラでは、「MC_Axis000(0,MC1)」のように表示されます。かっこ内はそれぞれ、軸番号、この軸が割り付けられたモーション制御を表します。*1. 形NX102、形NX1P2 コントローラの場合だけ表示されます。

2. MC_Axis000 をダブルクリックします。

![図(元マニュアル p.447)](images/sysmac/05-07-03.webp)

エディットウィンドウに軸パラメータ設定ビューが表示されます。初期画面では、「軸基本設定」が表示されます。

3. サーボ軸を利用する場合、以下を選択します。a)「軸使用」： 「使用軸」、または「未使用軸（使用軸に切替可）」b)「軸種別」： 「サーボ軸」

4. 出力デバイス1 に、登録されたサーボを選択します。

![図(元マニュアル p.447)](images/sysmac/05-07-04.webp)

（注）スレーブの設定については、「5-2 ［EtherCAT］の構成・設定」を参照してください。

参考

NJ シリーズ CPU ユニットがユニットバージョン1.04 以下の場合は、「入力デバイス」で登録したサーボを選択します。

5. 軸パラメータ設定ビューの左端に表示されるアイコンボタンで設定画面を切り替えます。それぞれのボタンで設定できるパラメータは、以下の通りです。切り替えアイコンボタンと軸パラメータとの関係

| 名称 | 概要 |
|---|---|
| 軸基本設定 | モーション制御を実行するタスク*1、軸の使用・未使用設定、軸番号、軸の種別、制御機能*2、入力または出力デバイス、チャネルを設定します。 |
| 単位変換設定 | 表示単位、および電子ギアのギア比を設定します。モータ1 回転あたりのパルス数、および移動量で設定します。CPU ユニット Ver.1.11 以降では、さらに減速機使用時の設定をすることができます。 |
| 動作設定 | 速度、加速度、減速度、トルク警告値、および監視用パラメータを設定します。 |
| 拡張動作設定 | サーボドライブのI/O の設定を行います。 |
| リミット設定 | ソフトウェアリミット、および位置偏差リミットの設定を行い ます。 |
| 原点復帰設定 | 原点復帰の動作設定を行います。 |
| 位置カウント設定 | コントローラのカウントモードの設定を行います。 |
| サーボドライバ設定 | サーボドライバのパラメータ設定を行います。 |

*1.形NX701 コントローラの場合に設定します。NJ シリーズコントローラでは設定項目は表示されません。*2.形NX102、形NX1P2 コントローラの場合に表示されます。初期状態では、［モーション制御軸］を追加した場合は［すべて］、［単軸位置制御軸］を追加した場合は［単軸位置制御のみ］が選択されます。

各項目の詳細は、『NJ/NX シリーズ CPU ユニット ユーザーズマニュアル モーション制御編（SBCE-433）』または『NY シリーズ 産業用パネル型PC ／産業用ボックス型PC ユーザーズマニュアル モーション制御編（SBCE-379）』を参照してください。

安全上の要点

［軸基本設定］画面の「詳細設定」を変更した場合は、本運転の前に、装置や機械が意図どおりに動作するかを確認してください。モーション制御機能モジュールの機能と、軸に割り当てたEtherCAT スレーブのプロセスデータとの関係が正しくないと、装置や機械が想定外の動作をする恐れがあります。

参考

軸パラメータ設定ビューで変更したパラメータは、自動的に全軸設定テーブルに反映されます。

モーション制御を実行するタスクの設定軸を登録すると、初期状態ではモーション制御を実行するタスクに「プライマリ定周期タスク」が設定されます。形NX701 コントローラでは、モーション制御を実行するタスクとして「プライマリ定周期タスク」または「定周期タスク（優先度5）」を設定できます。モーション制御を実行するタスクとして「定周期タスク（優先度5）」を設定する場合は、次の手順で行います。

1. ［タスク設定］で「定周期タスク（優先度5）」を登録します。 ［タスク設定］でのタスクの登録方法は、「5-9-1 タスクの登録」を参照してください。

![図(元マニュアル p.449)](images/sysmac/05-07-05.webp)

2. ［タスク設定］の［I/O を制御するタスク設定］で、軸に割り付けるスレーブ／ユニットのタスクとして「定周期タスク（優先度5）」のタスクを選択します。

![図(元マニュアル p.449)](images/sysmac/05-07-06.webp)

3. ［軸基本設定］の［入力デバイス］または［出力デバイス］で、「定周期タスク（優先度5）」を割り付けたスレーブ／ユニットを選択します。

![図(元マニュアル p.449)](images/sysmac/05-07-07.webp)

4. ［モーション制御］で「定周期タスク（優先度5）」を選択します。

![図(元マニュアル p.449)](images/sysmac/05-07-08.webp)

モーション制御を実行するタスクとして「定周期タスク（優先度5）」が設定されます。

使用上の注意

- スレーブ／ユニットを軸に割りつけた場合、該当するスレーブ／ユニットに割り付けるタスクは［タスク設定］の［I/O を制御するタスク設定］での設定よりも、［軸基本設定］での設定が優先されます。形NX701 コントローラで、軸に割り付けたスレーブ／ユニットに割り付けるタスクを変更したい場合は、［軸基本設定］の［モーション制御］で変更してください。該当するスレーブ／ユニットの［タスク設定］の［I/O を制御するタスク設定］に表示されるタスクが、自動的に［軸基本設定］で設定したタスクに変更されます。

- 軸に割り付けたスレーブ／ユニットに、［タスク設定］の［I/O を制御するタスク設定］で別のタスクを割り付けたい場合は、［軸基本設定］で［入力デバイス］または［出力デバイス］で該当するスレーブ／ユニットの選択を解除してから行ってください。

全軸設定テーブル

- 全軸設定テーブルとは登録されているすべての軸パラメータを表形式で表示したものを軸設定テーブルといいます。軸設定ビューと同じく、各軸のパラメータを編集可能です。

- 全軸設定テーブルの起動

1. マルチビューエクスプローラから［構成・設定］–［モーション制御設定］–［軸設定］を右クリックし、［全軸設定テーブル］を選択します。

![図(元マニュアル p.450)](images/sysmac/05-07-09.webp)

全軸設定テーブルが表示されます。

![図(元マニュアル p.450)](images/sysmac/05-07-10.webp)

（注）行が軸パラメータを、列が軸番号を表します。登録された軸をすべて表示します。軸パラメータのカテゴリ別に折りたたみが可能です。

- 軸パラメータを変更する

1. 変更したいパラメータをクリックします。

2. 変更するパラメータを入力します。

- 軸パラメータをコピーする

1. コピー元のセルを右クリックして［コピー］を選択します。

2. 貼り付け先のセルを右クリックして［貼り付け］を選択します。

参考

Ctrl キーまたはShift キーを押してセルをクリックすることで複数のセルを選択することができます。

- すべての軸パラメータを一度にコピーする

1. コピー元の軸名称を右クリックして［列のコピー］を選択します。

2. 貼り付け先の軸名称を右クリックして［貼り付け］を選択します。

- 軸を追加する軸名称を右クリックして［軸の追加］を選択すると、全軸設定テーブルの右端に新規に軸が追加されます。

参考

既存の軸をコピーして追加することができます。コピー元の軸名称を右クリックして［コピー］を選択したあと、右クリックメニューから［軸を追加して貼り付け］を選択します。

- 軸パラメータをフィルタ表示する利用頻度が高い基本的な軸パラメータと、すべての軸パラメータ表示を切り替えて表示します。

1. 全軸設定テーブルを表示します。

2. 全軸設定テーブルの上部の［表示パラメータ選択］で、［すべて］または［基本設定］を選択します。

- 軸を選択表示する軸の選択表示により、表示する軸パラメータを選んで表示することが可能です。

1. 全軸設定テーブルを表示します。

2. 全軸設定テーブルの任意の場所で右クリックし、［軸の表示選択］を選択します。以下の列の表示選択ダイアログボックスが表示されます形NX701 コントローラの場合

![図(元マニュアル p.452)](images/sysmac/05-07-11.webp)

全軸設定テーブルで変更を行った軸パラメータは、軸パラメータ設定ビューに自動反映されます。

軸設定をドライブの情報で更新する軸種別がサーボ軸で、1S シリーズサーボドライバまたはG5 シリーズサーボドライバ（リニアモータタイプを除く）を割り付けている場合は、サーボドライバから情報を取得して軸設定を更新することができます。更新対象となる軸設定の項目は、次のとおりです。

単位変換設定：モータ1 回転のパルス数、モータ1 回転の移動量、ワーク側1 回転の移動量動作設定：最高速度、ジョグ最高速度拡張動作設定：正方向トルク制限上限値、負方向トルク制限上限値位置カウント設定：エンコーダ種別

操作はコントローラにオンライン接続した状態で行います。

1. マルチビューエクスプローラの［軸設定］を右クリックし、［軸設定をドライブのデータで更新］を選択します。

![図(元マニュアル p.453)](images/sysmac/05-07-13.webp)

更新の実行確認ダイアログボックスが表示されます。

2. ［はい］ボタンをクリックします。軸設定の更新が完了すると、結果が表示されます。

![図(元マニュアル p.453)](images/sysmac/05-07-14.webp)

3. 更新された軸設定の項目の値を確認します。
