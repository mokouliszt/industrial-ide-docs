# 1-3 仕様

## 1-3-1 商品形式

Sysmac Studio の利用にはソフトウェアのインストールとライセンスの購入が必要です。ソフトウェアはライセンスポータルサイトからインストーライメージファイルを入手いただくか（64ビット版のみ）、DVD メディアを購入してください。

ライセンスは、64 ビット版向けのネットワークライセンスと32 ビット版向けのスタンドアロンライセンスがあります。

ネットワークライセンス（64 ビット版向け）とは、インターネット上に登録されたユーザID とライセンス番号およびパソコンを紐付けて認証するライセンスです。ライセンスの紛失がなくなり、ライセンスの認証／解除／譲渡を行うことができます。

スタンドアロンライセンス（32 ビット版向け）とは、Sysmac Studio の起動時にライセンス番号を入力することで使用可能なライセンスです。複数ユーザで使用する場合、製造装置の一部として納入する保全用のパソコンにバンドルする場合などに適しています。

ネットワークライセンス（64 ビット版のみ）

ネットワークライセンスを新規購入の際、インストールのためのインストーライメージファイルは、ライセンスポータルサイトにユーザ登録することでダウンロードすることができます。詳細はhttps://license-user.automation.omron.com/を参照してください。

## 1-3-5 ライセンスの種類

ライセンスにはネットワークライセンスとスタンドアロンライセンスの2 種類があります。

ネットワークライセンスネットワークライセンスには2 種類のグレードがあります。

| ライセンスのグレード | 説明 |
|---|---|
| ベーシックライセンス | 「1-3-4 使用可能な機種」のすべてを使用することができます。 |
| アドバンストライセンス | ベーシックライセンスに加え、3D シミュレーションオプション機能とチーム開発オ プション機能が使用できます。 |

スタンドアロンライセンススタンドアロンライセンスの種類は以下のとおりです。

| エディション | 説明 |
|---|---|
| スタンダードエディショ ン | 「1-3-4 使用可能な機種」のすべてを使用することができます。 |

| エディション | 説明 |
|---|---|
| ビジョンエディション | 画像センサFQ-M シリーズおよび画像センサFH シリーズのみ使用することがで きます。 |
| 変位センサエディション | ZW シリーズ変位センサのみ使用することができます。 |
| NX-I/O エディション | NX シリーズEtherNet/IP カプラユニットおよびPROFINET カプラユニットを使 用したスレーブターミナルのみ使用することができます。 |
| ドライブエディション | 1S シリーズ／G5 シリーズサーボドライバのみ使用することができます。 |
| セーフティエディション | NX シリーズ通信コントロールユニットまたはNX シリーズEtherNet/IP カプラ を使用したセーフティコントロールシステムのみ使用することができます。 |
| お試し版 （ライセンスは必要ありま せん） | インストール後30 日間限定で、スタンダードエディションと同じ機能を使用する ことができます。 |

複数のライセンスを同時に登録することができます。ライセンスの効力の優先度は次のとおりです。

| 優先度 | ライセンスの種類 |
|---|---|
| 1（高い） | スタンダードエディション |
| 2 | ビジョンエディション 変位センサエディション NX-I/O エディション ドライブエディション セーフティエディション |
| 3（低い） | お試し版 |

- 異なる優先度の複数のライセンスが同時に登録された場合は、優先度の高いライセンスの効力が有効になります。例：お試し版としてインストールしたSysmac Studio に、ビジョンエディションのライセンスを登録すると、画像センサだけを使用することができます。お試し版としては使用できません。

- 同じ優先度の複数のライセンスが同時に登録された場合は、それぞれのライセンスの効力が有効になります。例：変位センサエディションのライセンスを登録したSysmac Studio に、ビジョンエディションのライセンスを登録すると、変位センサと画像センサのどちらも使用することができます。ライセンスの登録方法は、「3-3-12 ライセンスの表示と登録」を参照してください。
