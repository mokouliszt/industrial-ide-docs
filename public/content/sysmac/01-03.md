# 1-3 仕様

## 1-3-1 商品形式

Sysmac Studioの利用にはソフトウェアのインストールとライセンスの購入が必要です。ソフトウェアはライセンスポータルサイトからインストーライメージファイルを入手いただくか（64ビット版のみ）、DVDメディアを購入してください。

ライセンスは、64ビット版向けのネットワークライセンスと32ビット版向けのスタンドアロンライセンスがあります。

ネットワークライセンス（64ビット版向け）とは、インターネット上に登録されたユーザIDとライセンス番号およびパソコンを紐付けて認証するライセンスです。ライセンスの紛失がなくなり、ライセンスの認証／解除／譲渡を行うことができます。

スタンドアロンライセンス（32ビット版向け）とは、Sysmac Studioの起動時にライセンス番号を入力することで使用可能なライセンスです。複数ユーザで使用する場合、製造装置の一部として納入する保全用のパソコンにバンドルする場合などに適しています。

ネットワークライセンス（64ビット版のみ）

ネットワークライセンスを新規購入の際、インストールのためのインストーライメージファイルは、ライセンスポータルサイトにユーザ登録することでダウンロードすることができます。詳細はhttps://license-user.automation.omron.com/を参照してください。

ライセンスにはネットワークライセンスとスタンドアロンライセンスの2種類があります。

ネットワークライセンスネットワークライセンスには2種類のグレードがあります。

| ライセンスのグレード | 説明 |

スタンドアロンライセンススタンドアロンライセンスの種類は以下のとおりです。

| エディション | 説明 |
|---|---|
| スタンダードエディション | 「1-3-4使用可能な機種」のすべてを使用することができます。 |
| ビジョンエディション | 画像センサFQ-Mシリーズおよび画像センサFHシリーズのみ使用することができます。 |
| 変位センサエディション | ZWシリーズ変位センサのみ使用することができます。 |
| NX-I/Oエディション | NXシリーズEtherNet/IPカプラユニットおよびPROFINETカプラユニットを使用したスレーブターミナルのみ使用することができます。 |
| ドライブエディション | 1Sシリーズ／G5シリーズサーボドライバのみ使用することができます。 |
| セーフティエディション | NXシリーズ通信コントロールユニットまたはNXシリーズEtherNet/IPカプラを使用したセーフティコントロールシステムのみ使用することができます。 |
| お試し版 （ライセンスは必要ありません） | インストール後30日間限定で、スタンダードエディションと同じ機能を使用することができます。 |

複数のライセンスを同時に登録することができます。ライセンスの効力の優先度は次のとおりです。

| 優先度 | ライセンスの種類 |
|---|---|
| 1（高い） | スタンダードエディション |
| 2 | ビジョンエディション変位センサエディションNX-I/Oエディションドライブエディションセーフティエディション |
| 3（低い） | お試し版 |

- 異なる優先度の複数のライセンスが同時に登録された場合は、優先度の高いライセンスの効力が有効になります。例：お試し版としてインストールしたSysmac Studioに、ビジョンエディションのライセンスを登録すると、画像センサだけを使用することができます。お試し版としては使用できません。

- 同じ優先度の複数のライセンスが同時に登録された場合は、それぞれのライセンスの効力が有効になります。例：変位センサエディションのライセンスを登録したSysmac Studioに、ビジョンエディションのライセンスを登録すると、変位センサと画像センサのどちらも使用することができます。ライセンスの登録方法は、「3-3-12ライセンスの表示と登録」を参照してください。
