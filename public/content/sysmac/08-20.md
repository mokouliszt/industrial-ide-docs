# 8-20 STプログラムのインポート

MathWorks®社製Simulink® PLC Code™にてOMRON Sysmac Studio 向けに出力されたxml ファイルをインポートする機能です。

- インポート対象プログラム、ファンクション、ファンクションブロック、データ型、グローバル変数

- インポート手順

1. ［ツール］メニューから［ST プログラムのインポート］を選択します。

![図(元マニュアル p.795)](images/sysmac/08-20-01.webp)

ファイル選択ダイアログボックスが表示されます。2インポート対象のxml ファイルを選択し、［開く］ボタンをクリックします。

![図(元マニュアル p.795)](images/sysmac/08-20-02.webp)

インポートが実行され、プロジェクト内にxml ファイルのプログラム、ファンクション、ファンクションブロック、データ型、グローバル変数が追加されます。インポートにより追加された内容は、アウトプットウィンドウに表示されます。

参考

R2013a 以降のSimulink PLC Coder にてOMRON Sysmac Studio 向けにコード生成可能です。Simulink PLC Coder の詳細については、MathWorks 社の『Simulink PLC Coder User's Guide』を参照してください。
