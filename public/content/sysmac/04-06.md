# 4-6 STプログラミング

- POU（プログラム、ファンクションブロック、ファンクション）内のアルゴリズムを表現するために使用できるプログラム言語は、ラダー図言語（LD）、ST 言語（Structured Text：構造化テキスト）の2 種類です。本節ではST 言語の入力手順を説明します。

- ST プログラミングとはST(Structured Text)言語を使用して処理アルゴリズムを構築することです。ST エディタ上で数値演算処理を記述します。

- ST の編集とはST 言語の文を組み合わせてアルゴリズムを記述することです。文の構成や表現の詳細については、『NJ/NX シリーズ CPU ユニット ユーザーズマニュアル ソフトウェア編（SBCA-467）』または『NY シリーズ 産業用パネル型PC／産業用ボックス型PC ユーザーズマニュアル ソフトウェア編（SBCA-436）』を参照してください。

## 4-6-2 ST エディタの基本操作

代入文の入力

変数「ccc」に変数「aaa」＋変数「bbb」の結果を代入する以下の代入文を例で、操作手順を説明します。「ccc := aaa + bbb ;」

変数「aaa」、「bbb」、「ccc」をあらかじめ変数テーブルに登録しておきます。ST エディタ上で未登録の変数名を入力したときに、変数テーブルに新規登録することもできます。詳細は、「ST エディタ上での変数の登録」を参照してください。

1. ST エディタ上で、変数名「ccc」の先頭文字（c）を入力します。ドロップダウンリストに変数候補が表示されます。

![図(元マニュアル p.331)](images/sysmac/04-06-03.webp)

2. 変数「ccc」を選択し、[Enter]キーを押します。変数「ccc」が入力されます。

3. スペースおよび「:」（コロン）を入力します。代入のキーワード「:=」が入力されます。

4. 変数「aaa」の先頭文字を入力します。5ドロップダウンリストの変数候補から変数「aaa」を選択し、[Enter]キーを押します。変数「aaa」が入力されます。

6. スペースおよび「＋」を入力し、変数「bbb」の先頭文字を入力します。7ドロップダウンリストの変数候補から「bbb」を選択し、「;」（セミコロン）を入力し、[Enter]キーを押します。代入文の入力が完了します。

参考

使用上の注意

- ST 言語入力時の制限（演算子の個数、POU 引数の個数、POU 呼び出しのネスト数、条件／反復文のネスト数の制限）は、『NJ/NX シリーズ CPU ユニット ユーザーズマニュアル ソフトウェア編（SBCA-467）』または『NY シリーズ 産業用パネル型PC／産業用ボックス型PC ユーザーズマニュアル ソフトウェア編（SBCA-436）』を参照してください。

- ひとつのPOU（プログラム、ファンクションまたはファンクションブロック）で記述できる行数は最大10000 行です。

- ST エディタでファンクションブロックを使用する場合は、あらかじめ変数テーブルにファンクションブロックインスタンスを登録しておきます。ST エディタ上でファンクションブロックインスタンスを登録することはできません。

- ST 要素を貼り付けた後は、使用されている変数とデータ型が正しいかどうか必ず確認してください。
