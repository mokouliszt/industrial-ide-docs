# 8-23 モータ選定ツール結果のインポート

モータ選定ツールの結果をインポートする機能です。

- インポート対象

EtherCAT の構成・設定、ドライブ設定、軸設定および軸グループ設定

- インポート手順

1. ［ツール］メニューから［モータ選定ツール結果のインポート］を選択します。

![図(元マニュアル p.806)](images/sysmac/08-23-01.webp)

インポート確認ダイアログボックスが表示されます。

![図(元マニュアル p.806)](images/sysmac/08-23-02.webp)

2. ［はい］ボタンをクリックします。ファイル選択ダイアログボックスが表示されます。

3. インポート対象のxml ファイルを選択し、［開く］ボタンをクリックします。現在選択中のコントローラとモータ選定ツール結果ファイルのコントローラが異なる場合は、インポートの継続確認ダイアログボックスが表示されます。

![図(元マニュアル p.806)](images/sysmac/08-23-03.webp)

4. ［はい］ボタンをクリックします。インポートが実行され、プロジェクトにドライブ、軸、軸グループが追加されます。インポートが完了すると、完了メッセージボックスが表示されます。インポートによって追加された内容やエラー情報は、アウトプットウィンドウに表示されます。

参考

モータ選定ツールの詳細については、『モータ選定ツール スタートアップガイド（SBCE-081）』を参照してください。
