# A-5 オンラインヘルプ

オンラインヘルプとは

- Sysmac Studio のメニューから起動し、命令語の説明やショートカットキーを確認することができるものです。次の4 種類のヘルプがあります。

- Sysmac Studio 本体のヘルプ（「目次」と表示されます）

- 命令語リファレンス

- ショートカットキーリファレンス

- システム定義変数リファレンス

## A-5-1 Sysmac Studio 本体のヘルプ（目次）

Sysmac Studio 本体のヘルプとは

Sysmac Studio の使い方を説明しています。

## A-5-2 命令語リファレンス

命令語リファレンスとは

NJ/NX/NY シリーズで使用できる基本命令およびモーション命令について説明しています。各命令の説明は、次のマニュアルでの説明と同じ内容です。『NJ/NX シリーズ コマンドリファレンスマニュアル 基本編（SBCA-468）』『NJ/NX シリーズ コマンドリファレンスマニュアル モーション編（SBCE-434）』『NY シリーズ コマンドリファレンスマニュアル 基本編（SBCA-437）』『NY シリーズ コマンドリファレンスマニュアル モーション編（SBCE-380）』

- 命令語リファレンスの表示手順次の2 通りの操作方法があります。方法1: メインメニューからの表示方法2: ラダーエディタ、ST エディタ、および命令ナビゲーションからの表示方法1: メインメニューからの表示メインメニューで［ヘルプ］–［命令語リファレンス］を選択します。「Sysmac Studio　コマンドリファレンスヘルプ」が表示されます。

方法2: ラダーエディタ、ST エディタ、および命令ナビゲーションからの表示ラダーエディタまたはST エディタ上で、入力済みの命令を選択するか、ツールボックスの命令ナビゲーションから目的の命令を選択し、［F1］キーを押します。「Sysmac Studio　コマンドリファレンスヘルプ」が表示され、選択した命令のページが表示されます。

## A-5-3 ショートカットキーリファレンス

ショートカットキーリファレンスとは

Sysmac Studio を使用する上で便利なショートカットキーを一覧表示できます。

- ショートカットキーリファレンスの表示手順メインメニューから［ヘルプ］–［ショートカットキーリファレンス］を選択します。ショートカットキーリファレンスヘルプが表示されます。詳細は、「A-6 ショートカットキー」を参照してください。

## A-5-4 システム定義変数一覧

システム定義変数一覧とは

Sysmac Studio で使用するシステム定義変数の説明を一覧表示できます。

- システム定義変数一覧の表示手順メインメニューから［ヘルプ］–［システム定義変数リファレンス］を選択します。システム定義変数一覧が表示されます。
