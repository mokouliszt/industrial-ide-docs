# 4-5 ラダープログラミング

POU（プログラム、ファンクションブロック、ファンクション）内のアルゴリズムを表現するために使用できるプログラム言語は、ラダー図言語（LD）、ST 言語（Structured Text：構造化テキスト）の2 種類です。本節ではラダー図言語の入力手順を説明します。• ラダープログラミングとは回路部品を接続線でつなぎ、アルゴリズムを構築することです。ラダーエディタ上で回路部品や接続線の入力を行います。

## 4-5-1 ラダーエディタの起動／操作手順

プログラムの場合

1. マルチビューエクスプローラの［プログラミング］–［POU］–［プログラム］で、登録済みのラダープログラムのセクションをダブルクリックします。または、右クリックして［編集］を選択します。

![図(元マニュアル p.260)](images/sysmac/04-05-01.webp)

プログラムのラダーエディタが表示されます。

![図(元マニュアル p.260)](images/sysmac/04-05-02.webp)

ラダープログラムの登録方法は、「4-2-2 プログラムの登録」を参照してください。

ファンクションブロック／ファンクションの場合

1. マルチビューエクスプローラの［プログラミング］–［POU］–［ファンクションブロック（またはファンクション）］から登録済みのラダーファンクションブロックまたはラダーファンクションをダブルクリックします。または、右クリックメニューから［編集］を選択します。

![図(元マニュアル p.261)](images/sysmac/04-05-03.webp)

ファンクションブロックまたはファンクションのラダーエディタが表示されます。

![図(元マニュアル p.261)](images/sysmac/04-05-04.webp)

ラダーファンクションブロック／ラダーファンクションの登録方法は、「4-2-3 ファンクションブロックの登録」および「4-2-4 ファンクションの登録」を参照してください。

## 4-5-2 セクションの追加／削除

- セクションとはラダープログラムは任意の管理単位に分割することができます。この分割単位を「セクション」と言います。

セクションの追加

1. マルチビューエクスプローラの［プログラミング］–［POU］–［プログラム］で、登録済みのラダープログラムを選択します。

![図(元マニュアル p.262)](images/sysmac/04-05-05.webp)

2. 右クリックして［追加］–［セクション］を選択します。

![図(元マニュアル p.262)](images/sysmac/04-05-06.webp)

選択したラダープログラムの下に、セクションが追加されます。

![図(元マニュアル p.262)](images/sysmac/04-05-07.webp)

セクションの順序の変更プログラムは、マルチビューエクスプローラに表示されるセクションの並び順に従って上から下へ順に実行されます。実行順を変更するにはセクションの順序を変更します。

1. マルチビューエクスプローラの［プログラミング］–［POU］–［プログラム］以下から、順序を変更するセクションを選択します。

![図(元マニュアル p.262)](images/sysmac/04-05-08.webp)

2. 右クリックして、上に移動するには［上に移動］、下に移動するには［下に移動］を選択します。また、ドラッグ＆ドロップによる移動も可能です。

![図(元マニュアル p.263)](images/sysmac/04-05-09.webp)

対象のセクションの順序が上または下に変更されます。

![図(元マニュアル p.263)](images/sysmac/04-05-10.webp)

既存のセクションを右クリックして、［上に挿入］メニューから［セクション］を選択することで、選択したセクションの上に新しいセクションを挿入することもできます。

セクションの削除

1. マルチビューエクスプローラの［プログラミング］–［POU］–［プログラム］で、登録済みのラダープログラム以下から、削除するセクションを選択します。

![図(元マニュアル p.263)](images/sysmac/04-05-11.webp)

2. 右クリックして［削除］を選択します。またはメインメニューから［編集］–［削除］を選択します。[Delete]ボタンを押して削除することも可能です。

![図(元マニュアル p.264)](images/sysmac/04-05-12.webp)

対象のセクションが削除されます。

セクションのコピー／貼り付け

- セクションのコピーには、次の種類があります。

- ［コピー］: セクションの回路データだけをコピーします。

- ［参照先も含めてコピー］: セクションの回路データに加えて、セクション内で参照している変数と、変数に使用されているデータ型もコピーします。

1. マルチビューエクスプローラの［プログラミング］–［POU］–［プログラム］で、登録済みのラダープログラムから、コピーするセクションを選択します。

![図(元マニュアル p.264)](images/sysmac/04-05-13.webp)

2. [Ctrl]+[C]キーを押します。または、右クリックして［コピー］を選択します。参照している変数や、変数に使用されているデータ型も含めてコピーする場合は、［参照先も含めてコピー］を選択します。

![図(元マニュアル p.264)](images/sysmac/04-05-14.webp)

3. [Ctrl]+[V]キーを押します。または、右クリックして［貼り付け］を選択します。

![図(元マニュアル p.265)](images/sysmac/04-05-15.webp)

- ［コピー］を選択した場合コピーされたセクションが追加されます。同じ名前のセクションが存在する場合、上書きを確認するメッセージボックスが表示されます。［はい］を選択するとセクションは上書きされます。［いいえ］を選択すると「コピー元のセクション名のコピー1」というセクション名で追加されます。

![図(元マニュアル p.265)](images/sysmac/04-05-16.webp)

- ［参照先も含めてコピー］を選択した場合［コピー］メニューを選択した場合の貼り付けに加えて、セクションで使用されている変数とデータ型も、貼り付け先のプログラムに追加されます。貼り付け先のプログラムに同じ名前の変数やデータ型が存在し、それらの属性が異なる場合は［貼り付け時の照合］ダイアログボックスが表示されます。［貼り付け時の照合］ダイアログボックスの詳細は、「4-7 貼り付け時の照合」を参照してください。

使用上の注意

- マルチビューエクスプローラ上でのセクションに対する操作は、元に戻す／やり直すことはできません。セクションを削除すると元に戻すことはできませんので注意してください。

- プログラムに対する変更は、そのプログラムの下のすべてのセクションに反映されます。

- セクションを貼り付けた後は、使用されている変数とデータ型が正しいかどうか必ず確認してください。

参考

複数のセクションの表示を切り替えるには、対象のセクション名のタブを選択します。

## 4-5-3 ラダーエディタの基本操作

回路部品の基本操作

- 回路部品の挿入とはラダーエディタ上で、アルゴリズムの作成のために回路部品を入力することです。回路部品には以下の種類があります。

| 回 | 内容 |
|---|---|
| 接点 | 入力です |
| コイル | 出力です |
| ファンクションブロック | 動作を定義した命令です |
| ファンクション | 単機能の命令です |
| インラインST | ラダープログラム内でST プログラムを記述します。 |
| ジャンプ | 他の回路へのジャンプの起点です |
| ジャンプラベル | ジャンプ先の回路を示すラベルです |

- 各回路部品の挿入操作手順回路部品を挿入する基本的な操作方法は次の4 通りです。方法1: ツールボックスから回路部品をドラッグ＆ドロップする方法2: 接続線を選択して、右クリックメニューから［回路部品の挿入］を選択する方法3: 接続線を選択して、回路部品の挿入のショートカットキーを押す方法4: 接続線を選択して、メインメニューの［挿入］–［回路部品］から各回路部品を選択する接点／コイルの挿入

使用上の注意

- コピーする回路内でインラインST を使用している場合、回路パターンコピーが実行できません。

- オンラインエディット中に回路パターンコピーを使用する場合は、本機能でグローバル変数を作成することができません。変数名の変更対象のチェックを外して実行してください。
