# 10-1 Sysmac Studioの周辺ツール

- 周辺ツールの概要

Sysmac Studio の周辺ツールの概要を示します。周辺ツールの操作方法は、それぞれの参照マニュアルまたはヘルプを参照してください。

| 周辺ツール名 | 概要 | 参照マニュアル／ヘルプ |
|---|---|---|
| CX-Integrator | ネットワーク構築（データリンク、ルーチングテ ーブル、通信ユニット設定など）を行います。 | CX-Integrator オペレーションマニ ュアル（SBCA-347） |
| CX-Designer | プログラマブルターミナルNS シリーズの画面 データを作成します。 | CX-Designer ユーザーズマニュアル （SBSA-532） |
| CX-Protocol | CJ シリーズシリアルコミュニケーションユニッ トと接続された、汎用外部機器とのデータ送受信 手順（プロトコル）を作成します。 | CX-Protocol オペレーションマニュ アル（SBCA-307） |
| Network Configurator | NJ/NX シリーズCPU ユニットまたはNY シリ ーズ産業用PC の内蔵EtherNet/IP ポートでタ グデータリンクの設定を行います。 | NJ/NX シリーズ CPU ユニット 内蔵 EtherNet/IP ポート ユーザーズマニ ュアル（SBCD-377）またはNY シ リーズ 産業用パネル型PC／産業用 ボックス型PC ユーザーズマニュア ル 内蔵EtherNet/IP ポート編 （SBCD-369） |
| CX- ConfiguratorFDT | IO-Link マスタユニットとスレーブ間の通信設 定を行います。 | CX-ConfiguratorFDT ヘルプ |
| Adept Robot IP ア ドレス設定ツール | Adept Robot、SmartController EX のIP アド レス、サブネットマスクを変更します。 | Adept Robot IP アドレス設定ツー ルヘルプ |
