# 10-1 Sysmac Studioの周辺ツール

- 周辺ツールの概要

Sysmac Studioの周辺ツールの概要を示します。周辺ツールの操作方法は、それぞれの参照マニュアルまたはヘルプを参照してください。

| 周辺ツール名 | 概要 | 参照マニュアル／ヘルプ |
|---|---|---|
| CX-Integrator | ネットワーク構築（データリンク、ルーチングテーブル、通信ユニット設定など）を行います。 | CX-Integratorオペレーションマニュアル（SBCA-347） |
| CX-Designer | プログラマブルターミナルNSシリーズの画面データを作成します。 | CX-Designerユーザーズマニュアル （SBSA-532） |
| CX-Protocol | CJシリーズシリアルコミュニケーションユニットと接続された、汎用外部機器とのデータ送受信手順（プロトコル）を作成します。 | CX-Protocolオペレーションマニュアル（SBCA-307） |
| Network Configurator | NJ/NXシリーズCPUユニットまたはNYシリーズ産業用PCの内蔵EtherNet/IPポートでタグデータリンクの設定を行います。 | NJ/NXシリーズCPUユニット内蔵EtherNet/IPポートユーザーズマニュアル（SBCD-377）またはNYシリーズ産業用パネル型PC／産業用ボックス型PCユーザーズマニュアル内蔵EtherNet/IPポート編 （SBCD-369） |
| CX- ConfiguratorFDT | IO-Linkマスタユニットとスレーブ間の通信設定を行います。 | CX-ConfiguratorFDTヘルプ |
| Adept Robot IPアドレス設定ツール | Adept Robot、SmartController EXのIPアドレス、サブネットマスクを変更します。 | Adept Robot IPアドレス設定ツールヘルプ |
