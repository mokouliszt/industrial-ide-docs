# 10-1 Sysmac Studioの周辺ツール

- 周辺ツールの概要

Sysmac Studio の周辺ツールの概要を示します。周辺ツールの操作方法は、それぞれの参照マニュアルまたはヘルプを参照してください。

| 周辺ツール名 | 概要 | 参照マニュアル／ヘルプ |
|---|---|---|
| CX-Integrator | ネットワーク構築（データリンク、ルーチングテーブル、通信ユニット設定など）を行います。 | CX-Integrator オペレーションマニュアル（SBCA-347） |
| CX-Designer | プログラマブルターミナルNS シリーズの画面データを作成します。 | CX-Designer ユーザーズマニュアル（SBSA-532） |
| CX-Protocol | CJ シリーズシリアルコミュニケーションユニットと接続された、汎用外部機器とのデータ送受信手順（プロトコル）を作成します。 | CX-Protocol オペレーションマニュアル（SBCA-307） |
| Network Configurator | NJ/NX シリーズCPU ユニットまたはNY シリーズ産業用PC の内蔵EtherNet/IP ポートでタグデータリンクの設定を行います。 | NJ/NX シリーズ CPU ユニット 内蔵EtherNet/IP ポート ユーザーズマニュアル（SBCD-377）またはNY シリーズ 産業用パネル型PC／産業用ボックス型PC ユーザーズマニュアル 内蔵EtherNet/IP ポート編（SBCD-369） |
| CX-ConfiguratorFDT | IO-Link マスタユニットとスレーブ間の通信設定を行います。 | CX-ConfiguratorFDT ヘルプ |
| Adept Robot IP アドレス設定ツール | Adept Robot、SmartController EX のIP アドレス、サブネットマスクを変更します。 | Adept Robot IP アドレス設定ツールヘルプ |
