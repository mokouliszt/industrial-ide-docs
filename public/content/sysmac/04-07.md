# 4-7 貼り付け時の照合

マルチビューエクスプローラやラダーエディタ、ST エディタ上で［参照先も含めてコピー］メニューを選択してコピーしたデータを貼り付けたとき、貼り付け先に同じ名前の変数やデータ型が存在し、それらの属性が異なる場合は、該当データについて上書きするかどうかを確認するための［貼り付け時の照合］ダイアログボックスが表示されます。相違内容を確認して、上書きを実施します。

4-7-1［貼り付け時の照合］ダイアログボックス

(a) (b)

![図(元マニュアル p.345)](images/sysmac/04-07-01.webp)

| | 項目 | 説明 |
|---|---|---|
| (a) | 貼り付け前のデータ | 相違のある箇所について、貼り付け前のデータを表示します。 相違があるデータの行は赤字で表示されます。 |
| (b) | 貼り付けるデータ | 相違のある箇所について、貼り付けるデータを表示します。 |
| (c) | ［詳細照合］ボタン | 相違の詳細な内容を表示する詳細照合画面を表示します。 |
| (d) | ［すべての差異を上書き］ボ タン | 相違があるすべての貼り付けデータを、貼り付け前のデータに上書きしま す。 |
| (e) | ［選択したデータを上書き］ ボタン | 選択行の貼り付けデータを貼り付け前のデータに上書きします。 |
| (f) | ［閉じる］ボタン | 画面を閉じます。 |
