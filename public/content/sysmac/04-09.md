# 4-9 プログラムチェック

- プログラムチェックとは作成したPOU（プログラム、ファンクションブロック、ファンクション）の書式に誤りがないかどうかを確認することです。プログラムチェックには、次の2 つの方法があります。a) 全プログラムチェックプロジェクト内のすべてのプログラムをチェックします。b) 部分プログラムチェック選択したプログラムをチェックします。

## 4-9-1 全プログラムチェック

1. メインメニューから［プロジェクト］–［全プログラムチェック］を選択します。プログラムチェック結果がビルドウィンドウに表示されます。

![図(元マニュアル p.354)](images/sysmac/04-09-01.webp)

## 4-9-2 部分プログラムチェック

1. マルチビューエクスプローラ上で、対象のPOU またはセクションを選択します。2メインメニューから［プロジェクト］–［部分プログラムチェック］を選択します。プログラムチェック結果がビルドウィンドウに表示されます。

![図(元マニュアル p.354)](images/sysmac/04-09-02.webp)

- プログラムチェック結果の表示プログラムチェックの結果はビルドウィンドウに次のように表示されます。

![図(元マニュアル p.355)](images/sysmac/04-09-03.webp)

| 項目 | 内容 | 備考 |
|---|---|---|
| エラー数 | エラーの総数を表示します。 | |
| 警告数 | 警告の総数を表示します。 | |
| エラー／警告番号 | エラー／警告を見つかった順に表示します。 | |
| エラー／警告内容 | エラー／警告の内容を表示します。 | |
| エラー／警告箇所 | エラー／警告の発生箇所を表示します。 | エラー発生箇所へジャンプすることができます。 |

参考

特定の警告を表示しないようにすることができます。詳細は、「8-26 Sysmac Studio オプション設定」を参照してください。

- エラー発生箇所へのジャンプビルドウィンドウの対象エラーの行をダブルクリックします。エディットウィンドウに、該当箇所が表示されます。

![図(元マニュアル p.355)](images/sysmac/04-09-04.webp)

エラーの内容については、「11-1 ラダープログラムチェックのエラーメッセージ一覧」、「11-2 ST プログラムチェックのエラーメッセージ一覧」を参照してください。

- マルチビューエクスプローラでの表示エラーのあるPOU は、マルチビューエクスプローラに「」マークが表示されます。

![図(元マニュアル p.356)](images/sysmac/04-09-05.webp)
