# 8-8 ユニット生産情報の表示

オンライン状態で、CPU ユニットやCPU ユニットに装着されたユニット、EtherCAT スレーブなどの生産情報を生産情報ダイアログボックスに表示します。生産情報とは、形式、バージョン、ロット番号などの情報を指します。共通の画面操作、および各ユニットの表示項目について説明します。

表示画面

生産情報ダイアログボックスには、簡易表示と詳細表示があります。

![図(元マニュアル p.718)](images/sysmac/08-08-01.webp)

![図(元マニュアル p.718)](images/sysmac/08-08-02.webp)

簡易表示：詳細表示

［簡易表示］ボタンまたは［詳細表示］ボタンをクリックすると、簡易表示と詳細表示が切り替わります。

ファイル出力

生産情報をファイルに保存することができます。

1. 生産情報ダイアログボックスの［ファイル出力］ボタンをクリックします。［ファイルを保存］ダイアログボックスが表示されます。

2. 任意のファイル名を付けて、［保存］ボタンをクリックします。拡張子.csv のCSV 形式（カンマ区切り）のファイルが保存されます。

## 8-8-1 CPU ユニット生産情報の表示

NX/NY シリーズコントローラ

CPU ユニットや産業用PC の生産情報、およびX バスユニットやNX ユニットが装着されている場合は各ユニットの生産情報が表示されます。また、形NX1P2 CPU ユニットの場合は、オプションボードの生産情報も表示されます。

- 表示方法

1. マルチビューエクスプローラで［CPU・増設ラック］–［CPU ラック］を右クリックして［生産情報表示］を選択します。生産情報ダイアログボックスが表示されます。

- 表示項目

表示項目は以下のとおりです。

| CPU ユニット／産業用PC | X バスユニット*1／NX ユニット*2 | オプションボード*3 |
|---|---|---|
| • 形式、ユニットバージョン • LOT 番号 • シリアル番号 • ハードウェアリビジョン • ソフトウェアバージョン • 累積通電時間*4 | • 形式、ユニットバージョン • LOT 番号 • シリアル番号 • ハードウェアバージョン • ソフトウェアバージョン • 累積通電時間 | • 形式、ユニットバージョン • LOT 番号 • シリアル番号 • ハードウェアバージョン • ソフトウェアバージョン |

*1.形NX502 CPU ユニットにX バスユニットを装着している場合だけ表示されます。*2.形NX502、形NX102、形NX1P2 CPU ユニットにNX ユニットを装着している場合だけ表示されます。*3.形NX1P2 CPU ユニットにオプションボードを装着している場合だけ表示されます。*4.形NX502、形NX102、形NX1P2 CPU ユニットの場合だけ表示されます。

表示項目の内容の詳細は、各ユニットのマニュアルを参照してください。

NJ シリーズコントローラ

CPU ユニットおよびCJ 高機能ユニットの生産情報が表示されます。

- 表示方法

1. マルチビューエクスプローラで［構成・設定］–［CPU・増設ラック］をダブルクリックします。または、［構成・設定］–［CPU・増設ラック］を右クリックして［編集］を選択します。エディットウィンドウにユニットエディタが表示されます。

2. ユニットエディタでユニットのない場所を右クリックして［生産情報表示］を選択します。生産情報ダイアログボックスが表示されます。

- 表示項目

表示内容は以下のとおりです。

| CPU ユニット | CJ 高機能ユニット |
|---|---|
| • ユニット形式、ユニットバージョン • LOT 番号 • ハードウェアバージョン/ハードウェアリビジ ョン*1 • ソフトウェアバージョン | • ラック番号、スロット番号、号機No.／ユニット番号 • ユニット形式、ユニットバージョン • LOT 番号 • ユニットレビジョン • PCB レビジョン • ソフトウェアレビジョン |

*1.ハードウェアリビジョンが「なし」のユニットでは、ハードウェアリビジョンの表示はありません。

表示項目の内容の詳細は、各ユニットのマニュアルを参照してください。

## 8-8-2 EtherCAT スレーブ生産情報の表示

EtherCAT スレーブのスレーブに設定されているノードアドレス、形式*1、シリアル番号が表示されます。*1. 形式を特定できない場合（ESI ファイルがない場合など）は、ベンダーID、プロダクトコード、リビジョンナンバーが表示されます。

- 表示方法

1. マルチビューエクスプローラで［構成・設定］–［EtherCAT］をダブルクリックします。または、［構成・設定］–［EtherCAT］を右クリックして［編集］を選択します。エディットウィンドウにEtherCAT 構成編集画面が表示されます。

2. EtherCAT 構成編集画面でマスタを右クリックして［生産情報表示］を選択します。生産情報ダイアログボックスが表示されます。

- 表示項目

- 表示項目は以下のとおりです。

- 形式、リビジョン

- シリアル番号

- ハードウェアバージョン

- ソフトウェアバージョン表示項目の内容の詳細は、各スレーブのマニュアルを参照してください。

## 8-8-3 EtherCAT スレーブターミナル構成ユニットの生産情報の表示

EtherCAT スレーブターミナルを構成するユニットの生産情報が表示されます。

- 表示方法

1. マルチビューエクスプローラで［EtherCAT］上の対象のEtherCAT カプラユニットをダブルクリックします。または、［EtherCAT］上の対象のEtherCAT カプラユニット右クリックして［編集］を選択します。

エディットウィンドウにスレーブターミナル構成編集画面が表示されます。

2. スレーブターミナル構成編集画面上で、右クリックして［生産情報表示］を選択します。生産情報ダイアログボックスが表示されます。

- 表示項目

- 表示項目は以下のとおりです。

- 形式、ユニットバージョン

- LOT 番号

- シリアル番号

- ハードウェアバージョン

- ソフトウェアバージョン

- 累積通電時間表示項目の内容の詳細は、各ユニットのマニュアルを参照してください。
