# 8-8 ユニット生産情報の表示

オンライン状態で、CPUユニットやCPUユニットに装着されたユニット、EtherCATスレーブなどの生産情報を生産情報ダイアログボックスに表示します。生産情報とは、形式、バージョン、ロット番号などの情報を指します。共通の画面操作、および各ユニットの表示項目をまとめます。

表示画面

生産情報ダイアログボックスには、簡易表示と詳細表示があります。

![図(元マニュアル p.718)](images/sysmac/08-08-01.webp)

![図(元マニュアル p.718)](images/sysmac/08-08-02.webp)

簡易表示：詳細表示

［簡易表示］ボタンまたは［詳細表示］ボタンをクリックすると、簡易表示と詳細表示が切り替わります。

ファイル出力

生産情報をファイルに保存することができます。

1. 生産情報ダイアログボックスの［ファイル出力］ボタンをクリックします。［ファイルを保存］ダイアログボックスが表示されます。

2. 任意のファイル名を付けて、［保存］ボタンをクリックします。拡張子.csvのCSV形式（カンマ区切り）のファイルが保存されます。

## 8-8 -1 CPUユニット生産情報の表示

NX/NYシリーズコントローラ

CPUユニットや産業用PCの生産情報、およびXバスユニットやNXユニットが装着されている場合は各ユニットの生産情報が表示されます。また、形NX1P2 CPUユニットの場合は、オプションボードの生産情報も表示されます。

- 表示方法

1. マルチビューエクスプローラで［CPU・増設ラック］–［CPUラック］を右クリックして［生産情報表示］を選択します。生産情報ダイアログボックスが表示されます。

- 表示項目

表示項目は以下のとおりです。

| CPUユニット／産業用PC | Xバスユニット*1／NXユニット*2 | オプションボード*3 |
|---|---|---|
| / 形式、ユニットバージョン / LOT番号 / シリアル番号 / ハードウェアリビジョン / ソフトウェアバージョン / 累積通電時間*4 | / 形式、ユニットバージョン / LOT番号 / シリアル番号 / ハードウェアバージョン / ソフトウェアバージョン / 累積通電時間 | / 形式、ユニットバージョン / LOT番号 / シリアル番号 / ハードウェアバージョン / ソフトウェアバージョン |

*1.形NX502 CPUユニットにXバスユニットを装着している場合だけ表示されます。*2.形NX502、形NX102、形NX1P2 CPUユニットにNXユニットを装着している場合だけ表示されます。*3.形NX1P2 CPUユニットにオプションボードを装着している場合だけ表示されます。*4.形NX502、形NX102、形NX1P2 CPUユニットの場合だけ表示されます。

表示項目の内容の詳細は、各ユニットのマニュアルを参照してください。

NJシリーズコントローラ

CPUユニットおよびCJ高機能ユニットの生産情報が表示されます。

- 表示方法

1. マルチビューエクスプローラで［構成・設定］–［CPU・増設ラック］をダブルクリックします。または、［構成・設定］–［CPU・増設ラック］を右クリックして［編集］を選択します。エディットウィンドウにユニットエディタが表示されます。

2. ユニットエディタでユニットのない場所を右クリックして［生産情報表示］を選択します。生産情報ダイアログボックスが表示されます。

- 表示項目

表示内容は以下のとおりです。

| CPUユニット | CJ高機能ユニット |
|---|---|
| / ユニット形式、ユニットバージョン / LOT番号 / ハードウェアバージョン/ハードウェアリビジョン*1 / ソフトウェアバージョン | / ラック番号、スロット番号、号機No.／ユニット番号 / ユニット形式、ユニットバージョン / LOT番号 / ユニットレビジョン / PCBレビジョン / ソフトウェアレビジョン |

*1.ハードウェアリビジョンが「なし」のユニットでは、ハードウェアリビジョンの表示はありません。

表示項目の内容の詳細は、各ユニットのマニュアルを参照してください。

## 8-8 -2 EtherCATスレーブ生産情報の表示

EtherCATスレーブのスレーブに設定されているノードアドレス、形式*1、シリアル番号が表示されます。*1. 形式を特定できない場合（ESIファイルがない場合など）は、ベンダーID、プロダクトコード、リビジョンナンバーが表示されます。

- 表示方法

1. マルチビューエクスプローラで［構成・設定］–［EtherCAT］をダブルクリックします。または、［構成・設定］–［EtherCAT］を右クリックして［編集］を選択します。エディットウィンドウにEtherCAT構成編集画面が表示されます。

2. EtherCAT構成編集画面でマスタを右クリックして［生産情報表示］を選択します。生産情報ダイアログボックスが表示されます。

- 表示項目

- 表示項目は以下のとおりです。

- 形式、リビジョン

- シリアル番号

- ハードウェアバージョン

- ソフトウェアバージョン表示項目の内容の詳細は、各スレーブのマニュアルを参照してください。

## 8-8 -3 EtherCATスレーブターミナル構成ユニットの生産情報の表示

EtherCATスレーブターミナルを構成するユニットの生産情報が表示されます。

- 表示方法

1. マルチビューエクスプローラで［EtherCAT］上の対象のEtherCATカプラユニットをダブルクリックします。または、［EtherCAT］上の対象のEtherCATカプラユニット右クリックして［編集］を選択します。

エディットウィンドウにスレーブターミナル構成編集画面が表示されます。

2. スレーブターミナル構成編集画面上で、右クリックして［生産情報表示］を選択します。生産情報ダイアログボックスが表示されます。

- 表示項目

- 表示項目は以下のとおりです。

- 形式、ユニットバージョン

- LOT番号

- シリアル番号

- ハードウェアバージョン

- ソフトウェアバージョン

- 累積通電時間表示項目の内容の詳細は、各ユニットのマニュアルを参照してください。
