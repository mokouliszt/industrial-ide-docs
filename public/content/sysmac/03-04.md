右クリックメニューから、［すべて展開］または［すべて折りたたみ］を選択して、部品の表示を展開または省略することができます。

検索／置換ウィンドウは、マルチビューエクスプローラのプログラミング以下のデータを対象として、任意の文字列により検索・置換することができます。

## 3-4-5 ウィンドウ表示位置の変更

Sysmac Studio のメインウィンドウを構成する各ウィンドウを任意の位置に配置したり、ウィンドウの表示／非表示を切り替えたりすることができます。

ウィンドウのドッキング／アンドッキングエディットウィンドウやウォッチウィンドウなどのウィンドウを、Sysmac Studio のメインウィンドウから切り離して任意の位置に配置したり、複数のウィンドウをまとめてタブグループ表示にしたりすることができます。

メインウィンドウから切り離す操作をアンドッキング、再びメインウィンドウ内に配置する操作をドッキングと呼びます。また、メインウィンドウから切り離した状態のウィンドウをフローティングウィンドウ、メインウィンドウ内に配置されたウィンドウをドッキングウィンドウと呼びます。

- ウィンドウをアンドッキングする

1. 対象のウィンドウのタイトルバー、または対象のタブをクリックし、そのままドラッグします。ドッキングガイドが表示されます。

![図(元マニュアル p.114)](images/sysmac/03-04-19.webp)

2. ドッキングガイド以外の任意の位置でマウスのボタンを離します。対象のウィンドウがフローティング状態になります。

![図(元マニュアル p.115)](images/sysmac/03-04-20.webp)

- ウィンドウをドッキングする

1. 対象のウィンドウのタイトルバー、または対象のタブをクリックし、そのままドラッグします。ウィンドウを配置可能な場所にドッキングガイドが表示されます。表示されるドッキングガイドのアイコンはマウスカーソルの位置によって異なります。アイコンには次の種類があります。

| アイコン | 説明 |
|---|---|
| | ドラッグしたウィンドウを、アイコンが表示されているウィンドウの上下左右に並べて配置します。 |
| | ドラッグしたウィンドウを、アイコンが表示されているウィンドウに重ねて、タブグループ化します。 |

ドッキングガイドのアイコンにマウスカーソルを合わせると、ウィンドウがドッキングされる位置に青色の網掛け領域が表示されます。

![図(元マニュアル p.115)](images/sysmac/03-04-21.webp)

2. ドッキングガイドの任意のアイコンにマウスポインタを合わせてマウスのボタンを離します。対象のウィンドウが青色の網掛けで表示されたドッキング位置に配置されます。

![図(元マニュアル p.116)](images/sysmac/03-04-22.webp)

ウィンドウはドッキング可能な場所に応じて、次の2 種類のグループに分けられます。

| | ウォッチウィンドウ | |
| | コントローラステータスウィンドウ | |
| | シミュレーションウィンドウ | |
| | アウトプットウィンドウ | |
| | ビルドウィンドウ | |
| | クロスリファレンスウィンドウ | |
| | 検索／置換結果ウィンドウ | |
| | 微分モニタウィンドウ | |

本項では、エディットウィンドウ以外のウィンドウのグループを総称してツールウィンドウと呼びます。同じグループのウィンドウ同士はタブグループ化することができます。

## 3-4-6 ウィンドウレイアウトの保存／復元

Sysmac Studio のメインウィンドウを構成する各ウィンドウの表示位置（以降、ウィンドウレイアウトと呼びます）をパソコンに保存し、その情報をもとにウィンドウレイアウトを復元することができます。ウィンドウレイアウトの保存／復元には、次の機能があります。

| 機能 | 説明 |
|---|---|
| ウィンドウレイアウトの自動保存と復元 | プロジェクトを閉じるときにウィンドウレイアウトを自動的にパソコンに保存します。プロジェクトを開くときに、保存したウィンドウレイアウトを自動的に復元します。 |
| ウィンドウレイアウトテンプレートの管理 | 表示中のウィンドウレイアウトをテンプレートとして保存します。保存したテンプレートは一覧表示されます。一覧から任意のテンプレートを選択することにより、表示中のウィンドウのレイアウトを変更することができます。 |

- ウィンドウレイアウトの保存／復元は、以下を除くすべてのウィンドウが対象です。

- ツールバー

- フィルタウィンドウ

- エディットウィンドウウィンドウの詳細については、「3-4-1 アプリケーションウィンドウ」を参照してください。
