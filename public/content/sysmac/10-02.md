# 10-2 周辺ツールの起動と終了

Sysmac Studio 周辺ツールの起動方法と終了方法について説明します。

使用上の注意

- Sysmac Studio および周辺ツールを起動する前に、以下の点にご注意ください。

- Sysmac Studio の使用に不要なアプリケーションは終了してください。ウィルスチェックソフトなどによりSysmac Studio の起動や動作に影響が出る場合は、Sysmac Studio をチェック対象から除外するなどの対策を実施してください。

- パソコンのハードディスクやプリンタをネットワークで他のパソコンに共有させている時は、共有を解除してください。

- ノートパソコンで、初期設定がUSB ポートをモデムや赤外線に割当てているものがあります。パソコンの説明書にしたがってUSB ポートを通常のシリアルポートとして使用できるように設定を変更してください。

- ノートパソコンで、初期設定が省電力のためにUSB およびEthernet ポートに規定の電力を供給しない設定になっているものがあります。省電力設定はWindows の設定のほか、パソコン独自のユーティリティやBIOS で設定されている場合があります。パソコンの説明書にしたがって、すべての省電力機能を停止させてください。

## 10-2-1 Sysmac Studio 周辺ツールの起動

Sysmac Studio 周辺ツールの起動手順を以下に示します。

1. タスクバー上の［スタート］ボタンをクリックして、［プログラム］–［OMRON］–［Sysmac Studio］を選択します。各周辺ツールのリストが表示されます。

2. 起動する周辺ツールを選択します。（例：CX-Integrator）

![図(元マニュアル p.846)](images/sysmac/10-02-01.webp)

周辺ツールのメインウィンドウが表示されます。

## 10-2-2 Sysmac Studio 周辺ツールの終了

Sysmac Studio 周辺ツールの終了手順を以下に示します。

1. ［ファイル］メニューから［終了］を選択します。

![図(元マニュアル p.847)](images/sysmac/10-02-02.webp)

Sysmac Studio 周辺ツールが終了します。

参考

［終了］を選択したとき、編集中のデータが保存されていない場合は、Sysmac Studio 周辺ツール が終了する前に、データの保存をうながすダイアログボックスが表示されますので、必要なデータは保存してください。詳細は、各マニュアルを参照してください。
