# 8-5 印刷

ここでは、印刷機能について説明します。

## 8-5-1 印刷対象

以下の項目を印刷することができます。印刷対象は任意に選択することができます。• EtherCAT ネットワーク構成情報とユニット動作設定

- 高機能ユニットパラメータ設定*1

- CPU・増設ラック*2

- I/O マップ

- コントローラ設定

- モーション制御設定（軸設定）

- カムデータ設定

- イベント設定

- タスク設定

- データトレース設定

- ラダー図エディタ（Program、Function、FunctionBlock）

- ST エディタ（Program、Function、FunctionBlock）

- ローカル変数テーブル（Program、Function、FunctionBlock）

- ユーザ定義データ型エディタ

- グローバル変数テーブル*1. NJ シリーズCPU ユニットの場合だけ表示されます。*2. NJ シリーズCPU ユニット、および形NX502、形NX102、形NX1P2 CPU ユニットの場合に表示されます。

参考

- トラブルシュート機能の印刷については、「8-16 トラブルシュート」を参照してください。

- オフライン照合結果の印刷については、「4-11-2 オフライン照合結果の印刷」を参照してください。

- コントローラ以外のデバイス固有の印刷機能については、それぞれのマニュアルを参照してください。

## 8-5-2 印刷設定

表紙および目次の印刷の有無、印刷時の余白、フォントなどの設定を行うことができます。

以下の手順で印刷設定を行います。

1. ［ファイル］–［ページ設定］を選択します。印刷設定画面が表示されます。

![図(元マニュアル p.708)](images/sysmac/08-05-01.webp)

2. 画面左側のボタンで、設定項目を選択します。

| 項目名 | 内容 |
|---|---|
| 全般ボタン | 表紙および目次を印刷するかどうかを設定します。 |
| 余白ボタン | 余白と、ヘッダー／フッターの余白を設定します。 |
| ヘッダーボタン | ヘッダーの記載内容を設定します。 |
| フッターボタン | フッターの記載内容を設定します。 |
| フォントボタン | 印刷で使用するフォントを選択します。 |

## 3各設定を行います。

印刷設定項目

印刷設定の項目について説明します。

- 全般

設定項目：説明

- 表紙を印刷：表紙を印刷するかどうかを設定します。表紙には次の項目が印刷されます。

- プロジェクト名

- プロジェクトのプロパティの各項目

- 現在のSysmac Studio のバージョン目次を印刷：目次を印刷するかどうかを設定します。目次には、印刷対象の見出しとページ番号が印刷されます。

- 余白

| 設定項目 | 説明 |
|---|---|
| 余白 | 印刷用紙の上下左右の余白を設定します。 |
| ヘッダー／フッター余白 | 用紙の上端からヘッダーまでの余白、および用紙の下端からフッターまでの余白を設定します。 |

- ヘッダー／フッターヘッダーおよびフッターに印刷する文字を設定します。それぞれ、用紙の左側、中央、右側に印刷する文字を指定することができます。

ヘッダー、フッターには以下のフィールドコードを使用することができます。

| 内容 | フィールドコード | 印刷例 | 備考 |
|---|---|---|---|
| 日付 | &d | 2015/02/24 | 印刷時の日付が印刷されます。 |
| 時間 | &t | 16:38 | 印刷時の時刻が印刷されます。 |
| ページ番号 | &# | 1 | - |
| ページ総数 | &o | 4 | - |
| 印刷対象項目 | &l | I/O マップ | - |
| プロジェクト名 | &j | 新規プロジェクト | - |
| コントローラ名 | &v | new_Controller_0 | - |
| コントローラ機種名 | &y | NJ501-1500(1.09) | コントローラの形式およびユニットバージョンが印刷されます。 |
| POU 名 | &p | Program0 | POU が印刷されたページだけに印刷されます。 |
| セクション名 | &s | Section1 | セクションが印刷されたページだけに印刷されます。 |

- フォント

［フォント設定］ボタンをクリックして表示されるフォント設定ダイアログボックスで、印刷に使用するフォントを設定します。フォントの種類、スタイル、修飾、サイズを設定することができます。
