# 8-4 ユーザメモリの使用量モニタ

- ユーザメモリの使用量モニタとは

Sysmac Studio で編集中のユーザプログラムの、コントローラのメモリサイズに対する使用量の目安を示します。ユーザプログラムの大きさがメモリサイズをオーバしている場合、コントローラへ転送すると、運転時にコントローラが異常となります。

使用上の注意

メモリ使用状況に表示される値は概略値です。現在のメモリ使用量の目安として参考にしてください。ユーザプログラムの大きさに関する制限については、『NJ/NX シリーズ CPU ユニット ユーザーズマニュアル ソフトウェア編（SBCA-467）』または『NY シリーズ 産業用パネル型PC／産業用ボックス型PC ユーザーズマニュアル ソフトウェア編（SBCA-436）』を参照してください。

メモリ使用状況の表示

1. ［プロジェクト］–［メモリ使用状況］を選択します。サイドタブ［総量表示］にメモリ使用状況の概要が表示されます。

![図(元マニュアル p.705)](images/sysmac/08-04-01.webp)

［総量表示］画面では以下の情報が表示されます。

| 項目 | 表示内容 |
|---|---|
| データ型定義数 | 使用するデータ型の数を表示します。 |
| 非保持変数定義数 | 使用する非保持型の変数の数を表示します。 |
| 非保持変数メモリ | 非保持型の変数で使用するメモリの使用量を表示します。 |
| 保持変数定義数 （アドレス指定なし） | アドレス指定なしで使用する保持型の変数の数を表示します。 |
| 保持変数メモリ | 保持型の変数で使用するメモリの使用量を表示します。 |
| POU インスタンス数 | 使用するPOU のインスタンス数を表示します。 |
| プログラムメモリ | プログラムで使用するメモリの使用量を表示します。 |

実行モジュールファイル数：実行するモジュールのファイル数を表示します。ファイルはPOU およびセクションなどに対して作成され、表示はPOU とセクションの数に応じて増減します。

2. サイドタブ［詳細表示］を選択します。タスクとグローバル変数のメモリ使用状況の詳細が表示されます。各タスクのメモリ使用状況は、プログラムPOU ごとに表示されます。

![図(元マニュアル p.706)](images/sysmac/08-04-02.webp)

［詳細表示］画面の［タスク詳細］には、以下の情報がタスクごと、プログラムPOU ごとに表示されます。

非保持変数定義数：使用する非保持型の変数の数を表示します。

非保持変数メモリ：非保持型の変数で使用するコントローラのメモリの使用量を表示します。

保持変数定義数（アドレス指定なし）：アドレス指定なしで使用する保持型の変数の数を表示します。

保持変数メモリ：保持型の変数で使用するコントローラのメモリの使用量を表示します。

変数メモリ（外部変数を含む）：外部変数を含むプログラムPOU で定義された変数で使用するメモリの使用量を表示します。

［詳細表示］画面の［グローバル変数］には、以下の情報が表示されます。

非保持変数定義数：使用する非保持型の変数の数を表示します。

非保持変数メモリ：非保持型の変数で使用するコントローラのメモリの使用量を表示します。

保持変数定義数（アドレス指定なし）：アドレス指定なしで使用する保持型の変数の数を表示します。

保持変数メモリ：保持型の変数で使用するコントローラのメモリの使用量を表示します。

参考

形NX-CSG 通信コントロールユニットの場合、サイドタブ［詳細表示］は表示されません。
