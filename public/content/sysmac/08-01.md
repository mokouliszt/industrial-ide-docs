# 8-1 SD メモリカード関連機能

- SD メモリカード機能とは

NJ/NX シリーズコントローラに装着したSD メモリカードのファイル操作、およびSD メモリカードとパソコン間のファイル転送（コピー）を実行する機能です。NY シリーズコントローラではSD メモリカードの代わりに仮想SD メモリカードが対象となります。以降の説明でのSD メモリカードは、NY シリーズコントローラにおいては仮想SD メモリカードに置き換えてお読みください。

## 8-1-1 SD メモリカードウィンドウの操作

1. オンライン時にメインメニューから［コントローラ］–［SD メモリカード］を選択します。SDメモリカードウィンドウが起動します。

SD メモリカードウィンドウの各部の概要

- ツールバー

ツールバーのボタン（またはショートカットキー）を使用して、SD メモリカードのファイル操作を実行します。

| 操作 （ショートカット） | 機能 |
|---|---|
| 戻る（alt+ ←） | 前回表示のフォルダに戻る |
| 進む（alt+ →） | 前回表示のフォルダに進む |
| 上へ（BackSpace） | 上のフォルダに進む |
| 更新（F5） | 最新の情報に更新する |
| プロパティ（Alt+Enter） | 選択したファイル／フォルダのプロパティを表示し、属性 を変更する*1（選択なし、複数選択時は操作できません） |
| SD メモリカードの初期化*2 （Ctrl+Alt+A） | SD メモリカードをフォーマットする |
| 削除（del） | 選択したファイル／フォルダを削除する |
| コピー（Ctrl+C） | 選択したファイル／フォルダをコピーする |
| 貼り付け（Ctrl+V） | コピーした内容を貼り付ける |

| 操作 （ショートカット） | 機能 |
|---|---|
| 新規作成（Ctrl+Alt+F） | カレント位置に新規フォルダを作成する |
| 名前の変更（F2） | 選択したファイル／フォルダの名前を変更する（選択なし、 複数選択時は操作できません） |
| SD メモリカードバックアップ （Alt +Shift+B） *3 | コントローラのデータをSD メモリカードにバックアップ する |
| SD メモリカードバックアップファ イル照合 （Alt+Shift+C）*3 | SD メモリカード内のバックアップファイルのデータとコ ントローラのデータを照合する |
| 終了（Alt+F4） | SD メモリカード操作を終了する |

*1.NY シリーズコントローラでは属性変更はできません。*2.NY シリーズコントローラでは使用できません。ボタンはグレーアウトされます。*3.SD メモリカードバックアップ機能の手順については、「8-10-2 SD メモリカードバックアップ機能」を参照してください。

- アドレスバー

SD メモリカード内のカレントフォルダの表示、およびカレントフォルダの指定が行えます。

- カレントフォルダ表示エリア

SD メモリカード内のカレントフォルダ内のファイル一覧が表示されます。表示内容は以下のとおりです。タイトルをクリックすることで、その項目で並び変えすることが可能です。

| タイトル | 内容 |
|---|---|
| 名前 | ファイル名／フォルダ名（ロングファイル名） |
| サイズ | 実ファイルサイズ（Kbyte） |
| 種類 | パソコンに登録されているファイルの種類 |
| 更新日時 | ファイルの更新日時 |
| 属性 | R：読み取り専用 |

- ステータスバー

SD メモリカード内のカレントフォルダ内のファイル情報を表示します。表示内容は、以下のとおりです。

| 項目 | 内容 |
|---|---|
| ステータス | ファイル一覧内のオブジェクト数（ファイル／フォルダ数） （注）SD メモリカード未装着（給電なし）の場合、またはSD メモリカードが未フォ ーマットの場合、［SD メモリカードが装着されていないか、フォーマットされ ていません］と表示されます。 |
| ボリュームラベル | SD メモリカードのボリュームラベル*1 |
| サイズ（空き容量） | SD メモリカード内の空き容量サイズ（Kbyte） |

*1.NY シリーズコントローラでは、ボリュームラベルは表示されません。
