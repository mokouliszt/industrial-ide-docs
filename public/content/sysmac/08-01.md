# 8-1 SD メモリカード関連機能

- SDメモリカード機能とは

NJ/NXシリーズコントローラに装着したSDメモリカードのファイル操作、およびSDメモリカードとパソコン間のファイル転送（コピー）を実行する機能です。NYシリーズコントローラではSDメモリカードの代わりに仮想SDメモリカードが対象となります。以降の説明でのSDメモリカードは、NYシリーズコントローラにおいては仮想SDメモリカードに置き換えてお読みください。

## 8-1 -1 SDメモリカードウィンドウの操作

1. オンライン時にメインメニューから［コントローラ］–［SDメモリカード］を選択します。SDメモリカードウィンドウが起動します。

SDメモリカードウィンドウの各部の概要

- ツールバー

ツールバーのボタン（またはショートカットキー）を使用して、SDメモリカードのファイル操作を実行します。

| 操作 （ショートカット） | 機能 |
|---|---|
| 戻る（alt+ ←） | 前回表示のフォルダに戻る |
| 進む（alt+ →） | 前回表示のフォルダに進む |
| 上へ（BackSpace） | 上のフォルダに進む |
| 更新（F5） | 最新の情報に更新する |
| プロパティ（Alt+Enter） | 選択したファイル／フォルダのプロパティを表示し、属性を変更する*1（選択なし、複数選択時は操作できません） |
| SDメモリカードの初期化*2 （Ctrl+Alt+A） | SDメモリカードをフォーマットする |
| 削除（del） | 選択したファイル／フォルダを削除する |
| コピー（Ctrl+C） | 選択したファイル／フォルダをコピーする |
| 貼り付け（Ctrl+V） | コピーした内容を貼り付ける |

| 操作 （ショートカット） | 機能 |
|---|---|
| 新規作成（Ctrl+Alt+F） | カレント位置に新規フォルダを作成する |
| 名前の変更（F2） | 選択したファイル／フォルダの名前を変更する（選択なし、 複数選択時は操作できません） |
| SDメモリカードバックアップ （Alt +Shift+B） *3 | コントローラのデータをSDメモリカードにバックアップする |
| SDメモリカードバックアップファイル照合 （Alt+Shift+C）*3 | SDメモリカード内のバックアップファイルのデータとコントローラのデータを照合する |
| 終了（Alt+F4） | SDメモリカード操作を終了する |

*1.NYシリーズコントローラでは属性変更はできません。*2.NYシリーズコントローラでは使用できません。ボタンはグレーアウトされます。*3.SDメモリカードバックアップ機能の手順については、「8-10-2 SDメモリカードバックアップ機能」を参照してください。

- アドレスバー

SDメモリカード内のカレントフォルダの表示、およびカレントフォルダの指定が行えます。

- カレントフォルダ表示エリア

SDメモリカード内のカレントフォルダ内のファイル一覧が表示されます。表示内容は以下のとおりです。タイトルをクリックすることで、その項目で並び変えすることが可能です。

| タイトル | 内容 |
|---|---|
| 名前 | ファイル名／フォルダ名（ロングファイル名） |
| サイズ | 実ファイルサイズ（Kbyte） |
| 種類 | パソコンに登録されているファイルの種類 |
| 更新日時 | ファイルの更新日時 |
| 属性 | R：読み取り専用 |

- ステータスバー

SDメモリカード内のカレントフォルダ内のファイル情報を表示します。表示内容は、以下のとおりです。

| 項目 | 内容 |
|---|---|
| ステータス | ファイル一覧内のオブジェクト数（ファイル／フォルダ数） （注）SDメモリカード未装着（給電なし）の場合、またはSDメモリカードが未フォーマットの場合、［SDメモリカードが装着されていないか、フォーマットされていません］と表示されます。 |
| ボリュームラベル | SDメモリカードのボリュームラベル*1 |
| サイズ（空き容量） | SDメモリカード内の空き容量サイズ（Kbyte） |

*1.NYシリーズコントローラでは、ボリュームラベルは表示されません。

SDメモリカード機能の操作

- SDメモリカードの初期化

1. フォーマットボタンをクリックします。

確認のためのダイアログボックスが表示されます。

![図(元マニュアル p.668)](images/sysmac/08-01-02.webp)

2. ［開始］ボタンをクリックします。

- プロパティの表示

1. ファイル／フォルダを選択します。

2. プロパティボタンをクリックします。

選択したファイル／フォルダのプロパティが表示されます。

- SDメモリカード内でのファイル／フォルダのコピー

1. ファイル／フォルダを選択し、コピーボタンをクリックします。

2. コピーする場所を指定したあと、貼り付けボタンをクリックします。

選択したファイル／フォルダがコピーされます。

**使用上の注意**

- マルチバイト文字（日本語など）は、パソコン上で使用できますが、CPUユニットでは使用できません。

- ルートディレクトリに作成できる最大ファイル数は、511ファイルです。MS-DOS 8.3形式の場合、ロングファイルネームの場合は上記より少なくなります。
