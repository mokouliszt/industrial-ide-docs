# 8-18 プログラムハッシュコードの出力

プログラムハッシュコードをCSV ファイルへ出力する機能です。プログラムハッシュコードを比較することで、ユーザプログラムの変更を検知することができます。出力されるプログラムハッシュコードはNJ/NX シリーズコントローラのプログラムハッシュコード取得命令(GetPrgHashCode)と同じコードです。

プログラムハッシュコード取得命令の詳細は、『NJ/NX シリーズ コマンドリファレンスマニュアル 基本編（SBCA-468）』を参照してください。

- プログラムハッシュコードの出力手順

1. ［ツール］メニューから［プログラムハッシュコードの出力］を選択します。

![図(元マニュアル p.792)](images/sysmac/08-18-01.webp)

［プログラムハッシュコードの出力］ダイアログボックスが表示されます。

2. ファイル名を指定して、［保存］ボタンをクリックします。CSV ファイルが保存されます。

- プログラムハッシュコードのファイルフォーマット本機能で出力されるプログラムハッシュコードのファイルフォーマットは以下のとおりです。ファイル形式: CSV（カンマ区切り）

データフォーマット:

![図(元マニュアル p.792)](images/sysmac/08-18-02.webp)

各項目の説明は次のとおりです。

| 項目 | 内容 | | | |
|---|---|---|---|---|
| Created | ファイル出力日時 | | | |
| Device Name | コントローラ名称 | | | |
| Model Name | コントローラの形式 | | | |
| Project Unit Version | コントローラの設定ユニットバージョン | | | |
| Program Hash Code | Code1 | Code2 | Code3 | Code4 |

Code1～Code4 は、プログラムハッシュコード取得命令(GetPrgHashCode)の出力変数（Code1～Code4）と同じ値になります。

使用上の注意

- プログラムハッシュコードは、プログラムの内容、コントローラの形式、および設定ユニットバージョンによって変化することがあります。

- プログラムハッシュコードを使用してプログラムの変更検知を行っている場合、設定ユニットバージョンを変更したときにプログラムハッシュコードを出力して値を確認してください。

- プログラムハッシュコードが意図せず変化している場合、同期機能や詳細照合機能を使って変更内容を確認してください。
