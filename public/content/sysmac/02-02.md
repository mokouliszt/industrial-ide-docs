# 2-2 Sysmac Studioのアンインストール

## 2-2 -1 Sysmac Studioをアンインストールする

既にインストールされているSysmac Studioをアンインストールする場合、以下の順を実行します。(1) Sysmac Studioのアンインストール(2) Communications Middlewareのアンインストール

(3) CX-Serverドライバ管理ツール*1、CX-Server*1のアンインストール*1. 周辺ツールをインストールした場合およびCX-OneをインストールしているパソコンにSysmac Studioをインストールした場合に表示されます。CX-Oneをインストールしている場合は、「CX-Oneセットアップマニュアル」を参照してアンインストールしてください。

**使用上の注意**

アドミニストレータかアドミニストレータ権限のあるユーザでWindowsにログインして、アンインストールしてください。権限がなくアクセスできないフォルダがあると、正しくアンインストールできないことがあります。

Sysmac Studioのアンインストール

1. ［スタート］メニューから、［コントロールパネル］–［プログラムと機能］を選択します。［プログラムのアンインストールまたは変更］ダイアログボックスが表示されます。

2. 「Sysmac Studio」を選択して、［アンインストール］ボタンをクリックします。

![図(元マニュアル p.66)](images/sysmac/02-02-01.webp)

Sysmac Studioの完全削除を確認するダイアログボックスが表示されます。

3. ［はい］ボタンをクリックします。

![図(元マニュアル p.67)](images/sysmac/02-02-02.webp)

Sysmac Studioのアンインストールが実行され、CX Common Toolsの完全削除を確認するダイアログボックスが表示されます。

4. CX Common Toolsの完全削除を継続するときは、［はい］ボタンをクリックします。CX Common Toolsの完全削除が実行されます。

![図(元マニュアル p.67)](images/sysmac/02-02-03.webp)

（注）必ず［はい］ボタンをクリックして完全に削除してください。

Sysmac Studioのアンインストールの完了を示すダイアログボックスが表示されます。

5. ［完了］ボタンをクリックします。

![図(元マニュアル p.67)](images/sysmac/02-02-04.webp)

Sysmac Studioのアンインストールが完了します。

Communications Middlewareのアンインストール

1. ［スタート］メニューから、［コントロールパネル］–［プログラムと機能］を選択します。［プログラムのアンインストールまたは変更］ダイアログボックスが表示されます。

2. 「Communications Middleware」を選択して［アンインストール］ボタンをクリックします。次のダイアログボックスが表示されます。

3. ［はい］ボタンをクリックします。

![図(元マニュアル p.68)](images/sysmac/02-02-05.webp)

Communications Middlewareを削除するアンインストーラが起動します。

4. ダイアログボックスのメッセージにしたがって、Communications Middlewareを削除してください。［メンテナンスの完了］ダイアログボックスが表示され、アンインストールが完了します。

![図(元マニュアル p.68)](images/sysmac/02-02-06.webp)

CX-Serverドライバ管理ツール、CX-Serverのアンインストール

1. ［スタート］メニューから、［コントロールパネル］–［プログラムと機能］を選択します。［プログラムのアンインストールまたは変更］ダイアログボックスが表示されます。

2. 「CX-Serverドライバ管理ツール」を選択して［アンインストールと変更］ボタンをクリックします。次のダイアログボックスが表示されます。

3. ［はい］ボタンをクリックします。

![図(元マニュアル p.68)](images/sysmac/02-02-07.webp)

アンインストール確認ダイアログボックスが表示されます。

4. ［はい］ボタンをクリックします。

![図(元マニュアル p.69)](images/sysmac/02-02-08.webp)

5. ダイアログボックスのメッセージにしたがって、CX-Serverドライバ管理ツールを削除してください。［アンインストール完了］ダイアログボックスが表示され、CX-Serverドライバ管理ツールのアンインストールが完了します。

![図(元マニュアル p.69)](images/sysmac/02-02-09.webp)

6. 続けて、「CX-Server」を選択して同様にアンインストールを行います。
