# 2-2 Sysmac Studioのアンインストール

## 2-2-1 Sysmac Studio をアンインストールする

既にインストールされているSysmac Studio をアンインストールする場合、以下の順を実行します。(1) Sysmac Studio のアンインストール(2) Communications Middleware のアンインストール

(3) CX-Server ドライバ管理ツール*1、CX-Server*1 のアンインストール*1. 周辺ツールをインストールした場合およびCX-One をインストールしているパソコンにSysmac Studio をインストールした場合に表示されます。CX-One をインストールしている場合は、「CX-One セットアップマニュアル」を参照してアンインストールしてください。

使用上の注意

アドミニストレータかアドミニストレータ権限のあるユーザでWindows にログインして、アンインストールしてください。権限がなくアクセスできないフォルダがあると、正しくアンインストールできないことがあります。

Sysmac Studio のアンインストール

1. ［スタート］メニューから、［コントロールパネル］–［プログラムと機能］を選択します。［プログラムのアンインストールまたは変更］ダイアログボックスが表示されます。2 「Sysmac Studio」を選択して、［アンインストール］ボタンをクリックします。

![図(元マニュアル p.66)](images/sysmac/02-02-01.webp)

Sysmac Studio の完全削除を確認するダイアログボックスが表示されます。3 ［はい］ボタンをクリックします。

![図(元マニュアル p.67)](images/sysmac/02-02-02.webp)

Sysmac Studio のアンインストールが実行され、CX Common Tools の完全削除を確認するダイアログボックスが表示されます。4 CX Common Tools の完全削除を継続するときは、［はい］ボタンをクリックします。CX Common Tools の完全削除が実行されます。

![図(元マニュアル p.67)](images/sysmac/02-02-03.webp)

（注）必ず［はい］ボタンをクリックして完全に削除してください。

Sysmac Studio のアンインストールの完了を示すダイアログボックスが表示されます。5 ［完了］ボタンをクリックします。

![図(元マニュアル p.67)](images/sysmac/02-02-04.webp)

Sysmac Studio のアンインストールが完了します。

Communications Middleware のアンインストール

1. ［スタート］メニューから、［コントロールパネル］–［プログラムと機能］を選択します。［プログラムのアンインストールまたは変更］ダイアログボックスが表示されます。2 「Communications Middleware」を選択して［アンインストール］ボタンをクリックします。次のダイアログボックスが表示されます。3 ［はい］ボタンをクリックします。

![図(元マニュアル p.68)](images/sysmac/02-02-05.webp)

Communications Middleware を削除するアンインストーラが起動します。4ダイアログボックスのメッセージにしたがって、Communications Middleware を削除してください。［メンテナンスの完了］ダイアログボックスが表示され、アンインストールが完了します。

![図(元マニュアル p.68)](images/sysmac/02-02-06.webp)

CX-Server ドライバ管理ツール、CX-Server のアンインストール

1. ［スタート］メニューから、［コントロールパネル］–［プログラムと機能］を選択します。［プログラムのアンインストールまたは変更］ダイアログボックスが表示されます。2 「CX-Server ドライバ管理ツール」を選択して［アンインストールと変更］ボタンをクリックします。次のダイアログボックスが表示されます。3 ［はい］ボタンをクリックします。

![図(元マニュアル p.68)](images/sysmac/02-02-07.webp)

アンインストール確認ダイアログボックスが表示されます。4 ［はい］ボタンをクリックします。

![図(元マニュアル p.69)](images/sysmac/02-02-08.webp)

5. ダイアログボックスのメッセージにしたがって、CX-Server ドライバ管理ツールを削除してください。［アンインストール完了］ダイアログボックスが表示され、CX-Server ドライバ管理ツールのアンインストールが完了します。

![図(元マニュアル p.69)](images/sysmac/02-02-09.webp)

6. 続けて、「CX-Server」を選択して同様にアンインストールを行います。
