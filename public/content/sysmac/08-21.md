# 8-21 IEC 61131-10 XMLファイルのインポート

IEC 61131-10 で定義されたXML ファイルをインポートする機能です。このXML ファイルは、IEC 61131-3 で定義されたデータ型、グローバル変数、およびラダー図またはST 言語ベースのPOU プログラミング関連のデータを含みます。Sysmac Studio 以外のソフトウェアで作成したプログラミングデータをSysmac Studio に取り込むことができます。

XML の記載例は、デフォルトでは下記のフォルダにインストールされています。本機能がサポートするデータを確認する場合は、同フォルダのXML スキーマを参照してください。

- Sysmac Studio 32 ビット版の場合

32 ビット版OS の場合: C:\Program Files\OMRON\Sysmac Studio\Sample\IEC 61131-10 XML\Controller 64 ビット版OS の場合: C:\Program Files (x86)\OMRON\Sysmac Studio\Sample\IEC 61131-10 XML\Controller

- Sysmac Studio 64 ビット版の場合

C:\Program Files\OMRON\Sysmac Studio\Sample\IEC 61131-10 XML\Controller

バージョン関連情報本機能は、形NX102 CPU ユニット Ver.1.31 以降、形NX502 CPU ユニット Ver.1.60、およびNJ/NX/NY シリーズ CPU ユニット Ver.1.19 以降のプロジェクトで使用できます。

## 8-21-1 インポートの実行

- インポート対象プログラム、ファンクション、ファンクションブロック、データ型、グローバル変数

- インポート手順

1. ［ツール］メニューから［IEC 61131-10 XML］–［インポート］を選択します。IEC 61131-10 XML インポート画面が表示されます。

![図(元マニュアル p.796)](images/sysmac/08-21-01.webp)

2. ファイル選択ダイアログボックス表示ボタンをクリックします。

![図(元マニュアル p.797)](images/sysmac/08-21-02.webp)

ファイル選択ダイアログボックスが表示されます。

3. ファイル選択ダイアログボックスで、インポート対象のXML ファイルを選択し、［開く］ボタンをクリックします。IEC 61131-10 XML インポート画面に戻り、インポートファイルにファイル選択ダイアログボックスで選択したファイルパスが表示されます。

4. ［実行］ボタンをクリックします。インポートが実行され、プロジェクト内にXML ファイルのプログラム、ファンクション、ファンクションブロック、データ型、グローバル変数が追加されます。すでにプロジェクト内に、同名のデータ型、グローバル変数、プログラム、ファンクション、ファンクションブロックがあり、それらの内容が異なる場合は、［インポート時の照合］ダイアログボックスが表示されます。［インポート時の照合］ダイアログボックスの詳細は、「8-21-2 インポート時の照合


」を参照してください。

使用上の注意

インポートしたプログラムは、十分な動作確認を行ったあと、本運転に移行してください。

## 8-21-2 インポート時の照合

インポート前にプロジェクト内にプログラムデータが存在しており、インポート対象に属性の異なる同じ名前の変数やデータ型が存在したり、異なるラダー図やST プログラムが存在したりする場合は、該当データについて上書きするかを確認するための［インポート時の照合］ダイアログボックスが表示されます。
