# モニタ・デバッグ・シミュレーション
実機モニタ、シミュレータによるオフラインデバッグ、動作記録/再現のカテゴリです。

## 用語対応

| 概念 | GX Works3 | KV STUDIO | Sysmac Studio |
|---|---|---|---|
| シミュレータ | GX Simulator3 (11章・付11) | シミュレータ機能 (9-11) | シミュレータ (7-3) |
| 実機モニタ | モニタ (14章) | モニタ機能 (9-2) | オンラインデバッグ (7-5) |
| 変数/デバイス監視 | ウォッチウィンドウ | 登録モニタ (9-3)・一括モニタ (9-4) | ウォッチウィンドウ |
| 波形/チャート | - | リアルタイムチャートモニタ (9-6)・高速タイムチャート (9-7) | データトレース |
| 現在値変更 | 現在値変更 | デバイス値一括変更 (9-9) | 設定値変更/強制値リフレッシュ |
| RUN中の変更 | RUN中書込み | オンラインエディット (9-10) | オンライン編集 |
| 動作記録・再現 | データロギング/メモリダンプ/レコーディング (17章) | 運転記録/リプレイ (10章) | データトレース (7章) |
| ステップ実行等 | (シミュレーション中の機能) | デバッグ (11章) | 一時停止/コマ送り (シミュレーション) |
| 実機との差異 | 付11 | 9-11内の注意 | 付A-4 |

## 主な参照先

- **GX Works3**: 11章(シミュレーション)・14章(動作確認)・17章(データの収集/再現)。
- **KV STUDIO**: 9章(モニタ/シミュレータ)・10章(運転記録/リプレイ)・11章(デバッグ)。
- **Sysmac Studio**: 7章(デバッグ)。シミュレーション専用命令は付A-7。
