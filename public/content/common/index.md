# FA開発環境 3社共通ガイド
本サイトは、三菱電機 **GX Works3**、キーエンス **KV STUDIO**、オムロン **Sysmac Studio** の公開資料を横断し、3つのPLC開発環境を共通のカテゴリ構造で整理した技術リファレンスです。

## 3製品の基本情報

| | GX Works3 | KV STUDIO | Sysmac Studio |
|---|---|---|---|
| メーカー | 三菱電機 | キーエンス | オムロン |
| 対象PLC | MELSEC iQ-R / iQ-F / MELSEC-Q(移行) / MXコントローラ | KV-8000 / KV-7000 / KV-5000/3000 / KV Nano | NJ / NX / NY シリーズ |
| 主なプログラム言語 | ラダー / ST / FBD/LD / SFC | ラダー / ST / スクリプト / KV FUNC | ラダー / ST |
| 準拠規格 | IEC 61131-3 (ラベルプログラミング) | 独自 + IEC風の変数/ST対応 | IEC 61131-3 準拠 |
| 変数の考え方 | デバイス + ラベル(グローバル/ローカル) | デバイス中心 + 変数/ラベル | 変数中心 (デバイス直接参照は基本なし) |
| シミュレータ | GX Simulator3 (内蔵) | シミュレータ機能 (内蔵) | シミュレータ (内蔵、EtherCAT/モーション対応) |

| 実機モニタ | モニタモード / ウォッチウィンドウ | モニタ / 登録モニタ / 一括モニタ | ウォッチウィンドウ / データトレース |

## カテゴリ構成

1. **導入・画面・基本操作** — インストール、動作環境、画面構成、基本操作
2. **プロジェクト管理** — 作成・保存・変換・照合・ファイル形式
3. **ハードウェア構成・パラメータ** — ユニット構成、ネットワーク構成、各種パラメータ
4. **変数・デバイス** — デバイス、ラベル、変数、データ型
5. **プログラミング** — 各言語での編集、検索/置換、部品化、変換/ビルド
6. **接続・転送** — PLCとの接続経路、書込み/読出し
7. **モニタ・デバッグ・シミュレーション** — モニタ、シミュレータ、動作記録
8. **保守・セキュリティ・運用** — データ保護、診断、バックアップ
9. **リファレンス・付録** — エラー一覧、ショートカット、予約語
