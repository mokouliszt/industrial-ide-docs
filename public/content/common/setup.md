# 導入・画面・基本操作
インストール、動作環境、画面構成、エディタの基本操作に関するカテゴリです。

## 用語対応

| 概念 | GX Works3 | KV STUDIO | Sysmac Studio |
|---|---|---|---|
| メイン画面のツリー | ナビゲーションウィンドウ | ワークスペース(プロジェクトツリー) | マルチビューエクスプローラ |
| 部品/命令の選択パレット | エレメント選択ウィンドウ | 命令パレット等 | ツールボックス |
| 出力/結果表示 | 出力ウィンドウ | 出力ウィンドウ | ビルドウィンドウ/出力タブ |
| ドッキング可能な補助画面 | ドッキングウィンドウ | 各種ウィンドウ | クロスリファレンスタブ等 |
| 動作モード | - | エディタモード/モニタモード等 | - |

## 主な参照先

- **GX Works3**: 1章(ご使用の前に)・2章(画面構成と基本操作)。プロジェクト作成からCPU稼働までの手順は 1.8 が起点。
- **KV STUDIO**: 1章(お使いの前に)。モード概念(エディタ/モニタ/シミュレータ等)は 1-5 基本操作が起点。6章(表示)にエディタの表示カスタマイズ。
- **Sysmac Studio**: 1章(はじめに)・2章(インストール)・3章(システムの設計)。画面各部は 3-4、機能一覧は 3-7。
