# プログラミング
各言語でのプログラム作成・編集、検索/置換、部品化(FB/マクロ/ライブラリ)、変換/ビルドのカテゴリです。

## 用語対応

| 概念 | GX Works3 | KV STUDIO | Sysmac Studio |
|---|---|---|---|
| プログラム単位 | プログラム本体 (POU) | モジュール / プログラム | POU (プログラム/FB/FUN) |
| ラダー編集 | ラダーエディタ | ラダー編集 (5章) | ラダーエディタ (4-5) |
| ST編集 | STエディタ / インラインST | ST/スクリプト (13章) | STエディタ (4-6)・インラインST |
| コンパイル | 変換 / 全変換 | 変換 (8章) | ビルド / リビルド (4-10) |
| 文法チェック | 変換時チェック | 変換時チェック | プログラムチェック (4-9) |
| 部品化 | FB / FUN / ライブラリ (10章) | ファンクションブロック / マクロ (7章) | FB / FUN / ライブラリ (9章) |
| 検索/置換 | データの検索 (9章) | 編集メニューの検索系 (5章) | 検索/置換 (4-8) |
| 相互参照 | クロスリファレンス | 使用状況の確認 | クロスリファレンスタブ |

## 主な参照先

- **GX Works3**: 6章(プログラムの作成)が中核。9章(データの検索)、10章(プログラムの部品化)。
- **KV STUDIO**: 5章(編集)・7章(プログラム)・8章(変換)・13章(ST/スクリプト)。
- **Sysmac Studio**: 4章(プログラムの設計)が中核。9章(プログラムの再利用)。
