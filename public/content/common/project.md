# プロジェクト管理
プロジェクトの作成・保存・変換・照合・セキュリティなどプロジェクトファイル操作のカテゴリです。

## 用語対応

| 概念 | GX Works3 | KV STUDIO | Sysmac Studio |
|---|---|---|---|
| 保存形式 | ワークスペース形式 / 1ファイル形式(.gx3) | プロジェクトフォルダ(.kpr ほか多数ファイル) | プロジェクトファイル(.smc2/.csm2) |
| 旧環境からの移行 | GX Works2/GX Developer形式の読出し | KV BUILDER/KZ用データの読み込み | CX-One(CX-Programmer)からは非互換(別ツール) |
| プロジェクト照合 | プロジェクト照合 | プロジェクト比較 | オフライン照合 |
| 他形式出力 | IEC61131-10 XML(付8) | - | IEC 61131-10 XML / AutomationML インポート |
| セキュリティ | セキュリティキー/ブロックパスワード(15章) | プロジェクト保護 | プロジェクトのセキュリティ設定(8-3) |

## 主な参照先

- **GX Works3**: 3章(プロジェクトの管理)が中核。機種変更・他形式置換えは付7・付10。
- **KV STUDIO**: 2章(プロジェクト管理)・4章(ファイル・印刷)。
- **Sysmac Studio**: 3-3(新規プロジェクトの作成)、4-11(オフライン照合)、8章のインポート/エクスポート各節。
