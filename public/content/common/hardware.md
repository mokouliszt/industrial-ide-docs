# ハードウェア構成・パラメータ
ユニット/スレーブ構成の設計と、CPU・ネットワーク等のパラメータ設定のカテゴリです。

## 用語対応

| 概念 | GX Works3 | KV STUDIO | Sysmac Studio |
|---|---|---|---|
| 構成設計ツール | ユニット構成図 (iQ-R) / モジュール構成図 | ユニットエディタ | CPU・増設ラック構成 / EtherCAT構成 |
| フィールドネットワーク | CC-Link IE (別途設定) | KL/EtherNet/IP等の子機設定 | EtherCAT (5-2〜5-4) |
| CPU設定 | CPUパラメータ | CPUシステム設定 (KVS 6-2) | コントローラ設定 (5-6) |
| I/O割付 | ユニット構成図から自動/手動 | リレー割付 (ユニットエディタ) | I/Oマップ (デバイス変数割付) |
| モーション設定 | シンプルモーション設定 (付9) | 位置決めユニット設定 | モーション制御設定 (5-7)・カムデータ (5-8) |
| タスク/実行制御 | プログラム実行タイプ | モジュール/周期実行 | タスク設定 (5-9) |

## 主な参照先

- **GX Works3**: 4章(ユニット構成図の作成とパラメータの設定)。
- **KV STUDIO**: 3章(ユニットエディタ)、6-2(CPUシステム設定)。
- **Sysmac Studio**: 5章(コントローラ構成・設定)。EtherCAT中心の構成が特徴。
